Aprovechamiento de las extensiones de servidores web Microsoft y Netscape

Si va a usar un servidor web Microsoft o Netscape para distribuir informes, el Web Component Server puede aprovechar directamente la potencia de su servidor web a través de interfaces de programación ISAPI o NSAPI. El Web Component Server admite ambas API en un solo archivo: wcsinsapi.dll. Las API mejoran el desempeño de la aplicación web mediante extensiones directas al servidor web.

Para más información acerca de ISAPI, consulte la documentación de Microsoft. Para más información acerca de NSAPI, consulte la documentación de Netscape.
