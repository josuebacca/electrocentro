La solución para la creación de informes en el web de Seagate

Las ediciones Professional y Developer de Seagate Crystal Reports incluyen tres productos que permiten dar acceso a los usuarios a Seagate Crystal Reports a través del web.

  • Visor de informes (Report Viewer)
  • Web Component Server
  • Page Server

Esta solución para la elaboración de informes ha sido diseñada para sitios web con servidores web Microsoft y Netscape, así como para la mayoría de los servidores que se ajustan al estándar CGI y que funcionan en un entorno Microsoft Windows NT.
