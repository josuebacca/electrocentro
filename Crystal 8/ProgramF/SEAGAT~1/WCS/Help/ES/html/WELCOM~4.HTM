Contenido de las secciones

A continuación presentamos una breve descripción de cada capítulo de este manual.

Bienvenidos a la administración de informes en el web

Esta sección proporciona una descripción general del manual del usuario y de la ayuda en línea. También presenta una lista de las convenciones de comandos, botones y teclas utilizadas en el manual del usuario.

Introducción general sobre la solución para la creación de informes en el web

En esta sección se explica cómo los productos web de Seagate funcionan conjuntamente con su servidor web actual. También incluye información acerca de nuevas funciones y mejoras.

Implementación del Web Component Server

En esta sección se describen los procedimientos necesarios para instalar y configurar el Web Component Server.

Administración del Web Component Server

Esta sección proporciona detalles sobre cómo usar la aplicación Crystal Web Components Manager incluida con el Web Component Server. Con esta aplicación puede personalizar los ajustes del Web Component Server.

Comandos del Web Component Server

En esta sección se presentan ejemplos de los comandos del Web Component Server que se pueden adjuntar a las solicitudes de URL. Estas solicitudes de URL pueden emplearse para pedir que los usuarios introduzcan información de seguridad o valores en los campos de parámetro.

Visores de informes

En esta sección se proporciona información sobre cómo deben configurarse los visores de informes para satisfacer de manera efectiva las necesidades de los usuarios.

Apendice A: Soporte del producto

En este sección se presentan los métodos y la información de contacto necesarios para obtener asistencia técnica para el producto.
