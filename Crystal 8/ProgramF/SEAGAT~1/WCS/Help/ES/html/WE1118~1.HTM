Introducción sobre el proceso de implementación

Para implementar un Web Component Server, complete los siguientes cinco pasos:

  1. Instale el Web Component Server.
  2. Configure los servicios NT.
  3. Asegúrese de que la instalación se haya completado correctamente.
  4. Cree un sitio web de prueba.
  5. Especifique el visor de informes.

Nota:    Los pasos "Configure los servicios NT" y "Especifique el visor de informes" son opcionales.
