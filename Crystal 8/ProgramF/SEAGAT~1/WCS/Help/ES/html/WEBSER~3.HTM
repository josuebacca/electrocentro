Acceso a Crystal Report Web Components Manager

Para acceder a Crystal Report Web Components Manager
  1. Busque el archivo wcmadmin.exe.
  2. Haga doble clic en wcmadmin.exe para lanzar la aplicación.

    Aparecerá la aplicación Crystal Report Web Components Manager. Esta aplicación tiene cinco fichas:

      • Web Component Server
      • Vistas de informes
      • Page Server
      • Ajustes de la memoria caché
