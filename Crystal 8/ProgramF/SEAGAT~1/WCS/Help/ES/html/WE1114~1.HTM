Conector WCS CGI

La aplicación de conector WCS CGI (wcscgi.exe) ha sido diseñada para compatibilidad con el estándar CGI. Puesto que la mayoría de los servidores web son compatibles con CGI, el Web Component Server se puede instalar en cualquier servidor web que se ajuste al estándar CGI.

El servidor web Lotus Domino utiliza la extensión WCS CGI.
