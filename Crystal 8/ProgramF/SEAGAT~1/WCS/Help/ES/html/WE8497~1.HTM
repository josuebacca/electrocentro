Consideraciones

  • Los visores ActiveX y Java no están disponibles en las versiones 3.02 y anteriores de Internet Explorer.
  • Si usa Internet Explorer, debe instalar Authenticode 2.0.
  • Netscape Navigator no es compatible con el visor de informes para ActiveX.
  • La versión 3.0 y anteriores de Netscape Navigator no son compatibles con visores de informes para Java.
