Comandos CMD# y EXPORT_FMT

Especifican que el informe debe exportarse al formato indicado.

cmd=EXPORT&EXPORT_FMT=<representación EXPORT_FMT>

<representación EXPORT_FMT> es una de las siguientes.

Formato de exportación Representación Export_FMT

HTML 3.2

U2FHTML:2

HTML 4.0

U2FHTML:3

Seagate Crystal Reports (RPT)

U2FCR:0

Excel 5.0 (XLS)

U2FXLS:3

Excel 5.0 (XLS) extendido

U2FXLS:4

Formato RFT (RTF)

U2FRTF:0

Documento de Word (DOC)

U2FWORDW:0

Por ejemplo:

Si un usuario desea descargar el informe prueba.rpt a su explorador en formato Microsoft Word, la URL (codificación ISO - Latín) debe ser:

http://mipc/
prueba.rpt?cmd=EXPORT&EXPORT_FMT=U2FWORDW%3A0
