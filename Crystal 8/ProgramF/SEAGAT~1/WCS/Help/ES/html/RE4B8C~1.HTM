Certificación AuthentiCode

El visor de informes para ActiveX está certificado por Microsoft AuthentiCode 2.0. Esta certificación requiere Microsoft Internet Explorer 3.02 o una versión posterior para abrir el control ActiveX. Si no tiene una versión reciente de Internet Explorer, visite el sitio web de Microsoft para obtener una actualización, o bien utilice un visor de informes distinto al diseñar sus sitios web.
