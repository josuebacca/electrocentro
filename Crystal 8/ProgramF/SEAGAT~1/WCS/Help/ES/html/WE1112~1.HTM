Instalación del Web Component Server

Este procedimiento supone que usted ya ha instalado un servidor web y que ha comprobado que funciona correctamente. Debe iniciar la sesión como administrador en la máquina local.

Nota:    Antes de iniciar la instalación, compruebe que su servidor web no esté ejecutándose.
