Consideraciones sobre las impresoras

Puede que sea necesario cambiar los ajustes del Page Server Seagate si el servidor web no tiene una impresora local instalada y usted usa la cuenta del sistema local (o de otro usuario local).

Las cuentas locales, incluyendo la cuenta del sistema local, no permiten el uso de una impresora de red y, por lo tanto, el motor de informes emplea un modo diferente para generar los archivos de páginas encapsuladas (EPF). La calidad de estos EPF puede no ser tan buena como lo sería con un controlador de impresora específico.
