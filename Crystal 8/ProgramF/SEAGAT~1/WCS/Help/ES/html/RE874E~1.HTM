Visor de informes para Java usando un explorador JVM

El visor de informes para Java es un applet Java estándar que puede ubicarse en una página HTML y visualizarse a través de cualquier explorador que sea compatible con Java. Netscape Navigator (versión 2.0 y posterior) muestra los informes utilizando el visor de informes para Java de forma predeterminada.

Nota:    El visor de informes para Java sólo acepta objetos de línea simple y doble.
