Compatible con las funciones de seguridad de su base de datos

Con el uso de Web Component Server, Seagate Crystal Reports sigue apoyando los procedimientos de seguridad establecidos anteriormente en el web. Web Component Server reconoce automáticamente las funciones de seguridad establecidas y solicita a los usuarios que proporcionen su identificación y contraseña para acceder a informes que utilizan fuentes de datos ODBC y SQL.

O como alternativa, puede usar los comandos de Web Component Server para manejar automáticamente la seguridad a través de hipervínculos y otros vínculos web a los informes.
