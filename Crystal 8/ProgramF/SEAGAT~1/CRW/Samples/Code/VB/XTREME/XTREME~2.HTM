Xtreme Mountain Bikes Demo

  • Purpose: Advanced Crystal Reports demonstration of techniques for integrating reports into an application.


  • Summary: There are two major components to the demonstration application:
    1. A series of data forms allow the user to add, edit and delete customer, purchase order, employee and mailing data in the Xtreme database.
    2. Six reports about the Xtreme Mountain Bikes store that the user can view and print; including a Product Catalog, Mail Labels, Employee Profiles, an Income Statement, a Balance Statement, and a WorldWide Sales report. These reports use many of the features available in the Visual Basic Crystal Reports Designer Component.
