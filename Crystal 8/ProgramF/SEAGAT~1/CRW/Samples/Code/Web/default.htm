Muestras de Report Server

La página Reports Server le permite ver informes de muestra que exhiben varias características con Web Component Server o ASP Reports Server. Estas muestras le permiten comparar resultados de informes en todas estas opciones de servidores dentro de diferentes Smart Viewers.

 

Muestras de Developer Active Server Page

Estos ejemplos de Active Server Page ilustran el uso de características de automatización en scripting con Active Server Page. Hay comentarios sobre cada página de muestra y se puede utilizar como ejemplo o como plantilla para integrar Crystal Reports a sus propios programas ASP.

 

Applet de muestra utilizando Report Viewer JavaBean

Este applet o programa Java demuestra el uso de todas las propiedades y métodos disponibles en Report Viewer JavaBean. Para obtener una descripción de los métodos y propiedades del Report Viewer JavaBean, consulte la ayuda del desarrollador. Para ejecutar una muestra como programa independiente, coloque JavaBeanSample.jar y ReportViewerBean.jar en su ruta de clase e invoque com.seagatesoftware.img.RVTester.RVTester con su entorno de ejecución preferido de Java 1.1. El código de muestra y archivo .jar está disponible en:
<Directorio de Crystal Reports>\Samples\Code\Web\Java.
