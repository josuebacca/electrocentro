Usar Report Viewer JavaBean con complemento Sun Java

Si no tiene instalado el complemento Java y su explorador es Internet Explorer 3.02 o posterior, al hacer clic en este vínculo va a descargarlo e instalarlo antes de lanzar este applet. Si no tiene instalado el complemento Java y su explorador es Netscape Navigator/Communicator 3.0 o posterior, al hacer clic en este vínculo se va a descargar el programa de instalación a su PC. Una vez descargado, cierre su explorador y ejecute el programa de instalación. Cuando haya terminado la instalación del complemento Java, inicie su explorador y haga clic en el vínculo superior para invocar este applet.

Usar applet de muestra Report Viewer JavaBean con Java Virtual Machine (JVM) de su explorador

Si va a utilizar Netscape Navigator/Communicator 4.06 o posterior, o Microsoft Internet Explorer 4.01 (con Java Virtual Machine (JVM) versión 3.1) o posterior, podrá ejecutar este applet con Java Virtual Machine interna de su explorador (JVM). Los usuarios de Internet Explorer 4.01 pueden descargar la versión 3.1 de Microsoft JVM desde www.microsoft.com/Java.

Importante: Dado que el applet no contiene una firma de seguridad, se ejecuta dentro del modelo de seguridad "sandbox" de Java.  En consecuencia, al especificar la URL para su informe, asegúrese de que el nombre de usuario sea el mismo que el mostrado en la URL de su explorador (ej.:http://localhost).
