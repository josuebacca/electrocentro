Ficha Estilo

Use la ficha Estilo para insertar un título de informe y elegir un estilo para el informe.

Título

Escriba un título para el informe.

Estilo

Este cuadro muestra los estilos de formato disponibles para los informes. Seleccione el estilo que desee de la lista.

Vista previa de muestra

Este cuadro proporciona una muestra del estilo seleccionado en la lista Estilo.

Agregar una imagen

Haga clic en este botón para especificar un archivo de imagen para el informe.
