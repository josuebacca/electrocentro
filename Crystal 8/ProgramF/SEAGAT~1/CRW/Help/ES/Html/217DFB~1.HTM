Variables compartidas

Las variables compartidas usan el mismo bloque de memoria para almacenar el valor de una variable en todo el informe principal y todos sus subinformes. Así, las variables compartidas son incluso más generales que las variables globales. Para usar una variable compartida, declárela en una fórmula del informe principal como en el ejemplo siguiente:

Shared NumberVar x := 1000;

Y declárela en una fórmula del subinforme como en el siguiente ejemplo:

Shared NumberVar x;

Para usar las variables compartidas, se debe declarar y asignar un valor a la variable antes de pasarla al informe principal y al subinforme.
