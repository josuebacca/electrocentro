Capa de base de datos

El archivo de base de datos al que se refiere la fuente de datos ODBC se puede ubicar en cualquier parte del sistema. Una vez que se configura la fuente de datos ODBC, Seagate Crystal Reports no necesita conocer el formato y la ubicación real de los datos. Por ello, la base de datos puede tener cualquier formato y se puede ubicar en cualquier parte de la red siempre y cuando ODBC se pueda comunicar con ella mediante la fuente de datos ODBC.
