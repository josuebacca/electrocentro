FRPriceEarningsRatio (precioDeMercado, dividendosPorAcción)

Sintaxis básica y Crystal.

Formato

FRPriceEarningsRatio (precioDeMercado, dividendosPorAcción)

Argumentos
  • precioDeMercado es un número o fórmula. precioDeMercado es el precio actual en el mercado de las acciones comunes de la compañía. Esta información es pública si las acciones de la compañía se negocian en la bolsa de valores.
  • dividendosPorAcción es un número o fórmula. dividendosPorAcción es la cantidad de ganancia atribuida a cada acción común. Esta información está disponible en los informes financieros de la compañía.
Acción

FRPriceEarningsRatio devuelve la relación entre precioDeMercado y dividendosPorAcción (por ejemplo, PriceEarningsRatio = precioDeMercado / dividendosPorAcción (12 meses)).

Uso(s) típico(s)

Esta relación se usa como una comparación entre el precio de mercado de las acciones y su habilidad de producir ganancias. Es una figura conveniente y puede ser usada fácilmente para compara la habilidad de producir ganancias de su compañía con la de otra compañía.

Ejemplo(s)

Los siguientes ejemplos son aplicables a las sintaxis básica y Crystal:

FRPriceEarningsRatio(3.5, 2)

Devuelve 1.75.

FRPriceEarningsRatio(27, 1.5)

Devuelve 18.

Nota:    Ninguno de los argumentos de entrada pueden ser negativos; debe revisar si hay valores negativos antes de pasar los argumentos de entrada.

Comentario

FRPriceEarningsRatio es una de las muchas herramientas para análisis financiero que se usan para interpretar la posición financiera de una compañía. Como en el caso de todas las relaciones, se debe usar con precaución. Se puede usar sólo como una guía y no como prueba firme para formar una opinión sobre el estado financiero de una compañía.

Ninguna de las relaciones se puede usar exclusivamente para hacer análisis financieros y no hay reglas fijas sobre los resultados de las relaciones. Aparte de las diferencias entre los tipos de industrias, el resultado varía entre las compañías diferentes dentro de la misma industria y dentro de períodos de cuenta diferentes. Los resultados sólo se deben usar para hacer comparaciones relativas y análisis de tendencias que es preferible a tratarlos como algo absoluto.
