Previous (campo)

Sintaxis Crystal.

PreviousValue (campo) y Previous son funciones equivalentes. Sin embargo, sólo puede usar Previous con sintaxis Crystal y PreviousValue sólo con sintaxis básica.

Argumentos

campo es un campo de base de datos o una fórmula válida en el informe.

Devuelve

Un valor de campo del mismo tipo que el pasado como el argumento campo.

Acción

PreviousValue devuelve el valor del campo especificado en el registro previo.

Uso(s) típico(s)

Puede utilizar Previous para identificar el último registro en un rango previo o el último registro que ocurre antes de que un nuevo rango comience, o para evaluar en busca de valores duplicados.

Ejemplo(s)

Los siguientes ejemplos son aplicables para la sintaxis Crystal:

If Previous({archivo.CANT}) <> 0 Then
    {archivo.CANT}/2
Else
    {archivo.CANT}

Evalúa el valor previo en el campo {archivo.CANT} para ver si es un valor cero. Si no es, divide el valor entre dos. Si es un valor cero, devuelve el valor.

If Previous ({cliente.CLIENTE ID}) = {cliente.CLIENTE ID} Then
    "Valor Repetido"
Else
    ""

Pone indicadores en los valores repetidos en el campo{cliente.CLIENTE ID}.

If Remainder(Previous ({archivo.NUMSERIE), 300) = 0 Then
    "Comenzando, nuevo bloque"
Else
    ""

Divide el valor previo en el campo {archivo.SERIALNUM} entre 300. Si no hay residuo, pone un indicador en el valor actual como "Comenzando, nuevo bloque". Si hay un residuo (Else) no imprime el indicador. (Esta fórmula divide números de serie en bloques de 300.)

Nota:    El uso de esta función en una fórmula obliga a que la fórmula sea evaluada al momento de impresión. Para obtener más información sobre las consideraciones de evaluación de hora, véase el Funciones de tiempo de evaluación.
