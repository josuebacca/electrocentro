FileAuthor

Sintaxis básica y Crystal.

Devuelve

Una cadena

Acción

FileAuthor inserta (en una fórmula) el nombre del autor del archivo el cual es incluido con el informe como un campo.

Uso(s) típico(s)

Puede usar esta función cuando desee imprimir el nombre del autor del archivo (como se específica en el cuadro de diálogo Propiedades del Documento).

Ejemplo(s)

El siguiente ejemplo es aplicable a las sintaxis básica y Crystal:

FileAuthor

Crea un campo conteniendo el nombre del autor del archivo. El uso de esta fórmula es equivalente a usar el campo Autor del archivo en la carpeta Campos especiales en el Explorador de campos.
