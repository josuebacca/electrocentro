Vista de la opción Avanzada

Campos disponibles

Esta lista muestra todos los campos disponibles que puede usar en el gráfico.

Cambio de

Use los botones A cambio de para agregar o quitar campos seleccionados de la lista A cambio de. Haga clic en el botón Agregar (>) para agregar campos seleccionados de la lista Campos disponibles a la lista A cambio de. Haga clic en el botón Quitar (<) para quitar campos de la lista A cambio de.

Evaluar

Use esta lista para determinar cuándo debe aparecer un valor en el gráfico. Si selecciona "Para cada registro", el gráfico va a mostrar un valor para cada registro. Si selecciona "Para todos los registros", el gráfico va a mostrar un total para todos los registros. Si selecciona "A cambio de" y luego especifica un campo "a cambio de" al usar los botones Cambio de y la lista Campos disponibles, el gráfico va a mostrar un valor para cada cambio de ese campo.

cuadro de diálogo Insertar grupoN Superiores

Haga clic en el botón N Superiores para ordenar el campo seleccionado según N Superiores, N Inferiores u Organizar todo. Cuando hace clic en este botón, aparece el Asistente de N Superiores/ordenamiento de grupo.

Este botón sólo se activa cuando se selecciona un solo campo en la lista A cambio de.

Orden

Haga clic en el botón Orden para cambiar el orden del campo seleccionado. Cuando hace clic en este botón, aparece el cuadro de diálogo Insertar grupo.

Este botón sólo se activa cuando se selecciona un solo campo en la lista A cambio de.

Mostrar valor

Use los botones Mostrar valor para agregar o quitar el (los) campo(s) seleccionado(s) de la lista Mostrar valores. Haga clic en el botón Agregar (>) para agregar campos seleccionados de la lista Campos disponibles a la lista Mostrar valores. Haga clic en el botón Quitar (<) para quitar campos de la lista Mostrar valores.

Mostrar valores

Esta lista muestra el campo elegido como valor para su gráfico o diagrama. Se trazarán puntos para estos valores en el gráfico cada vez que cambie la condición seleccionada en el cuadro de edición A cambio de. Por ejemplo, si crea un gráfico de las ventas del último año para todos sus clientes, seleccionará el campo Nombre del cliente como condición y el campo Ventas del último año como valor. Cada vez que cambie el Nombre del cliente, se trazará un punto de acuerdo al valor de sus ventas del último año.

Quitar (<<)

Haga clic en este botón para quitar todos los campos de la lista Mostrar valores.

No resumir valores

Use la casilla de verificación No resumir valores para determinar si el programa debe resumir automáticamente los valores que van a aparecer en el gráfico.

  • Si selecciona esta casilla de verificación, el programa no va a resumir los valores.
  • Si no selecciona esta casilla de verificación, el programa va a resumir los valores de mapa y el botón Establecer operación de resumen se va a activar. El programa agrega un campo de resumen a la lista de Valores de mapa.
Establecer operación de resumen

Use este botón para establecer o cambiar la operación de resumen del campo seleccionado. Cuando hace clic en este botón, aparece el cuadro de diálogo Cambiar operación de resumen. Este botón sólo se activa cuando la casilla de verificación No resumir valores no está marcada y ha seleccionado un campo en la lista Valores de mapa.

Examinar

Haga clic en este botón para examinar los datos del campo seleccionado en la lista Campos disponibles.
