Carta modelo

El Asistente de carta modelo le ofrece una solución fácil para la creación de cartas modelo que usan su base de datos como fuente de información de clientes. El asistente contiene mucha de la funcionalidad del Asistente de informe estándar, pero tiene la ficha Carta modelo, que le permite definir los campos de base de datos y texto que aparecerán en cada sección de su carta. También puede usar la ficha Carta modelo para importar texto que haya creado en otra aplicación.
