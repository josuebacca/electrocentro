Operador If-Then-Else

Sintaxis Basic

Uso

Existen dos tipos de enunciados If, el If de multilínea (o líneas múltiples) y el If de una línea. Generalmente, es mejor usar el If de multilínea dado que genera fórmulas más claras.

Enunciados If de multilínea

If x Then
    statementsA
End If

Si x es verdadero, entonces evaluar enunciados A.

If x Then
    statementsA
Else
    statementB
End If

Si x es verdadero, entonces evaluar enunciados A, de lo contrario evaluar enunciados B.

If x Then
    statementsA
ElseIf y Then
    statementsB
Else
    statementsC
End If

Si x es verdadero, entonces evaluar enunciados A; de lo contrario si y es verdadero, evaluar enunciados B; de lo contrario evaluar enunciados C.

Nota:    

  • Puede haber 0 o más cláusulas ElseIf y 0 o 1 cláusula Else.
  • La cláusula Else debe ser la última.
  • ElseIf es una palabra.
  • Los enunciados If de multilínea deben terminar en End If.
Enunciados If de una línea

If x Then y Else z

Si x es verdadero, entonces hacer y. Si x no es verdadero (Else), hacer z.

Ejemplos

El siguiente ejemplo es aplicable a sintaxis Basic:

If {Empleado.Departamento} = "Ventas" Then
    formula = {Empleado.Salario} * 0.06
Else
    formula = {Empleado.Salario} * 0.04
End If

Asigna tasa de aumento de salario basado en el departamento del empleado

Comentarios

La parte If de la expresión puede incluir texto, números, expresiones Booleanas (Cust#<"10000"), y fórmulas ({@FORMULA}), donde @FORMULA es Booleano.

Crystal Syntax

Uso

If x Then y Else z

Si x es verdadero Entonces haga y. Si x no es verdadero (De lo contrario), haga z.

Examplos

El siguiente ejemplo es aplicable para sintaxis Crystal:

If {cliente.CODIGO POSTAL} <= "49999" Then
    "Etiqueta azul"
Else
    "Por tierra"

Asigna método de transporte basado en la distancia desde el punto de embarque.

If ToNumber ({archivo.ARTICULO}) >= 2500 and ToNumber({archivo.ARTICULO}) < 2600 Then
    "Seasonal"
Else
    ""

La expresión If incluye un operador     and para la función función ToNumber.

If {archivo.CUENTA} >= 25 Then
    {archivo.DISTRIBUIDOR} * {archivo.CUENTA}
Else
    {archivo.DISTRIBUIDOR} * {archivo.CUENTA}

La cantidad solicitada establece el precio de lista.

If {archivo.A MANO}>10 Then
    {archivo.ORDENADO}
Else
    If {archivo.ORDENADO} < 5 Then
        {archivo.ORDENADO}
    Else
        2

Asignación basada en la cantidad solicitada usando anidar If-Then-Else    .

If PageNumber = 1 Then
    PrintDate
Else
    CDate(0,0,0)

Imprime la fecha de impresión (de la función PrintDate en la primera página , y no imprime [designada por la fecha vacía Date(0,0,0)] en las páginas restantes.

Comentarios
  • La parte If de la expresión puede incluir texto, números, expresiones Booleanas (Cust#<"10000"), y fórmulas ({@FORMULA}), donde @FORMULA es Booleano.
  • Las partes Then y Else, sin embargo, deben ser ambas del mismo tipo: (Then Texto, Else Texto; Then número, Else número). Mezclar acciones de Texto y número genera un mensaje de error.
