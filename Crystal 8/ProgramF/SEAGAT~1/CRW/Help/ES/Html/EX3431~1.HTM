Ficha Opciones

Con la ficha Opciones podrá establecer distintos parámetros de visualización de gráficos.

Color del gráfico

Use los botones de la opción Color del gráfico para especificar si el gráfico va a ser de color o en blanco y negro.

Puntos de datos

Use los botones de la opción Puntos de datos para especificar el tipo de etiqueta que va a aparecer para cada elemento de datos (es decir, para cada valor del gráfico). Puede hacer clic en Ninguno, Mostrar etiqueta o Mostrar valor. Si hace clic en Mostrar valor, se activa la lista Formato de número. Use la lista Formato de número para seleccionar un formato para los valores del gráfico. Por ejemplo, puede seleccionar un formato de 1 (numeral) o puede seleccionar un formato de $1,00 (monetario).

Fondo transparente

Cuando se selecciona esta casilla de verificación, el fondo del gráfico queda transparente con el fin de hacer visibles los objetos posicionados detrás.

Tamaño del marcador

Use esta lista para especificar el tamaño del marcador para los marcadores de valores de gráfico en un gráfico de línea.

Figura del marcador

Use esta lista para especificar la forma del marcador para los marcadores de valores de gráfico en un gráfico de línea.

Ángulo de visualización

Use esta lista para especificar el ángulo desde el cual se visualiza el gráfico (ángulo estándar, desde arriba, etc.).

Tamaño del gráfico circular

Use esta lista para seleccionar el tamaño del círculo para un gráfico circular (mínimo, pequeño, promedio, grande o máximo).

Tamaño de la barra

Use esta lista para seleccionar el tamaño de la barra para un gráfico de barra (Mínimo, Pequeño, Promedio, Grande o Máximo).

Separar sección del gráfico circular

Cuando se selecciona esta casilla de verificación, una sección se separa del gráfico circular. Puede hacer clic en La sección más pequeña o La sección más grande para especificar qué sección del gráfico desea separar.

Mostrar leyenda

Cuando se selecciona esta casilla de verificación, se muestra la leyenda para el gráfico o diagrama. Cuando la marca de verificación no aparece, la opción no queda seleccionada y no se muestra ninguna leyenda para el gráfico. La opción está seleccionada en forma predeterminada.

Ubicación

Use la lista Ubicación para especificar dónde desea que aparezca la leyenda del gráfico (esquina superior derecha del gráfico, a la izquierda, a la derecha, etc.).

Esta lista sólo se activa cuando se ha seleccionado la casilla de verificación Mostrar leyenda.

Diseño

Use esta lista para mostrar valores porcentuales, valores reales, valores reales y porcentuales o ningún valor.
