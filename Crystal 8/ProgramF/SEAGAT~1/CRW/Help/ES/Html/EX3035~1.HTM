Ficha Filas/Columnas (OLAP)

La ficha Filas/Columnas le permite elegir dimensiones y campos que puede mostrar en la cuadrícula.

Dimensiones

Use esta lista para seleccionar dimensiones del objeto.

Columnas

Esta lista muestra las dimensiones de las columnas.

Filas

Esta lista muestra las dimensiones de las filas.

Agregar columna

Use este botón para agregar una dimensión seleccionada a una columna.

Agregar fila

Use este botón para agregar una dimensión seleccionada a una fila.

Quitar

Use este botón para quitar una dimensión seleccionada de la dimensión de fila o columna.

Usar nombres de campo largos

Use esta casilla de verificación para incluir nombres de campo largos en el objeto de cuadrícula OLAP. Si desactiva esta casilla de verificación, el programa usará nombres de campo cortos, como PR01 y PR02 en lugar de Bicicletas de carreras y Bicicletas de montaña.
