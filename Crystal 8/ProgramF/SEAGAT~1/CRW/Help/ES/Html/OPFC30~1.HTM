Operador Dividir

Sintaxis Basic y Crystal.

Uso

x/y

Dividir valor x por valor y.

Examplos

Los siguientes ejemplos son aplicables a sintaxis Basic y Crystal:

25/5

Retorna 5

1/3

Retorna .333333

{archivo.VENTAS} / {archivo.PRONOSTICO}

Retorna .875, donde {archivo.VENTAS} = 52533, {archivo.PRONOSTICO} = 60000.

{archivo.DIAS PARA EL VENCIMIENTO} / 5

Retorna 22, donde {archivo.DIAS PARA EL VENCIMIENTO} = 110.

Nota:    Si el denominador es = 0, el informe se detiene con una advertencia de división por cero. Si quiere evitar este tipo de problema incluya un enunciado para manejar la situación.

El siguiente ejemplo es aplicable a sintaxis Crystal:

If {archivo.PRONOSTICO} = 0 Then
    0
Else
    {archivo.VENTAS} / {archivo.PRONOSTICO}
