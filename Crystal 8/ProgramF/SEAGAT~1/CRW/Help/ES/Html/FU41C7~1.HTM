Next (Campo)

Sintaxis Crystal.

Next y NextValue (Campo) son funciones equivalentes. Sin embargo, sólo puede usar Next con sintaxis Crystal y NextValue (Campo) sólo con sintaxis básica.

Argumentos

campo es cualquier campo de fórmula o de base de datos válido.

Devuelve

Un valor de campo del mismo tipo que el argumento.

Acción

Next(campo) devuelve el valor del siguiente registro del campo especificado.

Uso(s) típico(s)

Puede utilizar Next para identificar el primero o el último dato dentro de un rango, para identificar el punto inicial para un nuevo grupo, o para evaluar la existencia de valores duplicados.

Ejemplo(s)

Los siguientes ejemplos son aplicables para la sintaxis Crystal:

If Next({archivo.CANT})<>0 Then
    {archivo.CANT}/2
Else
    {archivo.CANT}

Evalúa el valor siguiente en el campo {archivo.CANT} para ver si es un valor cero. Si no es, divide el valor entre dos. Si es un valor cero, devuelve el valor.

If Remainder(Next ({archivo.NUMSERIE), 300) = 0 Then
    "Fin de bloque"
Else
    ""

Divide el valor siguiente en el campo {archivo.SERIALNUM} entre 300. Si no hay residuo, marca el valor actual como "Fin de bloque". Si hay residuo (Else) no imprime ningún indicador. (Esta fórmula divide números de serie en bloques de 300 ).

Nota:    El uso de esta función en una fórmula obliga a que la fórmula sea evaluada al momento de impresión. Para obtener más información sobre las consideraciones de evaluación de hora, véase el Funciones de tiempo de evaluación.
