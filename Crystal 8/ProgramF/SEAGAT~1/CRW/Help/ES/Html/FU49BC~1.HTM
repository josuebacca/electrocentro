FRDividendYield (dividendo, precioDeMercado)

Sintaxis básica y Crystal.

Formato

FRDividendYield (dividendo, precioDeMercado)

Argumentos
  • dividendo es un número o fórmula. dividendo es el dividendo anual indicado por acción común.
  • precioDeMercado es un número o fórmula. precioDeMercado es el precio de venta actual de la acción común de una compañía. Esta información se ofrece públicamente si las acciones de una compañía se venden en una bolsa de valores.
Acción

FRDividendYield devuelve la relación entre Dividendo y precioDeMercado (por ejemplo, DividendYield = Dividendo / precioDeMercado).

Uso(s) típico(s)

Esta relación se usa como una indicación del rendimiento de inversión para un inversionista de acciones comunes. Es una comparación superficial entre las acciones de diferentes compañías. Algunos analistas también podrían usar la misma función para calcular el rendimiento de la acción preferida.

Ejemplo(s)

Los siguientes ejemplos son aplicables a las sintaxis básica y Crystal:

FRDividendYield(3,25)

Devuelve 0.12.

FRDividendYield(5, 80)

Devuelve 0.0625.

Nota:    Ninguno de los argumentos de entrada pueden ser negativos; debe revisar si hay valores negativos antes de pasar los argumentos de entrada.

Comentario

FRDividendYield es una de las muchas herramientas para análisis financiero que se usan para interpretar la posición financiera de una compañía. Como en el caso de todas las relaciones, se debe usar con precaución. Se puede usar sólo como una guía y no como prueba firme para formar una opinión sobre el estado financiero de una compañía.

Ninguna de las relaciones se puede usar exclusivamente para hacer análisis financieros y no hay reglas fijas sobre los resultados de las relaciones. Aparte de las diferencias entre los tipos de industrias, el resultado varía entre las compañías diferentes dentro de la misma industria y dentro de períodos de cuenta diferentes. Los resultados sólo se deben usar para hacer comparaciones relativas y análisis de tendencias, y no se deben tratar como algo absoluto.
