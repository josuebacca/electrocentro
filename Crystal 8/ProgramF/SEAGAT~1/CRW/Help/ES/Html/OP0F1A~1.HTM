Operador Menor o igual que

Sintaxis Basic y Crystal.

Uso

x <= y

x ies menor o igual que y.

Examplos

Los siguientes ejemplos son aplicables a sintaxis Basic y Crystal:

{archivo.PESO} <= 200

VERDADERO, donde {archivo.PESO} = 150, {archivo.PESO} = 200, o {archivo.PESO} = 199.

{archivo.PESO} <= 200

FALSO, donde {archivo.PESO}= 400.

{archivo.COSTO} <= {archivo.PRECIO}

VERDADERO, donde {archivo.COSTO} =350, {archivo.PRECIO} = 400, o donde {archivo.COSTO} = 350 y {archivo.PRECIO} = 350.

{archivo.COSTO} <= {archivo.PRECIO}

FALSO, donde {archivo.COSTO} = 350 y {archivo.PRECIO}= 325.
