PreviousValue (campo)

Sintaxis básica

PreviousValue y Previous (campo) a son funciones equivalentes. Sin embargo, sólo puede usar PreviousValue con sintaxis básica y Previous sólo con sintaxis Crystal

Argumento

campo es el nombre del campo de base de datos.

Devuelve

Valor del campo especificado; el tipo depende del tipo de ese campo.

Acción

PreviousValue devuelve el valor del campo especificado en el registro previo.

Uso típico

Puede utilizar Previous para identificar el último registro en un rango previo o el último registro que ocurre antes de que un nuevo rango comience, o para evaluar en busca de valores duplicados.

Ejemplo(s)

Los siguientes ejemplos son aplicables a la sintaxis básica:

If PreviousValue ({DetallePedidos.Cantidad}) <> 0 Then
    formula = {DetallePedidos.Cantidad} / 2
Else
    formula = {DetallePedidosl.Cantidad}
End If

Evalúa el valor previo en el campo {Orders Detail.Quantity} para ver si es un valor cero. Si no es, divide el valor entre dos. Si es un valor cero, devuelve el valor.

If PreviousValue ({Pedidos.IDPedido}) Mod 10 = 0 Then
    formula = "Comienzo de nuevo bloque"
Else
    formula = "Mismo bloque"
End If

Divide los pedidos en bloques de 10. Si el anterior IDPedido es un múltiplo de 10, marca el valor actual como "Comienzo de nuevo bloque". De lo contrario, imprime "Mismo bloque".

If PreviousValue ({cliente.CLIENTE ID}) = {cliente.CLIENTE ID} Then
    formula = "Cliente Repetido"
Else
    formula = "Cliente Cambiado"
End If

Pone indicadores en los valores repetidos en el campo{cliente.CLIENTE ID}.

Nota:    El uso de esta función en una fórmula obliga a que la fórmula sea evaluada al momento de impresión. Para obtener más información sobre las consideraciones de evaluación de hora, véase Funciones de tiempo de evaluación.
