Tipo monetario

Use el signo dolar ($) para crear un valor monetario.

Ejemplo
$10000
-$20
$1.23

También puede usar la función CCur. La C inicial en CCur significa "convert" y se puede usar para convertir valores numéricos en valores de moneda.

CCur (10000)
CCur (-20)
CCur (1.23)
