StrReverse (cadenaDeEntrada)

Sintaxis básica y Crystal.

Argumento

cadenaDeEntrada es una cadena cuyos caracteres están invertidos.

Devuelve

Una cadena de texto.

Acción

StrReverse devuelve una cadena en la cual el orden de los caracteres de cadenaDeEntrada está invertido. Si cadenaDeEntrada tiene una longitud de cadena de cero (""), se devuelve una cadena de longitud cero.

Ejemplo(s)

El siguiente ejemplo es aplicable a las sintaxis básica y Crystal:

StrReverse("abc")

Devuelva "cba".

Comentario(s)

Esta función está diseñada para trabajar exactamente como la función de Visual Basic del mismo nombre.
