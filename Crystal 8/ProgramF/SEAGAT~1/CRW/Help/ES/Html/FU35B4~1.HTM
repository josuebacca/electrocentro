ExchGetOrganization (dirección)

Sintaxis básica y Crystal.

Argumentos

dirección es la dirección del emisor/receptor (tipo de datos de cadena).

Devuelve

Una cadena.

Acción

Esta función comenzará por determinar si la dirección está en formato X500 o X400. Una vez que esto ha sido solucionado, entonces la función determinará el Nombre de Organización.

  • Si el campo está en formato X500, la función extraerá la última instancia del código "/O=" (no sensible a mayúsculas/minúsculas).
  • Si el campo está en el formato X400, extraerá la instancia del código "P=" code (no sensible a mayúsculas/minúsculas).

Nota:    Los campos deben seguir los estándares de tipo de dirección para las funciones para operarlos.

Ejemplo(s)

Los siguientes ejemplos son aplicables para la sintaxis Crystal:

Campo = /o=Seagate Software/ou=CRYSTALNT/
cn=Configuration/
cn=Servers/cn=ESPRESSO/cn=Fredt
ExchGetOrganization({Campo})

Devuelve "Seagate Software".

Campo = c=US; a= ;p=Microsoft; o=apps-wga; da:ms=com/
Rich/James.
ExchGetOrganization({Campo})

Devuelve "Microsoft".

Campo = c=US; a= ; p=Microsoft; o=apps-wga;
da:smtp=James@crystal.com.
ExchGetOrganization({Campo})

Devuelve "Microsoft".
