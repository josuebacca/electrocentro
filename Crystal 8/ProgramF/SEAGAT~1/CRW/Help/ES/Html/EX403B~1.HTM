Vista de la opción Avanzada

Use la vista de la opción Avanzada cuando tenga valores múltiples de mapa, en lugar de uno sólo, o cuando no tenga ningún grupo ni resúmenes en el informe. Con el diseño avanzado puede determinar qué campos se usan para el desglose geográfico, el valor A cambio de y el valor de mapa. También puede establecer opciones de resumen.

Nota:    Debe crear un mapa sobre un campo geográfico, como un país, región o ciudad. El programa no puede generar un mapa basándose en un campo no geográfico.

Campos disponibles

La lista Campos disponibles le permite seleccionar qué campos usar como datos para el mapa.

Examinar

Haga clic en el botón Examinar para visualizar los datos contenidos en el campo seleccionado en la lista Campos disponibles.

Agregar

Haga clic en el botón Agregar (>) para copiar campos de la lista Campos disponibles al cuadro de texto Campo geográfico, al cuadro de texto A cambio de o a la lista Valores de mapa.

Quitar

Haga clic en el botón Quitar (<) para eliminar un campo seleccionado de la lista Valores de mapa.

Quitar todos

Haga clic en el botón Quitar todos (<<) para eliminar todos los campos de la lista Valores de mapa.

Campo geográfico

Use el cuadro de texto Campo geográfico para determinar el desglose geográfico del mapa. Por ejemplo, si inserta el campo País.Cliente en el cuadro de texto Campo geográfico, el mapa va a mostrar valores para cada país. Si inserta el campo Región.Cliente, el mapa va a mostrar valores para cada región.

A cambio de

Use la lista A cambio de para especificar un campo a usar como condición para la visualización de un campo de resumen. Los valores de resumen se mostrarán cada vez que cambie el valor en el campo de condición.

Valores de mapa

Use la lista Valores de mapa para crear una lista de todos los campos que desea que aparezcan como valores en el mapa.

No resumir valores

Use la casilla de verificación No resumir valores para determinar si el programa debe resumir automáticamente los valores que van a aparecer en el mapa.

  • Si selecciona esta casilla de verificación, el programa no resumirá los valores.
  • Si no selecciona esta casilla de verificación, el programa resumirá los valores del mapa y el botón Establecer operación de resumen se activará. El programa agrega un campo de resumen a la lista Valores de mapa.
Establecer operación de resumen

Use este botón para establecer o cambiar la operación de resumen del campo seleccionado. Cuando hace clic en este botón, aparece el cuadro de diálogo Cambiar operación de resumen. Este botón sólo se activa cuando la casilla de verificación No resumir valores no está marcada y cuando ha seleccionado un campo en la lista Valores de mapa.
