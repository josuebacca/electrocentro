Operador Igual

Basic and Crystal syntax.

Uso

x = y

x es igual a y

El operador igual le dice al programa que evalúe una expresión (x = y) y retorne VERDADERO (si x es igual     a y) o FALSO (si x no es igual a y).

Examplos

Los siguientes ejemplos son aplicables a sintaxis Crystal y Basic:

{archivo.CANTIDAD} = 3

VERDADERO, donde {archivo.CANTIDAD} tiene un valor de 3.

{archivo.CANTIDAD} = 3

FALSO, en todas las otras situaciones.

{archivo.TOTAL HASTA AHORA} = {archivo.MISMO PERIODO AÑO PASADO}

VERDADERO, donde el valor de campo {archivo. MISMO PERIODO AÑO PASADO} es idéntico al valor de campo {archivo.MISMO PERIODO AÑO PASADO}.

{archivo.TOTAL HASTA AHORA} = {archivo.MISMO PERIODO AÑO PASADO}

FALSO, en todas las otras situaciones.

({archivo.VENTAS} - {archivo.COGS}) = 22,654

VERDADERO, donde el cálculo de la expresión {archivo.VENTAS}-{archivo.COGS} genera el valor 22,654, por ejemplo {archivo.VENTAS} = 109,986} y {archivo.COGS} = 87,332}.

({archivo.VENTAS} - {archivo.COGS})= 22,654

FALSO, en todas las otras situaciones.

{cliente.APELLIDO} = "Johnson"

VERDADERO, donde la cadena de texto en el campo {cliente.APELLIDO} es "Johnson".

{cliente.APELLIDO} = "Johnson"

FALSO, en todas las otras situaciones.

Comentarios

Este Operador a menudo se usa con expresiones que contienen el Operador If-Then- Else operator. Por ejemplo en sintaxis Crystal:

If {archivo.COMPRAS} = 0 Then
    "Su cuenta no tuvo movimiento este mes."
Else
    "";

Que imprime la afirmación "Su cuenta..." si el campo{archivo.COMPRAS} tiene un valor cero y no imprime (indicado por la cadena de texto vacío "" ), si el valor del campo {archivo.COMPRAS} es diferente de cero.
