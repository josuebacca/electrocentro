Ficha Color

Use la ficha Color para establecer el color de fondo de la sección deseada.

Color de fondo

Para agregar un color de fondo a la sección seleccionada, marque esta casilla de verificación, luego elija un color de la lista.

Fórmula condicional

Haga clic en este botón si desea configurar una fórmula condicional de formato para la propiedad correspondiente. Vea Trabajo con Formato condicional para obtener más información.
