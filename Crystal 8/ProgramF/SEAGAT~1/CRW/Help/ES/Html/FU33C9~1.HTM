PreviousIsNull (campo)

Sintaxis básica y Crystal.

Argumentos

campo es todo campo numérico, de fórmula, memo o BLOB del informe.

Devuelve

Un valor booleano.

Acción

Evalúa el campo especificado en el registro previo y devuelve un VERDADERO si el campo contiene un valor nulo.

Uso(s) típico(s)

Puede utilizar esta función para evaluar el primer elemento en una lista y realizar alguna acción cuando el primer elemento es identificado.

Ejemplo(s)

El siguiente ejemplo es aplicable a la sintaxis Crystal:

If PreviousIsNull ({empleados.EMPLEADOS ID}) Then
    "Primer empledo del registro"
Else
    ""

En una base de datos de empleados sin valores nulos en un campo Empleados ID, se pone un indicador al primer empleado en la lista.

Comentario(s)

Algunas bases de datos soportan valores de datos nulos y otras no. Esta función no trabajará si la(s) base(s) de datos activa(s) no soportan valores nulos. Si la(s) base(s) de datos no soportan valores nulos, la función puede ser utilizada para referenciar los valores nulos que son creados como un resultado de una búsqueda fallido mientras combina.

Nota:    

  • El cero y el espacio en blanco (vacío o " ") no son valores nulos.
  • El uso de esta función en una fórmula obliga a que la fórmula sea evaluada al momento de impresión. Para obtener más información sobre las consideraciones de evaluación de hora, véase el Funciones de tiempo de evaluación.
