Ficha Texto

La ficha Texto le permite especificar títulos, notas al pie y títulos de eje para un gráfico.

Texto automático

Cuando se seleccionan estas casillas de verificación, los títulos de los gráficos se generan en forma automática. Si no se seleccionan, es posible especificar manualmente los títulos para los gráficos.

Título

Use este cuadro de texto para ingresar un título para el gráfico. El título aparece justo sobre el gráfico.

Subtítulo

Use este cuadro de texto para ingresar un subtítulo para el gráfico. El subtítulo aparece justo debajo del título, sobre el gráfico.

Nota al pie

Use este cuadro de texto para ingresar texto para una nota al pie. La nota al pie aparece justo debajo del gráfico del informe.

Título de grupo (eje x)

Use este cuadro de texto para ingresar un título para el eje x del gráfico.

Título de datos (eje y)

Use este cuadro de texto para ingresar un título para el eje y del gráfico.

Título de datos 2 (eje y 2)

Use este cuadro de texto para ingresar un título para el segundo eje y del gráfico (si va a crear un gráfico de tabla cruzada u OLAP).

Título de serie

Use este cuadro de texto para ingresar un título para la serie del gráfico.

Texto apilado

Use esta casilla de verificación para apilar el texto de las etiquetas.

Esta casilla de verificación sólo se activa cuando ha seleccionado etiquetas de grupos, etiquetas de datos o etiquetas de serie en la lista Formato.

Fuente

Haga clic en el botón Fuente para establecer las opciones de fuente para el conjunto de texto seleccionado en la lista Formato. Cuando hace clic en este botón, aparece el cuadro de diálogo Fuente.

Formato

Use esta lista para seleccionar el texto para el cual establece las opciones de fuente. Por ejemplo, podría seleccionar el título del gráfico, el subtítulo del gráfico, la leyenda del gráfico, etc.
