RGB (rojo, verde, azul)

RGB y Color son funciones equivalentes. Sin embargo, en la sintaxis básica se prefiere RGB y en Crystal, Color.

Argumentos
  • rojo es un valor Numérico que representa el componente rojo del color resultante, con un valor entre 0 y 255.
  • verde es un valor Numérico que representa el componente verde del color resultante, con un valor entre 0 y 255.
  • azul es un valor Numérico que representa el componente azul del color resultante, con un valor entre 0 y 255.
Devuelve

Atributo de color de fuente.

Acción

RGB devuelve el atributo de color como se especifica en los componentes rojo, verde y azul.

Uso típico

Para especificar un color personalizado al escribir fórmulas de formato condicional para el color de fuente o de fondo de un campo de informe.

Ejemplo

El siguiente ejemplo es aplicable a la sintaxis básica:

If CurrentFieldValue > 10000 Then
    formula = RGB (150,230,150)
Else
    formula = RGB (10,19,200)
End If

Esta fórmula formatea el campo Numérico para que se despliegue en el color que especifica RGB (150,230,150) si su valor actual es mayor que 10000; de lo contrario se despliega en RGB (10,19,200).
