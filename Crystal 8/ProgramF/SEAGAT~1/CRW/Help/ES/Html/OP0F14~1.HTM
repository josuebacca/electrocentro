Operador En matriz

Sintaxis Basic y Crystal.

Uso

x in [y]

¿Está x en la matriz y?

Examplos

El siguiente ejemplo es aplicable tanto para sintaxis Basic como para sintaxis Crystal:

Rem Basic syntax
{customer.REGION} In Array ("CA", "HI", "AK")

//Crystal syntax
{cliente.REGION} in ["CA", "HI", "AK"]

¿Está el valor de campo {cliente.REGION} en la matriz de abreviaturas de estados de la lista entre paréntesis?

Los siguientes ejemplos son aplicables para sintaxis Crystal:

{archivo.COLOR} in ["Rojo", "Blanco", "Azul"]

¿Está el valor de campo {archivo.COLOR} en la matriz de colores de la lista entre paréntesis?

DayofWeek({archivo.FECHA}) in [2, 4, 6]

¿Está el valor de campo {archivo.FECHA}, transformado a un número que represente DayOfWeek(x)en la matriz de números de lista entre paréntesis ? (domingo = 1, Sábado = 7)
