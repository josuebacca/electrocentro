Ficha Filtro/Página (OLAP)

La ficha Filtro/Página le permite elegir qué hacer con otras dimensiones que no sean filas ni columnas de la cuadrícula. La lista Filtro muestra todas las dimensiones que no usó en la ficha Filas/Columnas. La ficha Filtro/Página le permite especificar cómo las dimensiones que no aparecen en la cuadrícula afectarán a los datos de las dimensiones que sí aparecen en la cuadrícula.

Filtro

Elija un campo de cada una de las dimensiones. Las dimensiones para estos campos se establecen en la cuadrícula.

Mover (< >)

Haga clic en estos botones para mover las dimensiones seleccionadas de la lista Filtro a la lista Página y viceversa.

Orden

Haga clic en los botones de flecha hacia arriba y hacia abajo para cambiar el orden de las dimensiones de filtro y de página.

Página

Elija varios campos de cada una de las dimensiones. Se creará una cuadrícula separada para cada campo.

Seleccionar

Use este botón para seleccionar los campos que desea que se muestren para una dimensión de filtro o de página.

Cuando hace clic en este botón, aparece el cuadro de diálogo Seleccione un campo para la dimensión.

El campo que selecciona para una dimensión de filtro aparece a la derecha de esa dimensión.
