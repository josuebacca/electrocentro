Left (cadena, longitud)

Sintaxis básica y Crystal.

Argumentos
  • cadena es un valor de cadena de texto.
  • longitud es un número entero que indica el número de caracteres a ser extraídos de cadena.
Devuelve

Una cadena de texto.

Acción

Extrae el número específico de caracteres del lado izquierdo de la cadena.

Uso(s) típico(s)

Utilice esta función para obtener un cierto número de caracteres desde la extrema izquierda de una cadena de texto. Por ejemplo, puede utilizar la función Izquierda para obtener sólo el código de área de los valores dentro de un campo que contiene números telefónicos.

Ejemplo(s)

Los siguientes ejemplos son aplicables para la sintaxis Crystal:

Left("abcdefg", 4)

Devuelve "abcd".

Left({cliente.FAX}, 3)

Recupera el código de área de cada número de fax cuando los tres primeros caracteres del campo Fax contienen el código de área.

Comentario(s)

Esta función está diseñada para trabajar exactamente como la función de Visual Basic del mismo nombre.
