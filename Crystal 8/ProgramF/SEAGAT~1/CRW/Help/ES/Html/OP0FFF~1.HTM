literal fecha -hora (#...#)

Sintaxis Basic y Crystal.

Uso

# cadena alfanumérica #

Este operador convierte la cadena alfanumérica contenida en un valor FechaHora. La alfanumérica debe contener una fecha, una hora o una representación de fecha y hora que pueda ser comúnmente reconocida. Se admiten muchos formatos.

Ejemplos

Los siguientes ejemplos son aplicables tanto a sintaxis Basic como Crystal:

#10:10am#

Devuelve un valor FechaHora de 10:10:00am.

#Oct. 19, 1999 #

Devuelve un valor FechaHora de 19 de octubre de 1999 a las 12:00:00am.

#Oct. 19, 1999 10:10am#

Devuelve un valor FechaHora de 19 de octubre de 1999 a las 10:00:00am.

Comentarios

Si una fórmula devuelve un valor FechaHora, el formato del valor FechaHora devuelto es determinado por el formato de FechaHora aplicable a ese campo de fórmula.
