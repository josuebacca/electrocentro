PeopleSoft

PeopleSoft es un proveedor de soluciones de aplicaciones comerciales que ofrece el sistema ERP de PeopleSoft (planificación de recursos comerciales). Seagate Crystal Reports trabaja con datos de ERP de PeopleSoft usando ODBC.

Nota:    Se debe instalar y configurar correctamente el cliente PeopleSoft para usar el archivo de traducción ODBC (P2sodbc.dll) de Seagate Crystal Reports.
