Examplos

El siguiente ejemplo es aplicable a sintaxis Crystal:

If {file.CREDIT LIMIT} = 5000 or {file.SALESMAN} = "SP"
Then
    {file.AMOUNT}
Else
    0;

Esto significa que si el límite de crédito es 5000, o    , si el vendedor es SP, (cualquiera de las condiciones es VERDADERA), entonces retornar el valor en el campo t {archivo.CANTIDAD}, de lo contrario devuelva nada.

Los siguientes ejemplos son aplicables a sintaxis Basic y Crystal:

A > B or B > C

VERDADERO, donde A = 10, B = 6, y C = 3 (ambas condiciones VERDADERO).

A > B or B > C

VERDADERO, donde A=10, B=6, y C=7 (una de las dos condiciones VERDADERO).

A > B or B > C

FALSO, donde A=5, B=6, y C=7 (ninguna de las dos condiciones VERDADERO).

(A>B) or (A * C - D > E) or (E / D<= B)

VERDADERO, donde A = 5, B = 5, C = 3, D = 2, E = 12 (Por lo menos una de las tres condiciones es VERDADERO. Sólo en este caso (A * C - D > E) es VERDADERO).
