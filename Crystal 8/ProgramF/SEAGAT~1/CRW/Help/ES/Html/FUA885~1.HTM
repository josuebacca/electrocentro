Sgn (número)

Sintaxis básica y Crystal.

Argumento

Valor numérico

Devuelve

Devuelve 1, 0 ó -1

Acción

Sgn devuelve el signo de un número específico: 1 si el número es > 0, 0 si el número es 0 y -1 si el número es < 0.

Ejemplo(s)

Los siguientes ejemplos son aplicables a las sintaxis básica y Crystal:

Sgn (-10)

Devuelve -1.

Sgn (0)

Devuelve 0.

Sgn (10)

Devuelve 1.

Comentario

Esta función está diseñada para trabajar exactamente como la función de Visual Basic del mismo nombre.
