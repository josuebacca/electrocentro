FRReturnOnNetFixedAssets (beneficioNeto, activosFijosNetos)

Sintaxis básica y Crystal.

Formato

FRReturnOnNetFixedAssets (beneficioNeto, activosFijosNetos)

Argumentos
  • beneficioNeto es un número o fórmula. beneficioNeto es la ganancia de la compañía durante el año, de la venta de sus productos o servicios menos los gastos incurridos. Es la figura que aparece al final del estado de pérdidas y ganancias. Algunos analistas podrían usar las ganancias netas antes de algunos gastos extraordinarios y/o impuestos para calcular esta relación.
  • activosFijosNetos es un número o fórmula. activosFijosNetos es igual al total de los activos fijos menos la depreciación acumulada en la hoja de balance de la compañía.
Acción

FRReturnOnNetFixedAssets devuelve la relación entre beneficioNeto y activosFijosNetos (por ejemplo, ReturnOnNetFixedAssets = beneficioNeto / activosFijosNetos).

Uso(s) típico(s)

Esta relación se usa para medir el rendimiento del capital en activos. Normalmente se usa para medir la eficiencia de compañías manufactureras o de servicio que requieren activos de capital para proveer sus servicios (líneas aéreas, etc.). Cuanto más alta es la relación, más alto es el rendimiento.

Ejemplo(s)

Los siguientes ejemplos son aplicables a las sintaxis básica y Crystal:

FRReturnOnNetFixedAssets(34, 50)

Devuelve 0.68.

FRReturnOnNetFixedAssets(50.4, 42)

Devuelve 1.2.

Nota:    Ninguno de los argumentos de entrada pueden ser negativos; debe revisar si hay valores negativos antes de pasar los argumentos de entrada.

Comentario

FRReturnOnNetFixedAssets es una de las muchas herramientas para análisis financiero que se usan para interpretar la posición financiera de una compañía. Como en el caso de todas las relaciones, se debe usar con precaución. Se puede usar sólo como una guía y no como prueba firme para formar una opinión sobre el estado financiero de una compañía.

Ninguna de las relaciones se puede usar exclusivamente para hacer análisis financieros y no hay reglas fijas sobre los resultados de las relaciones. Aparte de las diferencias entre los tipos de industrias, el resultado varía entre las compañías diferentes dentro de la misma industria y dentro de períodos de cuenta diferentes. Los resultados sólo se deben usar para hacer comparaciones relativas y análisis de tendencias que es preferible a tratarlos como algo absoluto.
