SummaryFunction (campo)

Todas las funciones Summary se utilizan para resumir datos de campos de varias formas diferentes. Debido a sus características únicas de resumen, se han agrupado como funciones de resumen. El nombre de la función SummaryFunction reemplaza el nombre específico de una función.

Las funciones de resumen pueden diseñarse para realizar operaciones en datos de grupo.
