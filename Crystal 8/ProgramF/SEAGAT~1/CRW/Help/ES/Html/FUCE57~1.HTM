Chr (x)

Sintaxis básica y Crystal.

Argumentos

x es un número entero, específicamente, es cualquier valor ASCII.

Devuelve

Cadena de texto (un carácter de longitud)

Acción

La función (x) devuelve un carácter asociado con el valor ASCII pasado como x.

Uso(s) típico(s)

Utilice esta función siempre que necesite obtener un carácter asociado con un valor ASCII particular. Esta función es muy útil para programadores de aplicaciones que usen Crystal Reports Report Engine.

Ejemplo(s)

El siguiente ejemplo es aplicable a las sintaxis básica y Crystal:

Chr(65)

Devuelve "A".

Comentario(s)

Esta función está diseñada para trabajar exactamente como la función de Visual Basic del mismo nombre.
