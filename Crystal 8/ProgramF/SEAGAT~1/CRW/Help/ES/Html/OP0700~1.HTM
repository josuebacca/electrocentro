Declaradores de variables para sintaxis Basic

Los declaradores de variables     permiten definir una variable dando un nombre a la variable. Opcionalmente, también puede asignar un valor a una variable cuando la declara usando el operador de asignación. Los siguientes son declaradores de variables en sintaxis Basic.

Uso
  • Dim xt
  • Dim xt
  • Dim x As Boolean
  • Dim x As Number
  • Dim x As Currency
  • Dim x As Date
  • Dim x As Time
  • Dim x As DateTime
  • Dim x As String
  • Dim x As Number Range
  • Dim x As Currency Range
  • Dim x As Date Range
  • Dim x As Time Range
  • Dim x As DateTime Range
  • Dim x As String Range
  • Dim x () As Boolean
  • Dim x () As Number
  • Dim x () As Currency
  • Dim x () As Date
  • Dim x () As Time
  • Dim x () As DateTime
  • Dim x () As String
  • Dim x () As Number Range
  • Dim x () As Currency Range
  • Dim x () As Date Range
  • Dim x () As Time Range
  • Dim x () As DateTime Range
  • Dim x () As String Range.

    Declara una variable x que pueden contener datos de un tipo correspondiente al tipo de declarador usado.

Comentarios
  •     Un declarador de variable debe ser usado para declarar una variable antes que la variable pueda ser usada en una fórmula.
  • Dim especifica una variable con alcance local. Puede usar las palabras clave Local, Global y Shared en lugar de Dim para especificar diferentes alcances. Local y Dim son palabras clave equivalentes. Vea Alcance de las variables y Alcances de variable para sintaxis Basic.
