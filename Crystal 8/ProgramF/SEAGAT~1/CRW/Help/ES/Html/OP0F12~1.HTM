Operador de Asignación

Sintaxis Basic (x = y)

Puede usar el operador de asignación (=) y la palabra clave de asignación "Let" (Let x=y) sólo con sintaxis Basic. La sintaxis Crystal usa el otro operador de asignación (:=).

Uso

x = y

Asigna y a la variable x. x debe haber sido declarada antes de la asignación. y debe ser del mismo tipo que x, o de un tipo compatible con x.

Let x = y

Asigna y a la variable x. x debe haber sido declarada antes de la asignación. y debe ser del mismo tipo que x, o de un tipo compatible con x.

Examplos

El siguiente ejemplo es aplicable a la sintaxis Basic:

Dim n As Number
Dim b As Boolean
Dim c As Currency
Dim d As Date
Dim t As Time
Dim s As String
Dim matrizNum () As Number
'a n se le asigna 100
n = 100
'a b se le asigna el valor booleano verdadero
b = True
'a c se le asigna el valor monetario 5000
c = CCur (5000)
'a d se le asigna el valor de fecha de 10 de octubre de 1999
d = CDate ("10, 10, 1999")
'a t se le asigna el valor de hora 00:50
t = CTime ("12:50am")

'a s se le asigna una cadena que es una concatenación de 2 subcadenas y un valor de campo
s = "Las ventas totales ascienden a " & {Pedidos.Monto del pedido} & " millones."

'a matrizNum se le asigna una matriz de valores numéricos
matrizNum = Array (10, 20, 30)

formula = s

Comentarios
  • Una variable debe declararse en un enunciado separado antes de asignarle ningún valor en sintaxis Basic.
  • La asignación sólo asigna un valor a una variable. Cuando una fórmula devuelve, el valor asignado a la variable fórmula se formatea de acuerdo al formato especificado para ese campo de fórmula del tipo correspondiente.

Crystal syntax (x := y)

Puede usar el operador de asignación (:=) sólo con sintaxis Crystal. Para obtener más información, vea Declaraciones de variables.

Uso

x := n

Asigna el valor n a la variable x. (x tiene que haber sido declarado en la misma fórmula.)

Examplos

Los siguientes ejemplos son aplicables a sintaxis Crystal:

Cantidad:= 0

Inicializa la variable llamada Cantidad.

Cantidad:= 100

Asigna el valor 100 a la variable llamada Cantidad.

Cantidad:= Cantidad + {archivo.CNTD}

Asigna el resultado de un cálculo a la variable llamada Cantidad. El cálculo suma el valor de la cantidad del campo ({archivo.CNTD}) al valor vigente de la variable Cantidad.

Cantidad:= {archivo.CNTD1} + {archivo.CNTD2} + {archivo.CNTD3}

Suma el total de los tres campos de cantidad y asigna el total a la variable llamada Cantidad

Cliente:= "Westside Motors"

Asigna la Cadena "Westside Motors" a la variable llamada Cliente.

Cliente:= {cliente.NOMBRE} + {cliente.APELLIDO}

Concatena dos campos y asigna el valor concatenado de ambos campos a la variable llamada Cliente.

Cliente:= TrimRight({cliente.NOMBRE}) + {cliente.APELLIDO}

Recorta los espacios en blanco del campo del nombre propio({cliente.NOMBRE}), concatena el campo de apellidos({cliente.APELLIDO}) y asigna el valor concatenado a ambos campos de la variable llamada Cliente.

Cliente:= "Mr. " + {cliente.APELLIDO}

Concatena la Cadena "Mr. " con el valor de campo de apellidos{cliente.APELLIDO}, y asigna el valor concatenado a la variable llamada Cliente.

Cantidad:= 100; Cliente:= "Westside Motors"

Asigna la constante 100 a la variable de número llamada Cantidad, y asigna la cadena "Westside Motors" a la variable de cadena llamada Cliente. Se puede asignar valores a las variables múltiples, separando los enunciados de asignación     con puntos y coma.
