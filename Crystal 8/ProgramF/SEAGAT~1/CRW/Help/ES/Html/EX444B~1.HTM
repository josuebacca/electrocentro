Vista de la opción En rangos

Un mapa en rangos desglosa los datos en rangos, asigna un color específico a cada rango y luego codifica las áreas geográficas del mapa por color con el fin de mostrar el rango. Puede usar un mapa en rangos para mostrar cifras de ventas en un rango de intervalos iguales. Puede entonces usar tonos de colores para codificar regiones de ventas de acuerdo a dichas cifras. Este tipo de mapa es útil para determinar regiones con las ventas más altas simplemente mirando el color que exhiben.

Número de intervalos

Use este cuadro de texto para ingresar el número de intervalos que desea usar como rango. El mapa y la leyenda cambian de acuerdo a estos valores. Puede ingresar un número entre 1 y 16. El número predeterminado es 5.

Método de distribución

Use esta lista para asignar intervalos de rango de forma tal que los valores calcen con uno de los criterios disponibles:

  • La cuenta es igual

    Asigna intervalos de forma que el mismo número de regiones (o un número lo más cercano posible de regiones) aparezca en cada intervalo.

  • Los rangos son iguales

    Asigna intervalos de modo que los valores de resumen en cada intervalo sean iguales numéricamente.

  • Interrupción natural

    Asigna intervalos usando un algoritmo que intenta minimizar la diferencia entre los valores de resumen y el promedio de los valores de resumen para cada intervalo.

  • Desviación estándar

    Asigna intervalos de modo que el intervalo intermedio se interrumpa en la media (o promedio) de los valores de datos y los intervalos superiores o inferiores al rango intermedio sean un estándar de desviación superior o inferior a la media. La desviación estándar es la raíz cuadrada de la varianza. Esta opción proyecta cuánto se desvían diversos valores en un conjunto de valores de la media del conjunto.

Color del intervalo más alto

Use esta lista para seleccionar el color que desea usar para el intervalo más alto (más grande) en el rango. Este color se transforma en el extremo superior del rango de color.

Color del intervalo más bajo

Use esta lista para seleccionar el color que desea usar para el intervalo más bajo (más pequeño) en el rango. Este color se transforma en el extremo inferior del rango de color.

Permitir intervalos vacíos

Seleccione esta casilla de verificación si desea ver intervalos vacíos del mapa en rangos. Esta opción se selecciona en forma predeterminada.
