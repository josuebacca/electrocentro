Ficha Ejes

La ficha Ejes le permite establecer las propiedades de cada uno de los ejes del gráfico. La ficha Ejes cambia dependiendo del tipo de gráfico con el que trabaje.

Mayor

Use las casillas de verificación Mayor para determinar si se van a mostrar líneas de división mayores para los ejes del gráfico. Existe una casilla de verificación mayor para el eje de grupo, otra para el eje de serie, otra para el eje de datos y otra para el eje de datos 2. Estas casillas de verificación sólo se activan si el gráfico es de tabla cruzada u OLAP.

  • Seleccione una casilla de verificación mayor para mostrar la línea de división del eje correspondiente.
  • Desactive una casilla de verificación mayor para ocultar la línea de división del eje correspondiente.
Menor

Use las casillas de verificación Menor para determinar si se van a mostrar líneas de división mayores para los ejes del gráfico. Existe una casilla de verificación menor para el eje de grupo, otra para el eje de serie, otra para el eje de datos y otra para el eje de datos 2. Estas casillas de verificación sólo se activan si el gráfico es de tabla cruzada u OLAP.

  • Seleccione una casilla de verificación mayor para mostrar la línea de división del eje correspondiente.
  • Desactive una casilla de verificación mayor para ocultar la línea de división del eje correspondiente.
Escala automática

Cuando selecciona una de las casillas de verificación Escala automática, el número de etiquetas del eje de datos se ajusta para garantizar una visualización clara.

Rango automático

Cuando selecciona una de las casillas de verificación Rango automático, se asigna un rango predeterminado a los valores de datos del eje correspondiente.

Menor

Use los cuadros de texto Menor para ingresar el mínimo rango para los valores de datos en los ejes correspondientes. Existe un cuadro de texto de rango mínimo para el eje de datos y otro para el eje de datos 2. El segundo cuadro de texto de rango mínimo sólo se activa si el gráfico es de tabla cruzada u OLAP.

Si se selecciona la casilla de verificación Rango automático, el cuadro de texto de rango mínimo queda inactivo.

Máx.

Use los cuadros de texto Máx. para ingresar el máximo rango para los valores de datos en los ejes correspondientes. Existe un cuadro de texto de rango máximo para el eje de datos y otro para el eje de datos 2. El segundo cuadro de texto de rango máximo sólo se activa si el gráfico es de tabla cruzada u OLAP.

Si se selecciona la casilla de verificación Rango automático, el cuadro de texto de rango máximo queda inactivo.

Formato de número

Use las listas Formato de número para seleccionar un formato numérico para los valores de datos en el eje correspondiente. Existe una lista Formato de número para el eje de datos (y) y otra para el eje de datos 2. La segunda lista Formato de número sólo se activa si el gráfico es de tabla cruzada u OLAP.

Si se selecciona la casilla de verificación Rango automático correspondiente, la lista Formato de número queda inactiva.

Automática

Si hace clic en uno de los botones de la opción Automática, el programa establece de forma automática el número de divisiones para el eje correspondiente del gráfico. Existe una opción automática para el eje de datos y otra para el eje de datos 2. El segundo botón de la opción Automática sólo se activa si el gráfico es de tabla cruzada u OLAP.

Manual

Use los botones de la opción Manual para establecer manualmente el número de divisiones para los ejes correspondientes. Existe una opción manual para el eje de datos y otra para el eje de datos 2. El segundo botón de la opción Manual sólo se activa si el gráfico es de tabla cruzada u OLAP.

Cuando hace clic en uno de los botones de la opción Manual, el cuadro de texto adyacente se activa. Use el cuadro de texto para ingresar el número de divisiones.
