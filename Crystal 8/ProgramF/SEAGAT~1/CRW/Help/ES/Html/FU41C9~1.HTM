NextValue (Campo)

Sintaxis básica

NextValue y Next (Campo) son funciones equivalentes. Sin embargo, sólo puede usar NextValue con sintaxis básica y Next sólo con sintaxis Crystal.

Argumento

campo es el nombre del campo de base de datos.

Devuelve

Valor del campo especificado; el tipo depende del tipo de ese campo.

Acción

NextValue devuelve el valor del siguiente registro del campo especificado.

Uso típico

Puede utilizar NextValue para identificar el primero o el último dato dentro de un rango, para identificar el punto inicial para un nuevo grupo, o para evaluar la existencia de valores duplicados.

Ejemplo(s)

Los siguientes ejemplos son aplicables a la sintaxis básica:

If NextValue ({DetallePedidos.Cantidad}) <> 0 Then
    formula = {DetallePedidos.Cantidad} / 2
Else
    formula = {DetallePedidos.Cantidad}
End If

Evalúa el valor siguiente en el campo {Orders Detail.Quantity} para ver si es un valor cero. Si no es, divide el valor entre dos. Si es un valor cero, devuelve el valor.

If NextValue ({Pedidos.IDPedido}) Mod 10 = 0 Then
    formula = "Fin de bloque"
Else
    formula = "Más por venir"
End If

Esta fórmula divide los pedidos en bloques de 10. Si la siguiente ID de Pedido es un múltiplo de 10, marca el valor actual como "Fin de bloque". De lo contrario, imprime "Más por venir".

Nota:    El uso de esta función en una fórmula obliga a que la fórmula sea evaluada al momento de impresión. Para obtener más información sobre las consideraciones de evaluación de hora, véase el Funciones de tiempo de evaluación.
