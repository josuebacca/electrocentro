Remainder (numerador, denominador)

Sintaxis básica y Crystal.

Argumentos
  • Numerador es un valor fraccionario.
  • Denominador es un valor fraccionario.
Devuelve

Un valor fraccionario.

Acción

Remainder devuelve el resto después de que el numerador (dividendo) ha sido dividido entre el denominador (divisor). En una situación de división típica, Crystal Reports expresa un cociente como un número entero (si hay) y hasta seis lugares decimales. Sin embargo, cuando utiliza Remainder, Crystal Reports realiza la división internamente, determina el cociente del número entero y el resto, y devuelve sólo el resto.

Uso(s) típico(s)

Puede utilizar esta función para hacer conversiones (pies a millas, onzas a litros, etc.) Puede también seleccionar cada elemento n de una matriz.

Ejemplo(s)

Los siguientes ejemplos son aplicables a las sintaxis básica y Crystal:

Remainder(12,5)

Devuelve 2.

Rmainder(16,5)

Devuelve 1.

Sintaxis básica Rem
If Remainder ({archivo.Nº DE EXAMEN}, 7) = 0 Then
    formula = "*****"
End If

Sintaxis Crystal
If Remainder({archivo.Nº DE EXAMEN}, 7) = 0 Then
    "*****"
Else
    ""

Pone indicadores en cada séptimo examen para calificarlo en segunda línea compartida.

ToText(Truncate({archivo.DÍAS}/7)) + " semana(s), " + ToText(Remainder({archivo.DÍAS},7)) + " día(s)"

Devuelve "9 semana(s), 1 día(s)". Convierte días a semanas y días. Por ejemplo, si el campo tenía un valor de 64 días, la fórmula devolvería el valor de "9 semana(s), 1 día(s)."
