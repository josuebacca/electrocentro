Operador Menor que

Sintaxis Basic y Crystal.

Usage

x < y

x es menor que y

El operador menor que le dice al Editor de fórmula que evalúe una expresión (x<y) y retorne un VERDADERO (si x es menor que     y) o FALSO (si x es igual o mayor que y).

Examplos

Los siguientes ejemplos son aplicables a sintaxis Basic y Crystal:

{archivo.PESO} < 200

VERDADERO, donde {archivo.PESO} = 150, o {archivo.PESO} = 199.

{archivo.PESO} < 200

FALSO, donde {archivo.PESO} = 200 o {archivo.PESO}= 400.

{archivo.COSTO} < {archivo.PRECIO}

VERDADERO, donde {archivo.COSTO}=350, {archivo.PRECIO} = 400.

{archivo.COSTO} < {archivo.PRECIO}

FALSO, donde {archivo.COSTO} = 350 y {archivo.PRECIO} = 350, o donde {archivo.COSTO} = 350 y {archivo.PRECIO}= 325.
