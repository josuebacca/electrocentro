ReplicateString (cadena, #copias)

Sintaxis básica y Crystal.

Argumentos
  • cadena es una cadena de texto a ser repetida.
  • #copias es un número entero que indica el número de veces que cadena va a ser repetida.
Devuelve

Una cadena de texto.

Acción

Repite la cadena el número de veces especificado por el # de copias.

Uso(s) típico(s)

Puede utilizar esta función para insertar una línea de caracteres siempre que se necesiten. Algunos usos típicos son:

  • para poner indicadores en valores críticos en su informes
  • para construir gráficas de barras simples
  • para dividir su informe en secciones visibles
  • para resaltar totales, subtotales, y otros resúmenes de datos.
Ejemplo(s)

Los siguientes ejemplos son aplicables para la sintaxis Crystal:

If {archivo.VENTAS} < {archivo.COTIZ} Then
    ReplicateString("*",10)
Else
    ""

Imprime el carácter * diez veces como un indicador.

{archivo.NOMBRE}+ " " + ReplicateString("*", {archivo.MARCADOR})

Imprime una gráfica de barras simple que muestra los resultados de prueba. La fórmula imprime un asterisco para cada punto en un marcador de prueba (el valor del campo archivo.Marcador).

La fórmula imprime un asterisco por cada punto en un marcador de prueba (el valor del campo {archivo.MARCADOR}). Por ejemplo, dando los siguientes datos:
