Asc (cadena)

Sintaxis básica y Crystal.

Argumentos

cadena es una cadena de texto.

Devuelve

Número entero (el valor ASCII del primer carácter en una cadena).

Acción

La función Asc (cadena) devuelve el código ASCII del primer carácter de la cadena.

Uso(s) típico(s)

Utilice esta función siempre que necesite obtener el valor ASCII de un carácter. Esta función es la más útil para programadores que utilizan Report Engine.

Ejemplo(s)

El siguiente ejemplo es aplicable a las sintaxis básica y Crystal:

Asc("A")

Devuelve 65.

Comentario(s)

Esta función está diseñada para trabajar exactamente como la función de Visual Basic del mismo nombre.
