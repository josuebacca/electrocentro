Ficha Datos

La ficha Datos aparece en el Asistente de informes cuando hace clic en cualquiera de los botones Asistente de la Galería de Seagate Crystal Report al crear un nuevo informe.

Use la ficha Datos para seleccionar la fuente de datos y las tablas (bases de datos en terminología Essbase) y agregar índices a las tablas del nuevo informe.

Fuentes de datos

Las fuentes de datos disponibles se muestran como botones. Para seleccionar una fuente de datos, haga clic en el botón correspondiente.

Tablas

Una vez que ha elegido la fuente de datos deseada, aquí se mostrarán las tablas (bases de datos en terminología Essbase).

Quitar

Haga clic en este botón para eliminar una tabla (base de datos en terminología Essbase) seleccionada en la lista Tablas.

Agregar índice

Haga clic en este botón para agregar un índice a una base de datos tipo PC (archivo de datos). Los índices se usan para vincular.
