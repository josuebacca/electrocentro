WhileReadingRecords

Sintaxis básica y Crystal.

Devuelve

Nada

Acción

Obliga al programa a evaluar la fórmula mientras está leyendo datos de registros de bases de datos.

Uso(s) típico(s)

Las fórmulas son normalmente evaluadas en los siguientes momentos  :

  • Si ningún campo de grupo o de base de datos está incluido en la fórmula, la fórmula es evaluada antes del que el programa lea los registros de la base de datos.
  • Si una base de datos está incluida en la fórmula, la fórmula es evaluada mientras que el programa lee los registros de base de datos.
  • Si un campo de grupo, campo de página #, subtotal, etc. está incluida en la fórmula, la fórmula es evaluada después de que los registros de la base de datos son leídos y mientras los datos de los registros están siendo impresos en el informe.

WhileReadingRecords obliga a que la fórmula sea evaluada mientras que el programa lee los registros de base de datos. Cuando está función es utilizada en una fórmula, el Verificador de Fórmula devuelve un mensaje de error si intenta incluir elementos en la fórmula (grupos, campos de número de página, etc.) que deben de ser evaluados más tarde (mientras se imprimen los registros). Esta función también puede ser utilizada para obligar a la fórmula que no incluye campos de base de datos ni campos de grupo para procesar mientras lee los registros en lugar de antes de leer los registros.

Ejemplo(s)

Los siguientes ejemplos son aplicables para la sintaxis Crystal:

WhileReadingRecords;
ToNumber({Detalles del pedido.CANTIDAD})

Obliga a la fórmula (la cual contiene un campo de base de datos) a ser evaluada en su tiempo normal (mientras lee los registros).

Nota:    Si intenta incluir un grupo en esta fórmula, usted obtendrá un mensaje de error.

WhileReadingRecords;
ToNumber ("12345")

Obliga a la fórmula (la cual no contiene campos de bases de datos o grupos), para ser evaluada tiempo después de lo que normal. En este caso, causa que la fórmula sea evaluada mientras lee los registros en lugar de antes de leer los registros.

Nota:    Si intenta incluir un grupo en esta fórmula, obtendrá un mensaje de error.
