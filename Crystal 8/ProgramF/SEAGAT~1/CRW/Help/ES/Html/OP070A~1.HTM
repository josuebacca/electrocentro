Is < x

Sintaxis Basic y Crystal.

Is < x es preferido en sintaxis Basic, mientras que Hasta pero no incluido (upTo_ x) es preferido en sintaxis Crystal.

Uso

Is < x

Is < x se usa para especificar un rango de valores menores que el valor x. x puede ser de cualquier tipo: numérico, booleano, monetario, fecha, hora, fechahora y cadena.

Ejemplos

El siguiente ejemplo es aplicable para sintaxis Basic:

Dim FechasRestricción() As Date Range
Rem El período de restricción es cualquier día anterior a 1999 o cualquier fecha de marzo de 1999
FechasRestricción = Array (Is < CDate(1999, 1, 1), _
                       CDate(1999, 3, 1) To CDate (1999, 3, 31))
If CurrentDate In FechasRestricción Then
    formula = "Restricción"
Else
    formula = "Libre comercio"
End If

Si la fecha de hoy es 30 de diciembre de 1998, entonces la fórmula devuelve "Restricción" dado que está dentro del período de restricción anterior a 1999.

Si la fecha de hoy es 1 de enero de 1999, entonces la fórmula devuelve "Libre comercio" dado que no está dentro de ningún período de restricción.

Si la fecha de hoy es 31 de marzo de 1999, entonces la fórmula devuelve "Restricción" dado que está dentro del mes de restricción de marzo.

Si la fecha de hoy es 30 de octubre de 1999, entonces la fórmula devuelve "Libre comercio" dado que no está dentro de ningún período de restricción.
