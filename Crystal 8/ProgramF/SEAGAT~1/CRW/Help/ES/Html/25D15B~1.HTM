SAP

SAP es un proveedor de soluciones de aplicaciones comerciales cliente/servidor que ofrece un paquete de software ERP (planificación de recursos comerciales) que incluye bases de datos SQL sobre las que puede generar informes con Seagate Crystal Reports.

Seagate Crystal Reports ofrece acceso a las bases de datos creadas por ERP  de SAP. Los archivos P2srfc.dll y P2srfcaq.dll instalados con Seagate Crystal Reports traducen las solicitudes a la capa de conectividad de la red del servidor de aplicación SAP. Este servidor interpreta la solicitud y devuelve datos de la base de datos de SAP.

Nota:    Seagate Crystal Reports es compatible con el acceso a la base de datos de SAP mediante los archivos librfc32.dll y vrf32.dll.
