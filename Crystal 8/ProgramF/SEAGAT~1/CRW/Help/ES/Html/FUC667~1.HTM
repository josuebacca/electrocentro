Filter

Sintaxis básica y Crystal.

Sobrecargas
  • Filter (cadenasEntrada, cadenaBúsqueda)
  • Filter (cadenasEntrada, cadenabusqueda, incluir)
  • Filter (cadenasEntrada, cadenaBúsqueda, incluir, comparar)
Argumentos
  • cadenasEntrada es una matriz de cadenas dentro de la que debe realizarse la búsqueda.
  • cadenaBúsqueda es una cadena que debe buscarse.
  • incluir es un valor booleano opcional que indica si deben devolverse subcadenas que incluyan o excluyan cadenaBúsqueda. Si incluir es Verdadero, Filter devuelve el subconjunto de la matriz que contiene cadenaBúsqueda como subcadena. Si incluir es Falso, Filter devuelve el subconjunto de la matriz que contiene cadenaBúsqueda como subcadena. Si se omite, se supone el valor Verdadero.
  • comparar es un valor Numérico opcional que indica el tipo de cadena de comparación a usar:
      • Realiza una comparación, que es sensible a mayúsculas y minúsculas
      • Realiza una comparación, que es sensible a mayúsculas y minúsculas

        Si se omite, se realiza una comparación sensible a mayúsculas y minúsculas.

Devuelve

Matriz de valores de Cadena.

Acción

Filter busca una cadena especificada dentro de una matriz de cadenas y devuelve las cadenas en una matriz.

Ejemplo(s)

Los siguientes ejemplos son aplicables a las sintaxis básica y Crystal:

Los ejemplos suponen que cadenasEntrada es una matriz de Cadenas compuesta de 5 elementos: "ananá", "Manzana", "MANZANA", "uva" y "naranja".

Filter (cadenasEntrada, "ap")

Devuelve una matriz que contiene las 2 cadenas "ananá" y "uva".

Filter (cadenasEntrada, "Ap", Verdadero)

Devuelve una matriz que contiene 1 cadena "Manzana".

Filter (cadenasEntrada, "ap", Verdadero)

Devuelve una matriz que contiene las 4 cadenas "ananá", "Manzana", "MANZANA" y "uva". El proceso de concordancia con la cadena de búsqueda usa comparación sensible a mayúsculas y minúsculas.

Filter (cadenasEntrada, "ap", Falso)

Devuelve una matriz que contiene las 3 cadenas "Manzana", "MANZANA" y "naranja".

Filter (cadenasEntrada, "cd")

Devuelve una matriz que contiene 1 cadena: la cadena vacía "". La matriz devuelta no contiene ninguna cadena coincidente, pero sí 1 elemento, ya que el lenguaje de la fórmula no admite matrices vacías.

Comentario(s)

Esta función está diseñada para trabajar exactamente como la función de Visual Basic del mismo nombre.
