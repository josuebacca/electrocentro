Declaradores de variables para sintaxis Crystal

Los declaradores de variables le permiten definir una variable asignándole un nombre. Opcionalmente, puede también asignar un valor a una variable cuando la declare al usar el operador de asignación. Los siguientes son declaradores de variables en sintaxis Crystal.

Uso
  • Local BooleanVar x
  • Local NumberVar x
  • Local CurrencyVar x
  • Local DateVar x
  • Local TimeVar x
  • Local DateTimeVar x
  • Local StringVar x
  • Local NumberVar range x
  • Local CurrencyVar range x
  • Local DateVar range x
  • Local TimeVar range x
  • Local DateTimeVar range x
  • Local StringVar range x
  • Local BooleanVar array x
  • Local NumberVar array x
  • Local CurrencyVar array x
  • Local DateVar array x
  • Local TimeVar array x
  • Local DateTimeVar array x
  • Local StringVar array x
  • Local NumberVar range array x
  • Local CurrencyVar range array x
  • Local DateVar range array x
  • Local TimeVar range array x
  • Local DateTimeVar range array x
  • Local StringVar range array x.

    Declara una variable local x que puede contener datos de un tipo correspondiente al declarador de variable usado.

    Las variables locales mantienen su valor sólo para la evaluación de la fórmula en la que ocurren.

Comentarios
  • Un declarador de variable debe usarse para declarar una variable antes de que la variable pueda usarse en una fórmula.
  • Local especifica una variable con alcance local. Puede usar las palabras clave Local, Global y Shared en lugar de Local para especificar diferentes alcances. También puede omitir la palabra clave de alcance y comenzar la declaración de la variable con el nombre del tipo. Esto es como declarar una variable Global. Vea Alcance de las variables y Alcances de variable para sintaxis Crystal.
