StrCmp (cadena1, cadena2), StrCmp (cadena1, cadena2, comparar)

Sintaxis básica y Crystal.

Argumentos
  • cadena1 es la primer cadena de caracteres a ser comparada.
  • cadena2 es la segunda cadena de caracteres a ser comparada.
  • comparar es un valor Numérico opcional que indica el tipo de cadena de comparación a usar:
      • Realiza una comparación, que es sensible a mayúsculas y minúsculas
      • Realiza una comparación, que es sensible a mayúsculas y minúsculas

        Si se omite, se realiza una comparación sensible a mayúsculas y minúsculas.

Devuelve

Un número entero donde  :

  • -1 = Menor Que
  • 0 = Igual A
  • 1 = Mayor Que
Acción

La función StrCmp compara dos cadenas.

Uso(s) típico(s)

Utilice esta función para determinar si dos valores de cadena son idénticos.

Ejemplo(s)

Los siguientes ejemplos son aplicables a las sintaxis básica y Crystal:

StrCmp("abcd", "aa")

Devuelve = 1.

StrCmp("abcd", "aa", 1)

Devuelve = 1.

StrCmp ("abcd", "ac")

Devuelve = -1.

StrCmp ("aa", "aa")

Devuelve = 0.

Comentario(s)

Esta función está diseñada para trabajar exactamente como la función de Visual Basic del mismo nombre.
