Ficha Tabla cruzada

Use la ficha Tabla cruzada para crear y dar formato a un objeto de tabla cruzada para el informe.

Esta ficha tiene la misma funcionalidad que el cuadro de diálogo Opciones de Grupo de Tabla cruzada.

Columnas

Esta lista muestra los campos que ha seleccionado como encabezados de columna para la tabla cruzada. Si tiene varios encabezados de columna, éstos aparecerán en su informe en el orden en el que aparecen en esta lista.

Para agregar un encabezado de columna a esta lista, seleccione un campo de la lista Campos disponibles y luego arrástrelo a la lista o bien haga clic en el botón Agregar columna.

Filas

Esta lista muestra los campos que ha seleccionado como encabezados de filas para la tabla cruzada. Si tiene varios encabezados de fila, éstos aparecerán en su informe en el orden en que aparecen en esta lista.

Para agregar un encabezado de fila a esta lista, seleccione un campo de la lista Campos disponibles y luego arrástrelo a la lista o haga clic en el botón Agregar fila.

Campos resumidos

Esta lista muestra los campos que ha seleccionado como encabezados de campos resumidos para la tabla cruzada. Si tiene varios encabezados de campos resumidos, éstos aparecerán en su informe en el orden en el que aparecen en esta lista.

Para agregar un encabezado de campo resumido a esta lista, seleccione un campo de la lista Campos disponibles y luego arrástrelo a la lista o haga clic en el botón Agregar campo resumido.

Campos disponibles

Esta lista muestra todas las tablas y campos que ha activado para usarlos en el informe.

Seleccione el campo que desea usar y arrástrelo a la lista apropiada o haga clic en el botón Agregar que sea apropiado. Repita este proceso para agregar campos adicionales.

Agregar fila

Haga clic en este botón para agregar encabezados de fila a la tabla cruzada. Los campos seleccionados en la lista Campos disponibles se desplazan a la lista Filas.

Agregar columna

Haga clic en este botón para agregar encabezados de columna a la tabla cruzada. Los campos seleccionados en la lista Campos disponibles se desplazan a la lista Columnas.

Agregar campo resumido

Haga clic en este botón para agregar campos resumidos a la tabla cruzada. Los campos seleccionados en la lista Campos disponibles se desplazan a la lista Campo resumido.

Quitar

Haga clic en este botón para quitar el encabezado o campo resumido seleccionado de la tabla cruzada.

Opciones de grupo

Cuando hace clic en este botón, aparece el cuadro de diálogo Opciones de Grupo de Tabla cruzada. Use este cuadro de diálogo si desea ingresar un grupo para la tabla cruzada. Las opciones de grupo sólo están disponibles para filas y columnas.

Cambiar resumen

Cuando hace clic en este botón, aparece el cuadro de diálogo Cambiar operación de resumen. Use este cuadro de diálogo para cambiar el tipo de resumen de los campos resumidos de la tabla cruzada. Esta opción sólo está disponible para campos resumidos.

Nueva fórmula

Cuando hace clic en este botón, aparece el cuadro de diálogo Nombre de Fórmula. Ingrese un nombre para la nueva fórmula y luego haga clic en Aceptar. Aparece el Editor de Fórmulas para que pueda crear la nueva fórmula.

Editar fórmula

Cuando hace clic en este botón, aparece el Editor de Fórmulas. Use el Editor para editar la fórmula seleccionada. Esta opción sólo está disponible cuando un campo de fórmula está seleccionado en una de las listas.

Examinar

Use la opción Examinar para ver los valores de un campo seleccionado en las listas Campos disponibles o Campo resumido.
