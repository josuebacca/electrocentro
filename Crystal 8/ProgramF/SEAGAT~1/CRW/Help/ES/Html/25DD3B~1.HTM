Sistema de archivos locales

Puede usar Seagate Crystal Reports para informar sobre archivos de su sistema o unidades de red que incluyan información como nombre de archivo, número de versión, fecha, etc.

Seagate Crystal Reports proporciona el archivo de traducción P2sfs.dll para generar informes sobre su sistema de archivos locales.

Nota:    Debe tener acceso de lectura UNC (convención de nomenclatura universal) a su sistema de archivos locales. UNC es un formato estándar para rutas que incluyen un servidor de archivos de red de área local. La sintaxis básica es: \\Server\Share\ruta\nombre de archivo.
