Seagate Crystal Reports

Al trabajar con datos ODBC, Seagate Crystal Reports genera un enunciado SQL que solicita los datos apropiados de ODBC. El poderoso generador SQL incorporado en Seagate Crystal Reports está diseñado para crear un enunciado SQL que permitirá a la fuente de datos ODBC o al mismo ODBC hacer la mayor parte posible de la generación de informes, entregando solamente los datos necesarios para generar dicho informe. Cualquier usuario familiarizado con el lenguaje SQL puede visualizar y ajustar este enunciado SQL. Vea Consultas SQL.
