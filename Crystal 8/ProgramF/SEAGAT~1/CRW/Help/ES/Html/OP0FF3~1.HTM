Operador Not (no)

Sintaxis Basic y Crystal.

Uso

not (x)

Revierte el valor True (Verdadero) o False (Falso) de x.

Valor de x Not x

True

False

False

True

  •     Not (Not(False)) = False
  •     Not (Not(True)) = True
Examplos

Los siguientes ejemplos son aplicables a sintaxis Basic y Crystal:

not (A>B and B>C)

Si A=5, B = 4, C = 3, la expresión (A>B and B>C) es VERDADERA. Ambas condiciones unidas por el operador     Booleano Y son VERDADERAS; Entonces, toda la expresión tiene un valor VERDADERO. El operador     No     cambia el valor de la expresión a FALSO.

not (A>B and B>C)

Si A=3, B = 4, C = 3, la expresión (A>B and B>C) es FALSA. Una de las dos condiciones unidas por el operador     Booleano Y es FALSA; Entonces, toda la expresión tiene un valor FALSO. El operador     No     cambia el valor de la expresión a VERDADERO.

not ({archivo.A MANO} - {archivo.PEDIDO} > 0) 

Returna VERDADERO

Si {archivo.A MANO} = 10 y {archivo.PEDIDO} = 11.

not ({archivo.A MANO} - {archivo.PEDIDO} > 0) 

Returna FALSO

Si {archivo.A MANO} = 10 y {archivo.PEDIDO} = 9.
