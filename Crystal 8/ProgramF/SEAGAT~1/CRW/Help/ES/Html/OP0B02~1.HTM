Constantes de interpretación de texto

Las siguientes constantes están definidas para admitir la importación de datos que estén formateados previamente en formato RTF o HTML, e insertarlo en un informe como campos de cadena o memorándum.

Constante Descripción

crUninterpretedText

Interpreta datos como texto simple.

crRTFText

Interpreta datos como datos en RTF.

crHTMLText

Interpreta datos como datos HTML.
