Join

Sintaxis básica y Crystal.

Sobrecargas
  • Join (lista)
  • Join (lista, delimitador)
Argumentos
  • lista es una matriz de Cadenas que contiene las subcadenas a unir.
  • delimitador es una Cadena opcional que se usa para separar subcadenas dentro de la cadena devuelta. Si se omite, se usa el carácter de espacio (" "). Si el delimitador es una cadena de cero longitud (""), todos los elementos de la lista se unen sin delimitadores.
Devuelve

Una cadena de texto.

Acción

Join devuelve una Cadena creada mediante la unión de un cierto número de subcadenas contenidas en una matriz.

Uso típico

Para convertir elementos almacenados en una matriz de Cadenas en una Cadena única.

Ejemplo(s)

Los siguientes ejemplos son aplicables a las sintaxis básica y Crystal:

Los ejemplos presuponen que lista es una matriz de Cadenas compuestas de los 3 elementos: "Chocolate", "Vainilla" y "Fresa".

Join (lista)

Devuelve la Cadena "Chocolate Vainilla Fresa".

Join (lista, "***")

Devuelve la Cadena "Chocolate***Vainilla***Fresa".
