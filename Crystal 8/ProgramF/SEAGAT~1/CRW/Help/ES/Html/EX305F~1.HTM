Lista A cambio de

Seleccione el campo que desea usar como condición para trazar valores en un gráfico o diagrama. Se trazarán puntos cada vez que cambie el valor en el campo de condición. Por ejemplo, si crea un gráfico de las ventas del último año para todos sus clientes, seleccionará el campo Nombre del cliente como condición y el campo Ventas del último año como valor. Cada vez que cambie el Nombre del cliente, se trazará un punto de acuerdo al valor de sus ventas del último año.
