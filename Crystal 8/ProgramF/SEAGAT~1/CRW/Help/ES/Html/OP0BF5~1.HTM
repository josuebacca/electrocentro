Redim

Sintaxis Basic x(n)

Use Redim x[n] (con los corchetes) sólo en sintaxis Crystal, y Redim x(n) (con las llaves) sólo en sintaxis Basic.

Uso

Redim x(n)

Cambia la dimensión de la matriz x al tamaño n, donde x es una matriz y n es un número entero positivo que especifica el nuevo tamaño de n.

Ejemplos

El siguiente ejemplo es aplicable para sintaxis Basic:

Dim x() As String
'Iniciar los tres primeros elementos de la matriz
x = Array ("a", "bb", "ccc")
'Ajustar el tamaño de x a 4; los valores antiguos se omiten
'x ahora es igual a una matriz ("", "", "", "")
Redim x(4)
'x ahora es igual a una matriz ("", "", "", "dddd")
x(4) = "dddd"
formula = x(4)

Comentarios

Cuando se cambia la dimensión de una matriz con Redim, los elementos de la matriz se completan con los valores predeterminados para ese tipo. Vea Valores predeterminados para los tipos simples (sintaxis Basic).

Sintaxis Crystal x[n]

  • Use Redim x[n] (con los corchetes) sólo en sintaxis Crystal, y Redim x(n) (con las llaves) sólo en sintaxis Basic.
  • A diferencia de Redim x(n) en sintaxis Basic, un enunciado Redim en sintaxis Crystal no puede tener matrices múltiples.
Uso

Redim x[n]

Cambia la dimensión de la matriz x al tamaño n, donde x es una matriz y n es un número entero positivo que especifica el nuevo tamaño de n.

Ejemplos

El siguiente ejemplo es aplicable para sintaxis Crystal:

Local StringVar matriz x:= ["a", "bb", "ccc"];
// ajustar el tamaño de la matriz a 4; los valores antiguos se omiten
// y se completan con valores predeterminados de las cadenas vacías
Redim x [4];
x [4] := "dddd"; // sólo se inicia x[4]

Comentarios

Cuando se cambia la dimensión de una matriz con Redim, los elementos de la matriz se completan con los valores predeterminados para ese tipo. Vea Valores predeterminados para los tipos simples (Sintaxis Crystal).
