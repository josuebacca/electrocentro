Operador Hacer matriz

Sintaxis Crystal.

El operador Hacer una matriz se puede usar sólo con sintaxis Crystal; mientras que la función Matriz se puede usar sólo con sintaxis Basic. Vea Matrices (Sintaxis Basic).

Uso

[x,y,z,...n]

Construya una matriz conteniendo los elementos x, y, z,...n.

Examplos

Los siguientes ejemplos son aplicables a la sintaxis Crystal:

[100,200,300,400]

[{archivo.CNTDA}, {archivo.CNTDB}, {archivo.CNTDC}]

[({archivo.MON1} * .5),({archivo.MON2} * .5), ({archivo.MON3} * .25)]

[500, ({archivo.CNTD} / 3)]

Nota:    No puede tener más de un tipo de datos en una matriz
