Operadores de comparación

Los operadores de comparación se usan para comparar datos de un Campo de datos con una Constante, con el contenido de otro campo de datos o con el resultado de una Fórmula.

Nota:    Una constante es un valor fijo (por ejemplo, una cadena de texto, fecha, o un valor numérico o monetario que ingrese) que permanece igual cuando se realiza un cálculo.
