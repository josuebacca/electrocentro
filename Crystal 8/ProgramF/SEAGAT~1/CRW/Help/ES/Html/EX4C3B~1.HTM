Vista de la opción OLAP

La vista de la opción OLAP le permite crear un mapa sobre un objeto OLAP. La información OLAP se puede usar como base para el mapa.

A cambio de

Use la lista A cambio de para especificar un campo a usar como condición para la visualización de un campo de resumen. Los valores de resumen se mostrarán cada vez que cambie el valor en el campo de condición.

Subdividido por

Use la lista Subdividido por para seleccionar un campo adicional sobre el que crear el mapa, si se desea. El programa usa este campo para subdividir los gráficos circulares o de barras que aparecen en el mapa.

Otras Dimensiones

Haga clic en el botón Otras dimensiocuadro de diálogo Dar formato a otras dimensionesnes para abrir el cuadro de diálogo Dar formato a otras dimensiones.

Haga clic en una dimensión (mostrada con su configuración de campo) de la lista y luego haga clic en Seleccionar un valor de campo para especificar un nuevo campo y fijar así la dimensión.
