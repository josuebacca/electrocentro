Is >= x

Sintaxis Basic y Crystal.

Is >= x es preferido en sintaxis Basic, mientras que Desde (upFrom x) es preferido en sintaxis Crystal.

Uso

Is >= x

Is >= x se usa para especificar un rango de valores mayores o iguales que el valor x. x puede ser de cualquier tipo: numérico, booleano, monetario, fecha, hora, fechahora y cadena.

Ejemplos

Los siguientes ejemplos son aplicables a sintaxis Basic y Crystal:

Is >= 250

El rango de todos los números mayores o iguales que 250.

Is > CDate (2000, 3, 17)

El rango de fechas desde el 17 de marzo del 2000 en adelante, incluida dicha fecha.
