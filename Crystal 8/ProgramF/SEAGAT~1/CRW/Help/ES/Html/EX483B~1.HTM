Vista de la opción Tabla cruzada

Use el botón de diseño de tabla cruzada al crear un mapa sobre un objeto de tabla cruzada. La información de tabla cruzada se puede usar como base para el mapa. Un mapa de tabla cruzada no requiere campos de grupo o de resumen. Con la presentación de tabla cruzada, puede determinar el valor del mapa, el desglose de campo geográfico y si se van a subdividir o no los datos.

Campo geográfico

Use la lista Mostrar campo geográfico para determinar el desglose geográfico del mapa. Por ejemplo, si selecciona el campo Cliente.País, el mapa va a mostrar valores para cada país. Si selecciona el campo Región.Cliente, el mapa va a mostrar valores para cada región.

Subdividido por

Use la lista Subdividido por para seleccionar un campo adicional sobre el que crear el mapa, si se desea. El programa usa este campo para subdividir los gráficos circulares o de barras que aparecen en el mapa.

Asignar

La lista Asignar muestra las opciones de valor para el mapa. En otras palabras, los datos contenidos en el campo en la lista Mostrar se usan en el programa como valores de mapa. Si la lista Mostrar contiene el campo Suma de cliente.Ventas del último año, los valores del mapa serán las ventas del último año para cada cliente.
