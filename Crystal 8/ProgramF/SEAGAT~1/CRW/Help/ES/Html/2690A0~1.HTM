Creación de un campo de parámetro para uso en
una consulta

Se puede crear un campo de parámetro mientras se obtiene una vista previa de la consulta en Crystal SQL Designer.

  1. En el menú Editar, haga clic Campo de parámetro.

    Aparece el cuadro de diálogo Ingresar nombre.

  2. Ingrese el nombre que desea usar para identificar el parámetro y luego haga clic en Aceptar.

    Aparece el cuadro de diálogo Campo de parámetro.

  3. Ingrese la siguiente información en los campos que aparecen:
      • Texto de solicitud: Ingrese el texto que desee que aparezca cuando el programa le envíe un mensaje.
      • Tipo de valor: Ingrese el tipo de datos del campo de parámetro.
      • Valor de solicitud predeterminado: Ingrese el valor que desea que use el programa si usted no entrega un valor nuevo.
      • Examinar campo: Use esta lista desplegable para especificar un campo predeterminado.
  4. Haga clic en Aceptar.

Ahora el campo de parámetro se puede usar en la consulta.
