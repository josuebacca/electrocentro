La sintaxis Basic no distingue entre mayúsculas y minúsculas

Lo que significa que formula, Formula y FORMULA se consideran de la misma manera. Esto es válido para todos los nombres de funciones y palabras clave de variable usadas en una fórmula con sintaxis Basic.

Nota:    La única excepción a esta regla es para las cadenas. La cadena "Hola" no es igual que la cadena "hola".

Practique usando la base de datos de muestra Xtreme_es

Muchos de los ejemplos de este sección se refieren a la base de datos de muestra Xtreme_es. Esta base de datos está incluida en Seagate Crystal Reports.

Examine el directorio de muestras y localice la carpeta de bases de datos, que contiene la base de datos Xtreme_es.
