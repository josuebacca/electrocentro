Asistente de resaltado

El Asistente de resaltado le permite aplicar formato condicional en forma eficiente a campos monetarios y numéricos. El Asistente de resaltado se usa mayormente para destacar valores de campo que en alguna forma se distinguen de otros valores de un informe. Por ejemplo, si desea resaltar el campo {cliente.VENTAS DEL ÚLTIMO AÑO} en rojo cada vez que las ventas sean superiores a $25.000, podría especificar en el Asistente de resaltado que, si el valor es superior a $25.000, se aplique un color rojo al fondo del campo. Puede también usar el cuadro de diálogo para establecer color de fuente y estilo de borde condicionales.

Nota:    Considere el Asistente de resaltado como una alternativa al Editor de fórmulas. Esencialmente, este asistente ejecuta la siguiente ecuación: Si la condición es verdadera, entonces aplicar estas especificaciones de formato.

Para este propósito, el Asistente de resaltado está dividido en dos secciones: la sección Lista de elementos (que muestra la fórmula) y el Editor de elementos (que le permite establecer la fórmula). El Editor de elementos incluye una vista de muestra para mostrar las especificaciones de formato aplicadas.

Estos son algunos beneficios adicionales a considerar:

  • El Asistente de resaltado también puede ser útil para campos numéricos o monetarios en tablas cruzadas.
  • Se puede deshacer el resaltado.
  • También se pueden establecer prioridades de resaltado.
Lista de elementos

La lista de elementos muestra las fórmulas de resaltado creadas para el campo actualmente seleccionado.

Prioridad

Use los botones Prioridad de la Lista de elementos del Asistente de resaltado y establezca prioridades para las fórmulas.

Quitar

Use el botón Quitar para descartar fórmulas resaltadas no deseadas para el campo actualmente seleccionado.

Quitar todos

Use el botón Quitar todos para descartar todas las fórmulas resaltadas para el campo actualmente seleccionado.

El valor es

Use esta lista para seleccionar un calificador para el número de valor monetario. Por ejemplo, puede aplicar un color rojo de fondo cuando el valor contenido en un campo sea igual a $25.000. En la lista El valor es, puede determinar el calificador 'igual a'.

Valores de campo

Use este cuadro de texto para ingresar el valor monetario o valor numérico al cual desea aplicar condiciones de resaltado. Por ejemplo, puede aplicar un color rojo de fondo mediante el Asistente de resaltado cuando el valor contenido en un campo sea igual a $25.000. En el cuadro de texto Valores de campo, escribiría 25.000.

Color de fuente

Use esta lista para seleccionar el color de fuente para el valor numérico o monetario al cual desea aplicar una condición de resaltado.

Fondo

Use esta lista para seleccionar un color de fondo para el valor numérico o monetario al cual desea aplicar una condición de resaltado.

Borde

Use esta lista para seleccionar un estilo de borde para el valor numérico o monetario al cual desea aplicar una condición de resaltado.

Muestra

Aparece una muestra usando la configuración seleccionada en el asistente.
