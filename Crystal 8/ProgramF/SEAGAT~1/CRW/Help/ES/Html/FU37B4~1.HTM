FRNetProfitMargin (beneficioNeto, ventas)

Sintaxis básica y Crystal.

Formato

FRNetProfitMargin (beneficioNeto, ventas)

Argumentos
  • beneficioNeto es un número o fórmula. beneficioNeto es la cantidad de ganancias que una compañía recibe durante el año por la venta de sus productos o servicios, menos todos los gastos. Es la figura final en el estado de pérdidas y ganancias de una compañía. Algunos analistas podrían usar las ganancias netas antes de los elementos extraordinarios y/o impuestos de ingresos para calcular esta relación.
  • ventas es un número o fórmula. ventas es el total de las ventas durante el período que aparece en el estado de pérdidas y ganancias de una compañía.
Acción

FRNetProfitMargin devuelve la relación entre beneficioNeto y ventas (por ejemplo, NetProfitMargin = beneficioNeto / ventas).

Uso(s) típico(s)

Esta relación se usa para medir que bien le va a la compañía. Cuanto más alto es el resultado, más lucrativa es la compañía.

Ejemplo(s)

Los siguientes ejemplos son aplicables a las sintaxis básica y Crystal:

FRNetProfitMargin(3, 50)

Devuelve 0.06.

FRNetProfitMargin(5.6, 100)

Devuelve 0.056.

Nota:    Ninguno de los argumentos de entrada pueden ser negativos; debe revisar si hay valores negativos antes de pasar los argumentos de entrada.

Comentario

FRNetProfitMargin es una de las muchas herramientas para análisis financiero que se usan para interpretar la posición financiera de una compañía. Como en el caso de todas las relaciones, se debe usar con precaución. Se puede usar sólo como una guía y no como prueba firme para formar una opinión sobre el estado financiero de una compañía.

Ninguna de las relaciones se puede usar exclusivamente para hacer análisis financieros y no hay reglas fijas sobre los resultados de las relaciones. Aparte de las diferencias entre los tipos de industrias, el resultado varía entre las compañías diferentes dentro de la misma industria y dentro de períodos de cuenta diferentes. Los resultados sólo se deben usar para hacer comparaciones relativas y análisis de tendencias, y no se deben tratar como algo absoluto.
