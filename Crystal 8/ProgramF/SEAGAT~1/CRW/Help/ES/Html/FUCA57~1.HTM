Val (cadena)

Sintaxis básica y Crystal.

Argumentos
  • cadena es una cadena de texto.
Devuelve

Un número fraccionario.

Acción

La función Val(cadena) lee una cadena de texto conteniendo números (por ejemplo una dirección, número telefónico, o número de la seguridad social) y los convierte a valor decimal. Val deja de leer de la cadena cuando encuentra el primer carácter en la cadena que descubre que no puede reconocer como un número o espacio.

Uso(s) típico(s)

Utilice está función para extraer sólo la porción numérica de una cadena que contiene tanto números como texto. Por ejemplo, puede extraer sólo el número de calle de una dirección, o sólo la porción numérica de una ID que contiene letras y números.

Ejemplo(s)

El siguiente ejemplo es aplicable a las sintaxis básica y Crystal:

Val("2234 100th Street")

Devuelve 2234100.

Comentario(s)

Esta función está diseñada para trabajar exactamente como la función de Visual Basic del mismo nombre.
