CBool (número o moneda)

Sintaxis básica y Crystal.

Argumento

Un argumento único que puede ser un valor de Número o de Moneda, o una expresión.

Devuelve

Un valor Booleano.

Acción

CBool devuelve Verdadero si el argumento es positivo o negativo pero no 0, y devuelve Falso si el argumento es 0.

Ejemplo

El siguiente ejemplo es aplicable a las sintaxis básica y Crystal:

CBool({Elemento.VENTASNETAS})

Devuelve Verdadero si las ventas netas registran un aumento o reducción; devuelve Falso si las ventas netas son 0.

Comentario

Esta función está diseñada para trabajar exactamente como la función de Visual Basic del mismo nombre.
