Para exportar a una fuente de datos ODBC

  1. Con el informe que desea exportar activo, haga clic en el botón Exportar de la barra de herramientas estándar.

    Aparece el cuadro de diálogo Exportar.

    {bmc bt_expor.gif}

  2. En la lista desplegable Formato, seleccione la fuente de datos ODBC que corresponda al formato en que desea exportar el informe. Por ejemplo,
    ODBC - CRSS le permite exportar el informe a una base de datos Microsoft SQL Server.
  3. Destino se omite cuando se exporta un informe a una fuente de datos ODBC. No necesita hacer ningún cambio en el cuadro de edición Destino. Haga clic en Aceptar en el cuadro de diálogo Exportar.
  4. Si la base de datos ODBC especifica una base de datos en particular, el informe se exporta a esa base de datos. De lo contrario aparece el cuadro de diálogo Seleccionar base de datos. Seleccione la base de datos a la que se va a agregar este informe como tabla nueva y luego haga clic en Aceptar.
  5. Si la base de datos ODBC que seleccionó requiere una ID de inicio de sesión y una contraseña, aparecerá un cuadro de diálogo para que especifique la información necesaria. Introduzca su ID y contraseña y luego haga clic en Aceptar.

    Aparece el cuadro de diálogo Introducir nombre de tabla ODBC.

  6. Especifique el nombre que desea darle a la nueva tabla de la base de datos y luego haga clic en Aceptar.

    El programa exporta el informe como tabla nueva a la base de datos especificada.

Nota:    Si el informe contiene un campo binario, no podrá exportarlo correctamente a una fuente de datos ODBC.
