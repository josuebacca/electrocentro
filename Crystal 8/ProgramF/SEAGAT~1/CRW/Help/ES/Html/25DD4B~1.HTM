Archivos de registro de Web/IIS

Puede usar Seagate Crystal Reports para generar informes sobre los archivos de registro de Web. Estos archivos de registro realizan un seguimiento de los diferentes tipos de eventos que ocurren cuando se usa un explorador y un servidor de web.

El archivo de traducción que Seagate Crystal Reports proporciona para los archivos de registro de Web/IIS es P2swblg.dll. Como el archivo de registro se almacena en un sistema de archivos planos, el archivo de traducción accede a éste directamente.

Nota:    Debe tener acceso de lectura UNC (convención de nomenclatura universal) a su sistema de archivos locales. UNC es un formato estándar para rutas que incluyen un servidor de archivo de red de área local. La sintaxis básica es: \\Server\Share\ruta\nombre de archivo.
