RecordNumber

Sintaxis básica y Crystal.

Devuelve

Un número entero

Acción

RecordNumber devuelve el número de registro actual.

Uso(s) típico(s)

Puede utilizar esta función para crear registros fórmulas de selección de registros para imprimir algunos registros y excluir otros.

Ejemplo(s)

El siguiente ejemplo es aplicable a las sintaxis básica y Crystal:

RecordNumber

Crea un campo que contiene el número del registro actual que usted puede colocar en su informe. Utilizar esta fórmula es equivalente a utilizar el campo Número de registro en la carpeta Campos especiales en el Explorador de Campos.

Nota:    El uso de esta función en una fórmula obliga a que la fórmula sea evaluada al momento de impresión. Para obtener más información sobre las consideraciones de evaluación de hora, véase Funciones de tiempo de evaluación.
