Asistente de sección

Este asistente aparece cuando elige el comando Sección en el menú Formato.

Use el Asistente de sección para realizar cambios de formato que afectan a secciones completas del informe.

Secciones

Este cuadro muestra todas las secciones del informe activo. Cuando resalta una sección en la lista, el programa establece los botones del cuadro de diálogo para indicar las acciones que puede realizar y las propiedades que puede establecer para la sección seleccionada.

Insertar

Use este botón para insertar una nueva sección en el informe inmediatamente debajo y del mismo tipo que la sección que ha seleccionado en la lista Secciones. Por ejemplo, si ha seleccionado una sección Detalles y hace clic en el botón Insertar, el programa inserta una nueva sección Detalles inmediatamente debajo de la sección seleccionada.

Eliminar

Use este botón para eliminar la sección que ha seleccionado en la lista Secciones. Sólo puede eliminar secciones que se han agregado a las secciones de origen, es decir, no se puede eliminar ninguna de las secciones de origen.

Combinar

Use este botón para combinar la sección actual con la sección compañera que aparece inmediatamente debajo de ésta. Por ejemplo, si tiene tres secciones, Detalles A, Detalles B y Detalles C, y resalta Detalles B y hace clic en el botón Combinar, el programa combina Detalles B y Detalles C en una sola sección.

Nota:    No se pueden combinar secciones de tipos diferentes, por ejemplo, no se puede combinar una sección de detalles con una sección de encabezado de grupo.

Botones de flecha

Use estos botones para mover una sección hacia arriba o hacia abajo con respecto de las otras secciones relacionadas. Por ejemplo, si tiene tres secciones, Detalles A, Detalles B y Detalles C, si resalta Detalles B y hace clic en la flecha hacia arriba, el programa mueve la sección seleccionada y la deja en la parte superior de las tres secciones Detalles. Por el contrario, si resalta Detalles B y hace clic en la flecha hacia abajo, el programa mueve la sección seleccionada a la parte inferior de las tres.

Nota:    No se puede mover una sección más allá de una sección de un tipo diferente. Por ejemplo, no se puede mover una sección de detalles más allá de una sección de encabezado de grupo.

Fórmula condicional

Haga clic en este botón si desea configurar una fórmula condicional de formato para la propiedad correspondiente. Vea Trabajo con Formato condicional para obtener más información.
