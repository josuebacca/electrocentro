Ficha Campos

Use la ficha Campos para seleccionar los campos que desea incluir en un informe.

Campos disponibles

Esta lista muestra todos los campos disponibles que puede usar en el informe.

Mostrar campos

Esta lista muestra todos los campos actualmente activos en el informe.

Agregar ->

Haga clic en este botón para agregar un campo seleccionado de la lista Campos disponibles a la lista Mostrar campos.

Agregar todos ->

Haga clic en este botón para agregar todos los campos que aparecen en la lista Campos disponibles a la lista Mostrar campos.

<- Quitar

Haga clic en este botón para quitar un campo seleccionado de la lista Mostrar campos.

<- Quitar todos

Haga clic en este botón para quitar todos los campos de la lista Mostrar campos.

Examinar datos

Haga clic en este botón para examinar los datos del campo que se ha seleccionado en la lista Mostrar campos.

Fórmula

Haga clic en este botón para crear una nueva fórmula para el informe.

Encabezado de columna

Escriba el encabezado de columna deseado en este cuadro de edición.

Buscar campo

Use este botón para buscar un campo específico en el cuadro de desplazamiento Campos de base de datos. Encontrará útil el botón Buscar campo para encontrar un campo específico en una lista de campos de base de datos.

Orden

Use los botones de orden para mover hacia arriba o hacia abajo un campo seleccionado en la lista Campos del informe. Los campos que aparecen en la parte superior de la lista Campos del informe se ubicarán más cerca del margen izquierdo del informe. Es decir, tienen una mayor prioridad.
