Ficha Texto

La ficha Texto le permite determinar el título del mapa, información de la leyenda y el título de ésta.

Título del mapa

Use este cuadro de texto para crear un título para el mapa. Crear un título de mapa es opcional.

Leyenda completa

Seleccione el botón Leyenda completa para un desglose detallado de cómo se han representado los datos en el mapa.

Comprimir leyenda

Seleccione el botón Comprimir leyenda para un desglose mínimo de cómo se han representado los datos en el mapa.

Ninguna leyenda

Seleccione este botón si prefiere no tener una leyenda con el mapa.

Hecho por el mapa

Seleccione el botón Hecho por el mapa para crear un título de leyenda.

Especificar

Seleccione el botón Especificar para crear un título de leyenda propio.

Título

Use este cuadro de texto para crear un título de leyenda para el mapa.

Subtítulo

Use este cuadro de texto para crear un subtítulo de leyenda para el mapa.
