Cómo conectarse a una fuente de datos ODBC

  1. Elija Conectar/Desconectar servidor del menú Base de datos.

    Aparece el cuadro de diálogo Explorador de datos.

  2. Elija el archivo de base de datos ODBC que desea abrir y haga clic en Conectar.

    Si la fuente de datos requiere de un nombre de usuario y de contraseña o de cualquier otra información de conexión, aparece un cuadro de diálogo de conexión.

  3. Escriba la información de conexión que generalmente usa para acceder a la base de datos y haga clic en Aceptar.

    Si no especificó una base de datos con la fuente de datos ODBC, aparece el cuadro de diálogo Seleccionar base de datos. Seleccione el archivo de base de datos y luego haga clic en Aceptar.
