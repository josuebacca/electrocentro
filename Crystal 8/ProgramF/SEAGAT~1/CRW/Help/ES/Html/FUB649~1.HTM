Condiciones para las funciones de resumen

A continuación se presenta una lista de las posibles cadenas de texto de condición para las funciones de resumen:

condiciones Booleanas

  • algún cambio
  • cambiar a Sí
  • cambiar a No
  • cada Sí
  • cada No
  • el próximo es Sí
  • el próximo es No

Condiciones de fecha

  • diario
  • semanal
  • bisemanal
  • quincenal
  • mensual
  • trimestral
  • semestral
  • anual

Condiciones de hora

  • por segundo
  • por minuto
  • por hora
  • por AMPM
