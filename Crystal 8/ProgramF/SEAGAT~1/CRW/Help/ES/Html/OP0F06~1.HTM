Operadores de estructura de control

Las estructuras de control gestionan el flujo de lógica en una fórmula. Puede usarlas para crear condiciones de configuración de Fórmula que, si se cumplen, activan consequences específicas o repiten una secuencia de acciones bajo ciertas condiciones.
