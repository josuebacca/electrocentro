Para exportar a HTML

  1. Con el informe que desea exportar activo, haga clic en el botón Exportar de la barra de herramientas estándar.

    Aparece el cuadro de diálogo Exportar.

  2. En la lista desplegable Formato, seleccione HTML 3.2.
  3. Seleccione un destino en la lista desplegable Destino. El resto de esta sección asume la selección de Archivo de disco para almacenar el documento HTML en un directorio de un servidor web.
  4. Haga clic en Aceptar.

    Aparece el cuadro de diálogo Exportar a directorio. Cuando se exporta a formato HTML, se guarda un informe en un archivo HTML. Si prefiere, haga clic en la casilla de verificación para guardar cada página en un archivo separado. Por esta razón, el programa le pide el nombre del directorio de un destino de exportación y usa nombres predeterminados para los archivos HTML. La página inicial HTML se guarda como Reportfilename.HTM. Éste es el archivo que debe abrir si desea visualizar el informe a través del explorador web.

  5. Seleccione un directorio existente o cree un nuevo directorio para el informe.
  6. Para desplazarse por archivos HTML separados haga clic en la opción Navegador de páginas.
  7. Haga clic en Aceptar.

    El programa exporta el informe al formato HTML.
