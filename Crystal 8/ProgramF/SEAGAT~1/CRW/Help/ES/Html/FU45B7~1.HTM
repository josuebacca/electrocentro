Year (x)

Sintaxis básica y Crystal.

Argumentos

x es un valor de fecha o de fecha/hora.

Devuelve

Un número entero.

Acción

Year extrae el año a partir de una fecha y lo devuelve como un número.

Uso(s) típico(s)

Utilice esta función siempre que necesite utilizar un año, convertido a un número, en cálculos o comparaciones.

Ejemplo(s)

Los siguientes ejemplos son aplicables para la sintaxis Crystal:

Year({archivo.ULTIMA FACTURA EN})

Devuelve 1989, cuando la última factura fue en 1989.

If Year({archivo.ULTIMA ACTUALIZACIÓN}) < 1988 Then
    "P.D. Usted se está perdiendo los muchos beneficios de nuestra nueva y mejorada versión. "
Else
    ""

Imprime una P.D. para aquellos clientes que se actualizaron antes de 1988, y no imprime nada para aquellos clientes que muestran una actualización más reciente.

Year(CDateTime(1996, 3, 3, 10, 33, 20))

Devuelve 1996.

Comentario(s)

Esta función está diseñada para trabajar exactamente como la función de Visual Basic del mismo nombre.
