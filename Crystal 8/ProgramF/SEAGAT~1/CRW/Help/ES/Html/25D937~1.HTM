Visual FoxPro

Los datos de Microsoft Visual FoxPro se acceden mediante ODBC, mientras que los datos de FoxPro, versión 2.5 y anteriores, se acceden directamente desde el motor xBase. Si va a usar la versión 2.5 o anteriores de FoxPro, vea dBASE, FoxPro, Clipper.

Seagate Crystal Reports proporciona el archivo Vfpodbc.dll del controlador ODBC, que permite que ODBC funcione con datos de Visual FoxPro.
