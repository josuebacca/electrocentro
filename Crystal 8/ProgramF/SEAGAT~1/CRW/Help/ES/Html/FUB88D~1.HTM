Fix

Fix y Truncate (x), Truncate (x, #lugares) son funciones equivalentes. Sin embargo, se prefiere Fix en la sintaxis básica y Truncate en la sintaxis Crystal

Sobrecargas
  • Fix (número)
  • Fix (número, #lugares)
Argumentos
  • número es el valor numérico para truncar; puede ser positivo, 0 o negativo.
  • #lugares es un número opcional que indica el número de lugares decimales para truncar. Si se omite, se asume 0.
Devuelve

es una cadena que indica entero que puede ser positivo, 0 o negativo.

Acción

Fix trunca un número en el número de lugares decimales especificado y lo devuelve. Si se omite #lugares, se asume 0.

Ejemplo(s)

Los siguientes ejemplos son aplicables a las sintaxis básica y Crystal:

Fix (123.678)

Devuelve 123.

Fix (-123.678)

Devuelve -123.

Fix (123.678,1)

Devuelve 123.6.

Fix (123.678,2)

Devuelve 123.67.

Comentario(s)
  • Esta función con un sólo parámetro, Fix (número), se designa para trabajar de forma similar a la función de Visual Basic del mismo nombre.
  • Fix (n) y Int (número) funcionan de igual manera excepto cuando n es negativo, en cuyo caso Fix devuelve el primer entero mayor o igual a n , e Int devuelve el primer entero menor o igual a n. Por ejemplo,

Fix (-10.2)

Devuelve -10.

Int (-10.2)

Devuelve -11.
