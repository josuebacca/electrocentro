Rnd

Sintaxis básica y Crystal.

Sobrecargas
  • Rnd ()
  • Rnd (seed)
Argumento

seed es un argumento de valor numérico opcional

Devuelve

Un valor numérico.

Acción

Rnd devuelve un número aleatorio mayor que o igual a 0 y menor que 1. Si seed es igual a 0, Rnd devuelve el número aleatorio que fue devuelto de la llamada previa a Rnd. Si no se especifica seed o éste es mayor que 0, Rnd devuelve el siguiente número aleatorio de la secuencia de números aleatorios generada internamente. Si seed es menor que 0, entonces Rnd utiliza este valor de seed para comenzar una nueva secuencia de números aleatorios y devuelve el primer valor de la secuencia.

Uso(s) típico(s)

Se utiliza cuando la fórmula requiere un número generado aleatoriamente como los requeridos para cálculos estadísticos o para seleccionar registros de forma aleatoria para limitar los datos en un informe.

Comentario(s)
  • Esta función es similar a la función de Visual Basic del mismo nombre.
  • Se puede llamar a Rnd sin haber comenzado una nueva secuencia de números aleatorios especificando un argumento seed negativo. Si lo hace así, se generará un seed interno usando el reloj del sistema.
  • El motivo para comenzar una secuencia numérica aleatoria llamando a Rnd con un argumento seed negativo y luego hacer llamadas subsiguientes a Rnd sin un argumento (o con un argumento positivo) es que el informe se verá exactamente igual siempre que se haga una vista previa. En otras palabras, le permite hacer uso de números aleatorios y conseguir resultados reproducibles.
