Asistente de informe de carta modelo

El Asistente de informe de carta modelo ofrece una solución fácil para crear cartas modelo que utilizan su base de datos como fuente de información de clientes. El asistente contiene muchas de las funciones del Asistente de informe estándar, aunque dispone de la ficha Carta modelo que le permite definir los campos de texto y de base de datos que aparecerán en cada sección de su carta. También puede usar la ficha Carta modelo para importar texto que haya creado en otra aplicación.

El Asistente de informe de carta modelo tiene las siguientes fichas:

  • Ficha Datos
  • Ficha Campos
  • Ficha Grupo
  • Ficha Total
  • Ficha Seleccionar
  • Ficha Carta modelo
