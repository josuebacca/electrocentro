CurrentFieldValue

Sintaxis básica y Crystal.

Devuelve

Puede devolver cualquier tipo de campo

Acción

CurrentFieldValue devuelve el valor actual del campo que se va a imprimir.

Uso(s) típico(s)

Puede usar CurrentFieldValue para ayudarle a dar formato condicional a un elemento de campo, objeto de campo o resumen o etiqueta de tabla cruzada.

Ejemplo(s)

En sintaxis Crystal, puede introducir una fórmula como:

if CurrentFieldValue = 0 then crRed

En sintaxis básica, la fórmula será:

If CurrentFieldValue = 0 Then formula = crRed

Formatea el color de fuente del objeto a imprimirse en rojo cuando el valor del campo es cero.

Comentario(s)
  • CurrentFieldValue sólo está disponible cuando se introducen fórmulas de formato de campo.
  • El tipo devuelto depende del contexto de la fórmula. Por ejemplo, si está dando formato a un campo de cadena, CurrentFieldValue devuelve un tipo de cadena; si está dando formato a un campo de fecha, CurrentFieldValue devuelve un tipo de fecha.
