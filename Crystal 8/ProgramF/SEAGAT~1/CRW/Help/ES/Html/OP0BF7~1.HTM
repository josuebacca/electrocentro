Operador Mayor o igual que

Sintaxis Basic y Crystal.

Uso

x >= y

x es mayor o igual que y.

Examplos

{archivo.PESO} >= 200

FALSO, donde {archivo.PESO} = 150 o {archivo.PESO}= 199.

{archivo.PESO} >= 200

VERDADERO, donde {archivo.PESO} = 400, {archivo.PESO} = 200, o {archivo.PESO} = 201.

{archivo.COSTO} >= {archivo.PRECIO}

FALSO, donde {archivo.COSTO} =350, {archivo.PRECIO} = 400.

{archivo.COSTO} >= {archivo.PRECIO}

VERDADERO, donde {archivo.COSTO} = 350 y {archivo.PRECIO} = 325, o donde {archivo.COSTO} = 350 y {archivo.PRECIO} = 350.
