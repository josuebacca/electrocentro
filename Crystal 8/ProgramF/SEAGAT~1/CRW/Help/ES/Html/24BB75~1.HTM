El Asistente de Vínculo Visual

En el Asistente de Vínculo Visual le permite vincular fácilmente dos o más tablas. Cuando selecciona el comando Agregar base de datos al informe del menú Base de Datos y selecciona una tabla de base de datos adicional, el Asistente de Vínculo Visual aparece y muestra la tabla adicional. Buscar el Asistente de Vínculo Visual y el comando Agregar base de datos al informe en la Ayuda en línea de Seagate Crystal Reports.

La manera más fácil de vincular tablas de bases de datos es hacer clic en el botón Smart Linking en el cuadro de diálogo Asistente de Vínculo Visual. El Smart Linking selecciona automáticamente los vínculos para sus tablas basado en campos en común en las tablas o campos indexados (si su base de datos admite campos indexados).
