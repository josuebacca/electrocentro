Alcances de variable para sintaxis Basic

Declarar un alcance para la variable regula su disponibilidad a través de diferentes partes del informe. Vea Declaraciones de variables con Dim y Alcance de las variables (Sintaxis Basic).

Los alcances de variable disponibles son:

  • Dim
  • Local
  • Global
  • Shared

Dim especifica una variable con alcance local. Local y Dim son palabras clave equivalentes.
