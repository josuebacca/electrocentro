Int (número)

Sintaxis básica y Crystal.

Argumento

Valor numérico

Devuelve

es una cadena que indica entero

Acción

Int devuelve la porción entera de un número específico.

Ejemplo(s)

Los siguientes ejemplos son aplicables a las sintaxis básica y Crystal:

Int (123.678)

Devuelve 123.

Int (-123.678)

Devuelve -124.

Int (-123.1)

Devuelve -124.

Comentario(s)
  • Esta función está diseñada para trabajar exactamente como la función de Visual Basic del mismo nombre.
  • Fix (n) (o Truncate (n)) e Int (n) son iguales excepto cuando n es negativo, en cuyo caso Fix o Truncate devuelven el primer entero mayor o igual a n e Int devuelve el primer entero menor o igual a n. Por ejemplo,

Fix (-10.2)

Truncate (-10.2)

Ambos devuelven -10.

Int (-10.2)

Devuelve -11.
