Agregar una tabla de base de datos ODBC a un informe

  1. Seleccione el comando Agregar Base de Datos a Informe del menú Base de datos.

    El cuadro de diálogo Explorador de Datos aparecerá en su pantalla.

  2. Elija el archivo de base de datos ODBC que desea abrir y expándalo haciendo doble clic.

    Si la fuente de datos requiere de un nombre de usuario y de contraseña o de cualquier otra información de conexión, aparece el cuadro de diálogo Inicio de sesión.

  3. Escriba la información de conexión que generalmente usa para acceder a la base de datos y haga clic en Aceptar.

    Si no especificó una base de datos con la fuente de datos ODBC, aparece el cuadro de diálogo Seleccionar base de datos. Seleccione el archivo de base de datos y luego haga clic en Aceptar.

  4. Destaque la tabla de base de datos y luego haga clic en Agregar.
  5. Haga clic en Cerrar cuando termine de utilizar el cuadro Explorador de datos.

    Aparece el cuadro de diálogo Explorador de campo con Campos de base de datos seleccionado y expandido. Se agrega la nueva tabla.
