Ficha Datos (OLAP)

Nota:    La misma ficha Datos aparece también en el cuadro de diálogo Dar formato a cuadrícula OLAP y en el cuadro de diálogo Insertar objeto de cuadrícula OLAP.

Use la ficha Datos del Asistente de informe OLAP para entregar información específica acerca de la fuente de datos OLAP:

  • El cubo específico sobre el que desea crear el informe
  • La conexión con el cubo
  • El nombre de la base de datos
  • El tipo de fuente de datos OLAP

Nota:    No podrá avanzar en la ficha Datos OLAP hasta que haya seleccionado adecuadamente un cubo OLAP. Además, trabajar con datos OLAP requiere entender de antemano el concepto OLAP. Asegúrese de conocer los conceptos OLAP y su software OLAP antes de intentar diseñar informes basados en OLAP. Si tiene preguntas, contacte con el administrador de su sistema.

Tipo de servidor

Cuando hace clic en este botón, aparece el cuadro de diálogo Elegir tipo de OLAP, que le solicita elegir el tipo de OLAP que busca.

Nota:    Esta opción sólo está disponible mediante el Asistente de informe OLAP.

Cubo

Esta lista especifica en qué cubo está trabajando actualmente.

Tipo

Este cuadro de información especifica qué tipo de datos está usando.

Base de datos

Este cuadro de información le indica el directorio donde están los datos del cubo.
