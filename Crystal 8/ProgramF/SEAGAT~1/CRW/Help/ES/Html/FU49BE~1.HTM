LooksLike (cadena, máscara)

Sintaxis básica y Crystal.

Argumentos
  • cadena es la cadena de texto o campo que contiene los valores de cadena de texto que están siendo comparados con la máscara.
  • máscara es una cadena de texto que proporciona una máscara para comparar el valor en el argumento de cadena.
Devuelve

Un valor Booleano.

Acción

Le permite localizar valores de campo utilizando comodines estándar de DOS (? = comodín para un carácter, * = comodín para varios caracteres). Hace esto comparando una cadena de caracteres con una máscara que contiene uno o más comodines. La función devuelve un Verdadero si la cadena coincide con la máscara y un Falso si la cadena no coincide con la máscara.

Uso(s) típico(s)

Utilice esta función siempre que quiera ubicar registros basados en dos o más valores de campo que son deletreados de manera diferente aunque son parecidos (tienen esencialmente estructura similar).

Ejemplo(s)

Los siguientes ejemplos son aplicables a las sintaxis básica y Crystal:

LooksLike ("Snyder","Sn?der")

Devuelve VERDADERO.

LooksLike ("Schneider","S?der")

Devuelve FALSO.

LooksLike ("Schneider","S?der")

Devuelve VERDADERO.

El siguiente ejemplo es aplicable a la sintaxis Crystal:

If LooksLike ({cliente.NOMBRE DE CONTACTO},"?ris" Then
    {cliente.NOMBRE DE CONTACTO}
Else
    ""

Devuelve Cris, Kris, e Iris, pero no Chris (asumiendo que todos los valores están en la base de datos {cliente.NOMBRE DE CONTACTO})
