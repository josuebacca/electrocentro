Asistente de N Superiores/Ordenamiento de grupo

Este asistente aparece cuando elige el comando Asistente de N Sup/Ordenamiento de grupos en el menú Informe.

Puede usar este asistente para identificar grupos "superiores" (los mejores representantes de ventas en un informe de ventas, los mejores clientes en un informe de pedidos, los estados con la mayor cantidad de clientes, etc.).

También puede usarlo para ordenar todos los registros de un informe o sólo los grupos superior e inferior de los registros. Dependiendo de qué opción elija, el asistente aparece con diferentes opciones.

Grupo/Ordenar

Use esta lista para elegir una de las siguientes opciones de ordenamiento para el grupo superior:

  • Ordenar todos los grupos
  • Ordenar los grupos N Superiores
  • Ordenar los grupos N Inferiores
según

Esta lista entrega una lista de campos de resumen del informe. Seleccione de esta lista el campo para el cual desea seleccionar valores N Superiores o N Inferiores. Cuando la opción Ordenar todos los grupos esté seleccionada, use el cuadro de desplazamiento debajo de la lista para ver los campos elegidos para el ordenamiento.

donde N es

Ingrese el número de valores N Superiores o N Inferiores (donde N es el número de valores) que desea seleccionar en este cuadro de edición.

incluir otros, con el nombre

Seleccione esta opción si desea incluir todos los campos que no entran en su selección de N Superiores o N Inferiores en un nombre de grupo especificado. Escriba el nombre del grupo en el cual desea que aparezcan todos los campos que no entran en su selección de N Superiores o N Inferiores.

Eliminar

Haga clic en este botón para eliminar el campo de resumen seleccionado. Éste está disponible sólo cuando la opción Ordenar todos los grupos está seleccionada.

Ascendente/Descendente

Seleccione la opción Ascendente para ordenar los campos en orden ascendente (A a Z, 1 a 9).

Seleccione la opción Descendente para ordenar los campos en orden descendente (Z a A, 9 a 1).
