Asistente de informe de etiqueta de correo

El Asistente de informe de etiqueta de correo le permite crear un informe con un formato para imprimirlo en etiquetas de correo de cualquier tamaño. Puede usar la ficha Etiqueta para seleccionar un tipo de etiqueta comercial o puede definir su propia presentación de filas y columnas para cualquier informe de estilo de varias columnas.

El Asistente de informe de etiqueta de correo tiene las siguientes fichas:

  • Ficha Datos
  • Ficha Campos
  • Ficha Etiqueta
  • Ficha Seleccionar
