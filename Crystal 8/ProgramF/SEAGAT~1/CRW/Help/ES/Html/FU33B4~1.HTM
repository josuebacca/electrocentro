Filename

Sintaxis básica y Crystal.

Devuelve

Una cadena

Acción

Filename inserta (en una fórmula) el nombre y la ruta del archivo el cual es incluido con el informe como un campo.

Uso(s) típico(s)

Puede usar esta función cuando desee imprimir el nombre del archivo en el informe.

Ejemplo(s)

El siguiente ejemplo es aplicable a las sintaxis básica y Crystal:

Filename

Crea un campo conteniendo la ruta y el nombre del archivo. El uso de esta fórmula es equivalente a usar el campo Ruta y nombre del archivo en la carpeta Campos especiales en el Explorador de campos.
