Lotus Domino

Seagate Crystal Reports puede leer una base de datos Lotus Domino mediante ODBC. Para Windows 95, Windows 98 y Windows NT, la capa de traducción de Lotus Domino consta de tres archivos:

  • Nsql32.dll
  • Nsqlv32.dll
  • Nsqlc32.dll

Estos archivos usan los controladores instalados por el cliente Lotus Notes para trabajar con la base de datos Lotus Domino. Se debe instalar el componente cliente de Lotus Domino en el equipo local.

Nota:    Se puede sustituir el servidor de Domino por el cliente Lotus Notes en la capa de base de datos de este diagrama.
