Ficha Seleccionar

Use la ficha Seleccionar para elegir campos y seleccionar así (o filtrar) registros en un informe. Al seleccionar registros de esta forma, puede aumentar la velocidad de un informe.

Campos disponibles

Esta lista muestra los campos de informe actualmente disponibles para su selección.

Seleccionar campos

Esta lista muestra los campos actualmente seleccionados en el informe.

Agregar ->

Haga clic en este botón para agregar el campo seleccionado en la lista Campos del informe a la lista Campos seleccionados.

<- Quitar

Haga clic en este botón para quitar el campo seleccionado de la lista Campos seleccionados.

Examinar datos

Haga clic en este botón para examinar los datos del campo seleccionado en la lista Campos seleccionados.

Cuadro de desplazamiento

Este cuadro de desplazamiento es dinámico. Los distintos tipos de datos y criterios de selección cambian las opciones del asistente.

Mediante el cuadro de desplazamiento, seleccione la condición que mejor finalice esta oración:

"Deseo seleccionar todos los registros donde el valor del campo seleccionado sea"

Puede seleccionar condiciones primarias, condiciones de fecha o condiciones booleanas, dependiendo del tipo de campo que ha seleccionado.

Eliminar

Haga clic en este botón para quitar el elemento seleccionado de la lista. Este botón sólo está disponible cuando se selecciona es uno de.
