Asistente de selección

Este asistente aparece:

  • Cuando elige el comando Asistente de Selección en el menú Informe.
  • Cuando hace clic en el botón Asistente de selección de la barra de herramientas estándar.

Use el Asistente de selección para seleccionar los registros o grupos que desea incluir en el informe (si no desea incluirlos todos). Por ejemplo, puede tener una base de datos de clientes que contiene registros de clientes de cada estado y provincia canadiense pero desea elaborar un informe sólo de los clientes de Texas. Seleccionar informes le permite restringir el informe de modo que sólo se incluyan los clientes de Texas. Al usar el Asistente de selección, el programa genera automáticamente una fórmula de selección de registros o grupos basada en una configuración especificada en el cuadro de diálogo.

Nota:    Este asistente le permite configurar criterios razonablemente complejos, pero para tener un mayor control sobre la fórmula generada, use el Editor de fórmula.

Configuración de búsqueda

Use las listas para seleccionar la condición que mejor termine esta oración:

"Deseo seleccionar todos los registros donde el valor del campo seleccionado sea"

Puede seleccionar condiciones principales, condiciones de fecha o condiciones booleanas, dependiendo del tipo de campo seleccionado.

Sugerencia (rapidez)

Haga clic en este botón para obtener más información para acelerar el proceso de elaboración de informes.

Examinar

Se ha incluido el botón Examinar como ayuda en la selección de los valores que definen la fórmula de selección. Al hacer clic en este botón, el programa muestra una lista de valores de campo en la base de datos. Si el valor que desea está en la lista, seleccione el valor y haga clic en el botón Pegar datos. El programa ingresa el valor seleccionado en el cuadro de texto activo (el cuadro de texto que muestra el punto de inserción).

Nota:    Si necesita seleccionar valores múltiples (por ejemplo, si desea incluir registros en los cuales el valor de campo es uno de tres valores), el punto de inserción se mueve automáticamente hacia el cuadro de texto siguiente tan pronto como termina de pegar datos en el cuadro anterior.

Mostrar/Ocultar fórmula

Los criterios de selección de registros que ingrese a través del asistente generan automáticamente una fórmula de selección de registros. Puede revisar y modificar esta fórmula al hacer clic en el botón Mostrar. Al hacer clic en el botón, el asistente se expande para mostrar la fórmula. Cuando haya terminado de editar y revisar, haga clic en el botón Ocultar para ocultar la fórmula. El Asistente de selección se actualiza automáticamente con cualquier modificación que haga a la fórmula.

Selección de registro/Selección de grupo

Seleccione la opción Selección de registro para crear una fórmula de selección de registros.

Seleccione la opción Selección de grupo para crear una fórmula de selección de grupos.

Nuevo

Haga clic en la ficha o botón Nuevo para crear una nueva fórmula de selección. Cuando hace clic en esta ficha, aparece el cuadro de diálogo Elegir Campo donde selecciona un campo para crear la fórmula de selección.

Eliminar

Haga clic en este botón para eliminar la fórmula de selección activa y la ficha asociada.

Editor de fórmulas

Al hacer clic en este botón, aparece el Editor de fórmula con la fórmula de selección mostrada en el cuadro de texto Fórmula.
