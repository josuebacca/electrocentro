Asistente de vínculo visual

Puede víncular bases de datos de modo que registros de una base de datos coincidan con registros relacionados de otra. Por ejemplo, si activa una base de datos Películas y una base de datos Estudio, puede vincular las bases de datos de modo que cada película (de la base de datos Películas) pueda coincidir con el estudio que la filmó (de la base de datos Ecomando Agregar base de datos al informestudio).

Cuando vincula, se usa un campo que sea común a ambas bases de datos. El programa usa el vínculo para hacer coincidir registros de una base de datos con aquellos de otra. En este ejemplo, el vínculo garantiza que los datos de cada fila del informe se refieran a la misma película.

Use Vínculo visual para vincular dos o más bases de datos. Cuando elige el comando Agregar base de datos al informe del menú Base de datos y selecciona una base de datos adicional, aparece el Asistente de vínculo visual y muestra la base de datos adicional.cuadro de diálogo Elegir Tablas para usar en Vínculo Visual

Vínculo visual

Este cuadro de desplazamiento muestra todas las tablas actualmente en su informe, junto con todos los vínculos e índices.

Tablas

Cuando hace clic en este botón, aparece el cuadro de diálogo Elegir Tablas para usar en Vínculo Visual donde puede elegir las tablas que desea vincular en su informe y si se usa un vínculo inteligente.

Organizar tablas

Cuando hace clic en este botón, el programa organiza las tablas para que se ajusten de la forma más eficiente dentro del cuadro de desplazamiento Vínculo visual.cuadro de diálogo Opciones de vínculo

Vínculo

Haga clic en Por nombre para vincular tablas según sus nombres o Por clave para usar información de clave externa. Haga clic en Vínculo para vincular automáticamente las tablas.

Borrar vínculos

Haga clic en este botón para borrar todos los vínculos de tablas actuales.

Opciones de vínculo

Para activar este botón, debe haber seleccionado una línea de vínculo. Una vez que tiene seleccionado un vínculo, haga clic en este botón para abrir el cuadro de diálogo Opciones de vínculo donde puede elegir el tipo de índice, permitir coincidencias parciales de texto, etc. para el vínculo seleccionado.

Eliminar vínculo

Haga clic en un vínculo y luego haga clic en este botón para eliminarlo.
