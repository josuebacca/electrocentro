Operador Porcentaje

Sintaxis Crystal.

Uso

x % y

Calcular el valor de x como un porcentaje     del valor de y [(x/y) * 100].

Examplos

Los siguientes ejemplos son aplicables a sintaxis Crystal:

{archivo.BALANCE OUTSTANDING} % {archivo.LIMITE DE CREDITO}

Retorna 30.00, donde {archivo.BALANCE OUTSTANDING} = $1500 y {archivo.LIMITE DE CREDITO} = $5000.

{archivo.CANTIDAD} % {archivo.LIMITE DE CREDITO}

Retorna 32.26, donde {archivo.CANTIDAD} = 2257.87 y {archivo.LIMITE DE CREDITO} = 7000.

Nota:    Si el denominador = 0, el informe se detiene con una advertencia dividir por cero. Si quiere evitar este tipo de problema, incluya código para capturar el error.

If {archivo.PRONOSTICO} = 0 Then
    0
Else
    {archivo.VENTAS} % {archivo.PRONOSTICO}
