Color

Color y RGB (rojo, verde, azul) son funciones equivalentes. Sin embargo, en la sintaxis básica se prefiere RGB y en Crystal, Color.

Argumentos
  • rojo es un valor Numérico que representa el componente rojo del color resultante, con un valor entre 0 y 255.
  • verde es un valor Numérico que representa el componente verde del color resultante, con un valor entre 0 y 255.
  • azul es un valor Numérico que representa el componente azul del color resultante, con un valor entre 0 y 255.
Devuelve

Atributo de color de fuente.

Acción

Color devuelve el atributo de color como se especifica en los componentes rojo, verde y azul.

Uso típico

Para especificar un color personalizado al escribir fórmulas de formato condicional para el color de fuente o de fondo de un campo de informe.

Ejemplo

El siguiente ejemplo es aplicable a la sintaxis Crystal:

if CurrentFieldValue > 10000 Then
    Color (150,230,150)
else
    Color (10,19,200)

Esta fórmula formatea el campo Numérico para que se despliegue en el color que especifica Color (150,230,150) si su valor actual es mayor que 10000; de lo contrario se despliega en Color (10,19,200).

Temas relacionados

RGB (rojo, verde, azul)
