Vista de la opción Grupo

La vista de la opción Grupo le permite mostrar un campo de resumen (como Ventas del último año) a cambio de un campo geográfico (como Región). Con el fin de crear un mapa usando le presentación de grupo, debe tener por lo menos un campo de grupo y un campo de resumen en su informe.

A cambio de

Use la lista A cambio de para especificar un campo a usar como condición para la visualización de un campo de resumen. Los valores de resumen se mostrarán cada vez que cambie el valor en el campo de condición.

Mostrar

La lista Mostrar entrega las opciones de valor para el mapa. En otras palabras, los datos contenidos en el campo en la lista Mostrar se usan en el programa como valores de mapa. Si la lista Mostrar contiene el campo Suma de cliente.Ventas del último año, los valores del mapa serán las ventas del último año para cada cliente.
