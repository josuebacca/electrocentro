Examplos

El siguiente ejemplo es aplicable a sintaxis Crystal:

If {archivo.LIMITE DE CREDITO} = 5000 and {archivo.VENDEDOR} = "SP" Then
    {archivo.CANTIDAD}
Else
    0

Si el límite de crédito es 5000 y el vendedor es SP (ambas condiciones verdaderas), entonces retornar el valor en el campo {archivo.CANTIDAD}, de lo contrario retornar cero.

A > B and B > C

Retorna VERDADERO, donde A = 10, B = 6, and     C = 3 (ambas condiciones VERDADERO)..

Retorna FALSO, donde A=10, B=6, and     C=7 (sólo una de las dos condiciones VERDADERO).

(A>B) and (A * C - D > E) and (E / D <= B)

Retorna VERDADERO, donde A = 7, B = 5, C = 3, D = 2, E = 10 (las tres condiciones son VERDADERAS).
