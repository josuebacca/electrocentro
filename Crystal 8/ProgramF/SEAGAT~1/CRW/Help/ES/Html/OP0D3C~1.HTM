División de enteros (x \ y)

Sintaxis Basic y Crystal.

Uso

x \ y

Divide el valor numérico x entre el valor numérico y, y devuelve un coeficiente que es un número entero.

Nota:    Si x o y no es un número entero, entonces éste se redondea antes de la operación.

Examplos

Los siguientes ejemplos son aplicables a sintaxis Basic y Crystal:

23 \ 5

Returna 4.

3 \ 2

Returna 1.
