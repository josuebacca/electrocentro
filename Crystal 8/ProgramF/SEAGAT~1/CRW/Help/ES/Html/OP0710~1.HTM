Alcances de variable para sintaxis Crystal

Declarar un alcance para la variable regula su disponibilidad a través de diferentes partes del informe. Vea Alcance de las variables (Sintaxis Crystal).

Los alcances de variable disponibles son:

  • Global
    La variable está disponible para fórmulas a través de todo el informe actual.
  • Shared
    La variable se puede compartir con un subinforme así como también el informe actual completo.
  • Local
    La variable es específica y sólo se puede usar en la fórmula en la cual está definida.

Para usar alcances de variable se agrega el alcance que se desea declarar con su variable al inicio del enunciado de declaración de variable. También puede omitir la palabra clave de alcance y comenzar la declaración de la variable con el nombre del tipo. Esto es como declarar una variable Global.

Ejemplos

NumberVar Monto;

Declara una variable global denominada Monto que puede contener cualquier valor de un tipo de datos entero o decimal.

Global NumberVar Monto;

Declara una variable global denominada Monto que puede contener cualquier valor de un tipo de datos entero o decimal. Esta variable estaría disponible para las fórmulas en todo el informe actual

BooleanVar Pendiente;

Declara una variable global denominada Pendiente que puede contiene un valor VERDADERO o FALSO.

Shared BooleanVar Pendiente;

Declara una variable compartida denominada Pendiente que puede contiene un valor VERDADERO o FALSO. Esta variable estaría disponible tanto en el informe principal como en cualquier subinforme contenido en el informe principal.

StringVar Apellido := "Adán";

Declara una variable global denominada Apellido que puede contener un valor de cadena y asigna la cadena "Adán" a esa variable.

Local StringVar Apellido : = "Adán";

Declara una variable local denominada Apellido que puede contener un valor de cadena y asigna la cadena "Adán" a esa variable. Esta variable estaría disponible sólo dentro de la fórmula en la que fue declarada.

Comentarios
  • El enunciado de declaración de variable se usa para separar un bloque de memoria y contener así el valor asignado a la variable y para asignar un valor predeterminado a ese bloque de memoria. El valor predeterminado asignado depende del tipo de datos almacenados por el declarador de variable particular usado y de si se asignó un valor cuando se declaró la variable.
  • Una variable declarada en dos fórmulas usa el mismo bloque de memoria (es decir, se refiere al mismo valor). Por ejemplo:

//Fórmula nº 1
NumberVar x;
x:=5

//Fórmula nº 2
NumberVar x;
x:=x+15

x ahora tiene el valor de 20.
