Servidor DB2

El servidor DB2 de IBM es una aplicación cliente/servidor, compatible con SQL, que forma parte del conjunto de administración de bases de datos de IBM.

Seagate Crystal Reports abre los datos SQL creados por DB2 mediante el archivo P2sdb2.dll, instalado con Seagate Crystal Reports. Este archivo de traducción funciona con los controladores de bases de datos de DB2 para leer datos de DB2. Debe tener instalado y configurado correctamente el cliente DB2 antes de usar este archivo de traducción.
