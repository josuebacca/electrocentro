Redim Preserve

Sintaxis Basic x(n)

Uso

Redim Preserve x(n)

Cambia la dimensión de la matriz x al tamaño n, mientras mantiene los valores iniciales de x. x es una matriz y n es un número entero positivo que especifica el nuevo tamaño de n.

Ejemplos

El siguiente ejemplo es aplicable para sintaxis Basic:

Dim x() As String
'Iniciar los tres primeros elementos de la matriz
x = Array ("a", "bb", "ccc")
'Ajustar el tamaño de x a 4; los valores antiguos se retienen
Redim Preserve x(4)
'iniciar x(4)
x(4) = "dddd"
'devuelve "dddd"
formula = x(4)

Sintaxis Crystal x[n]

  • Use Redim Preserve x[n] (con los corchetes) sólo en sintaxis Crystal, y Redim Preserve x(n) (con las llaves) sólo en sintaxis Basic.
  • A diferencia de Redim Preserve x(n) en sintaxis Basic, no puede tener matrices múltiples en un enunciado Redim Preserve en sintaxis Crystal.
Uso

Redim Preserver x[n]

Cambia la dimensión de la matriz x al tamaño n, mientras conserva los valores iniciales de x. x es una matriz y n es un número entero positivo que especifica el nuevo tamaño de n.

Ejemplos

El siguiente ejemplo es aplicable para sintaxis Crystal:

Local StringVar matriz x := ["a", "bb", "ccc"];
// ajustar el tamaño de la matriz a 4 mientras se mantienen los valores antiguos.
Redim Preserve x [4];
// ahora x = ["a", "bb", "ccc", "dddd"]
x [4] := "dddd";
x[4] // devuelve "dddd"
