FRAccRecTurnover (cuentasPorCobrar, ventas, númeroDeDías)

Sintaxis básica y Crystal.

Formato

FRAccRecTurnover (cuentasPorCobrar, ventas, númeroDeDías)

Argumentos
  • AccountReceivable es un número o fórmula. AccountReceivable es el total de cuentas por cobrar en el balance general de una compañía.
  • ventas es un número o fórmula. ventas es el total de ventas durante el periodo que aparece en el estado de pérdidas y ganancias de una compañía.
  • númeroDeDías es un número o fórmula. Por lo general, es el número de días en un año. Algunos analistas usan 360 y otros usan 365. La forma predeterminada para esta función es 360.
Acción

FRAccRecTurnover devuelve la rotación promedio de AccountReceivable por días (por ejemplo, AccRecTurnover = (AccountReceivable / ventas) * 360).

Uso(s) típico(s)

Esta relación mide la rotación de cuentas a cobrar por días. Una relación más pequeña representa una rotación más alta, lo que significa que las cuentas a cobrar reciben pago un poco después de ser incurridas. Una rotación más alta es generalmente preferible sobre una rotación más baja, la cual puede ser una indicación de cuentas dudosas.

Ejemplo(s)

Los siguientes ejemplos son aplicables a las sintaxis básica y Crystal:

FRAccRecTurnover(100, 600, 360)

Devuelve 60.

FRAccRecTurnover(200, 300, 360)

Devuelve 24.

Nota:    Ninguno de los argumentos de entrada pueden ser negativos; debe revisar si hay valores negativos antes de pasar los argumentos de entrada.

Comentario

FRAccRecTurnover es una de las muchas herramientas para análisis financiero que se usan para interpretar la posición financiera de una compañía. Como en el caso de todas las relaciones, se debe usar con precaución. Se puede usar sólo como una guía y no como prueba firme para formar una opinión sobre el estado financiero de una compañía.

Ninguna de las relaciones se puede usar exclusivamente para hacer análisis financieros y no hay reglas fijas sobre los resultados de las relaciones. Aparte de las diferencias entre los tipos de industrias, el resultado varía entre las compañías diferentes dentro de la misma industria y dentro de períodos de cuenta diferentes. Los resultados sólo se deben usar para hacer comparaciones relativas y análisis de tendencias, y no se deben tratar como algo absoluto.
