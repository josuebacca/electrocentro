Aged0To30Days, Aged31To60Days, Aged61To90Days

Sintaxis básica y Crystal.

Devuelve

Nada

Acción

Especifica un rango de valores de fecha que caen dentro de cierto periodo de tiempo antes que la fecha actual. Si la fecha actual es 12/30/98, Aged0To30Days especifica el periodo desde 12/1/98 a la fecha actual, Aged31To60Days especifica el periodo desde 11/1/98 a 11/30/98 y Aged61To90Days especifica el periodo desde 10/2/98 a 10/31/98.

Ejemplo(s)

Los siguientes ejemplos son aplicables para la sintaxis Crystal:

If {pedidos.FECHA DEL PEDIDO}en Aged0To30Days Then
    {pedidos.MONTO DEL PEDIDO}
Else
    0
