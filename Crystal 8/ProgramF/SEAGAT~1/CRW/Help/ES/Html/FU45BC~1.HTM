ExchGetPath (ruta)

Sintaxis básica y Crystal.

Argumentos

ruta es la dirección del emisor/receptor (tipo de datos de cadena).

Devuelve

Una cadena

Acción

Esta función retornará la información de contenedor en un campo de dirección.

  • Si el tipo de dirección es X500, la función retornará toda la información desde la primera instancia del código "CN=" hasta la última instancia del código "CN=". Si sólo hay una instancia del código "CN=", la función retornará NULO.
  • Si el tipo de dirección es X400, la función retornará todos los códigos "OU*=" (residiendo entre los códigos "P=" y "O=").
  • Si la dirección inicia con "DDA:", la función retornará toda la información después del código "DDA:".
  • Si el campo está en blanco, se retornará "DIRECCIÓN DESCONOCIDA".
Ejemplo(s)

Los siguientes ejemplos son aplicables para la sintaxis Crystal:

Campo = /o=Seagate Software/ou=CRYSTALNT/
/cn=Configuration/cn=Servers/cn=ESPRESSO/cn=Fredt
ExchGetPath ({Campo})

Devuelve "cn=Configuration/cn=Servers/cn=ESPRESSO".

Campo - c=US; a= ; p=Microsoft; ou1=James;
ou2=RichCorp; o=apps-wga;
dda:smtp=James@Rich.com
ExchGetPath ({Campo})

Devuelve "ou1=James; ou2=Richcorp".

Campo = DDA:MS=CRYSTAL/JAMES/crystal
ExchGetPath9 ({Campo})

Devuelve "MS=CRYSTAL/JAMES/crystal".
