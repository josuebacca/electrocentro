IsNull (Campo)

Sintaxis básica y Crystal.

Argumentos

campo es todo campo de base de datos, memo o BLOB válido.

Devuelve

Un valor Booleano

Acción

Evalúa el campo especificado en el registro actual y devuelve VERDADERO si el campo contiene un valor nulo.

Uso(s) típico(s)

Puede utilizar esta función en una fórmula de selección de registro para limitar el informe a los registros que tienen algo diferente a un valor nulo en el campo especificado. Puede también utilizarla para que Crystal Reports realice alguna acción siempre que encuentre un valor nulo.

Ejemplo(s)

Los siguientes ejemplos son aplicables a las sintaxis básica y Crystal:

IsNull({pedidos.MONTO DEL PEDIDO}) = produce los resultados siguientes:

Not IsNull({pedidos.MONTO DEL PEDIDO})

Cuando se utiliza como una fórmula de selección de registro, incluye en el informe sólo aquellos registros que tienen algo diferente a un valor nulo en el campo {pedidos.MONTO DEL PEDIDO}.

Comentario(s)

Algunas bases de datos soportan valores de datos nulos y otras no. Esta función no trabajará si la(s) base(s) de datos activa(s) no soportan valores nulos. Si la(s) bases(s) de datos soportan valores nulos, la función puede ser utilizada para referenciar los valores nulos que son creados como el resultado de una búsqueda fallida mientras se combinaba.

Nota:    El cero y el espacio en blanco (vacío o " ") no son valores nulos.
