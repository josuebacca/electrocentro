Ficha Personalizar estilo (OLAP y Tabla cruzada)

Use la ficha Personalizar estilo para dar formato a una cuadrícula con color de fondo, líneas de cuadrícula y márgenes de celda.

Nota:    La ficha Personalizar estilo sólo da formato a las dimensiones principales de cuadrícula seleccionada en la ficha Filas/Columnas.

Columnas

Esta lista muestra la dimensión y niveles usados como aspecto de columna en la cuadrícula.

Filas

Esta lista muestra la dimensión y niveles usados como aspecto de fila en la cuadrícula.

Color de fondo

Use esta lista para seleccionar un color para el nivel seleccionado de fila o columna.

Alias para fórmulas

Escriba el alias para la dimensión seleccionada. Es preferible usar nombres cortos de dimensión al crear fórmulas de formato condicional.

Mostrar márgenes de celdas

Seleccione esta casilla de verificación para ampliar los anchos de columna y mostrar así toda la cuadrícula. Si desactiva esta casilla de verificación, los anchos de columna serán uniformes y parte de los datos quedarán ocultos detrás de la cuadrícula.

Aplicar sangría a etiquetas de fila

Seleccione esta casilla de verificación y luego escriba la medida de sangría para las etiquetas de fila en el cuadro correspondiente. Ésta puede tener hasta dos páginas de largo.

Cuando especifica una sangría, todas las etiquetas de fila tendrán una sangría igual, excepto la etiqueta del miembro de entrada del primer nivel.

Repetir etiquetas de fila

Seleccione esta cuadrícula de verificación para repetir etiquetas de fila en cada página para las cuadrículas que se impriman a lo ancho en más de una página.

Mantener columnas juntas

Seleccione esta cuadrícula de verificación para juntar las columnas en cada página para las cuadrículas que se impriman a lo ancho en más de una página.

Totales de filas en la parte superior

Seleccione esta casilla de verificación para mostrar los totales en la parte superior (en relación al miembro de entrada de fila correspondiente).

Totales de columna a la izquierda

Seleccione esta casilla de verificación para mostrar los totales a la izquierda (en relación al miembro de entrada de columna correspondiente).

Suprimir filas vacías

Seleccione esta casilla de verificación para suprimir filas vacías para el objeto de cuadrícula OLAP. Si desactiva la casilla de verificación, las filas vacías aparecerán en el objeto de cuadrícula OLAP.

Suprimir columnas vacías

Seleccione esta casilla de verificación para suprimir columnas vacías para el objeto de cuadrícula OLAP. Si desactiva la casilla de verificación, las columnas vacías aparecerán en el objeto de cuadrícula OLAP.

Dar formato a líneas de cuadrícula

Haga clic en este botón para especificar la configuración de las líneas de cuadrícula en el cuadro de diálogo Dar formato a líneas de cuadrícula.

Especifique la cantidad de espacio entre el borde izquierdo de la página del informe y la etiqueta en el cuadro Espacio horizontal.
