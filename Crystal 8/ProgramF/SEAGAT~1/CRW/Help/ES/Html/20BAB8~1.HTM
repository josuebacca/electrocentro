Introducción general sobre la sintaxis Basic

Al crear fórmulas, tiene la opción de usar sintaxis Crystal o Basic. Las reglas de sintaxis se usan para crear fórmulas correctas. Casi todas las fórmulas escritas con una sintaxis se pueden escribir con la otra. Los informes pueden contener fórmulas que usan sintaxis Basic así como también fórmulas que usan sintaxis Crystal.

Si está familiarizado con Microsoft Visual Basic u otras versiones de Basic, es posible que la sintaxis Basic le resulte más familiar. En general, la sintaxis Basic es similar a Visual Basic salvo que tiene extensiones específicas para manejar la elaboración de informes.

Nota:    El procesamiento de informes no se ralentiza al usar sintaxis Basic. Los informes que usan fórmulas con sintaxis Basic se pueden ejecutar en cualquier equipo con Seagate Crystal Reports. También, al usar las fórmulas con sintaxis Basic no es necesario distribuir archivos adicionales con sus informes.
