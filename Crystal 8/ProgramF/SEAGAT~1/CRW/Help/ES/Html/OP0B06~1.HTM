Operador Tal como (like)

Sintaxis Basic y Crystal.

El operador Like (Como) es útil para seleccionar registros que se van a incluir o excluir del informe.

Uso

x like y
{fieldname} like "c?n*"

Este operador     verifica si el contenido de {fieldname} coincide con un modelo que se especifica en una cadena de caracteres "c?n*". Si el contenido del campo coincide con el modelo "c?n*", entonces la fórmula devuelve el valor VERDADERO. Si el campo empieza con otra cosa la fórmula devuelve Falso.

Use los símbolos de comodín ? y * para representar caracteres variables. El símbolo ? indica un solo carácter. El símbolo * representa cualquier número de caracteres.

Examplos

Los siguientes ejemplos son aplicables tanto a sintaxis Basic como Crystal:

{cliente.NOMBRE} like "D?n"

VERDADERO, donde {cliente.NOMBRE} = Dan or Don.

{cliente.NOMBRE} like "D?n"

FALSO, donde {cliente.NOMBRE} = Doug or Rob.

{cliente.APELLIDO} like "*s?n*"

VERDADERO, donde {cliente.APELLIDO} = Johnson or Olson or Olsen.

{cliente.APELLIDO} like "*s?n*"

FALSO, donde {cliente.APELLIDO} = Johnston or Smith.
