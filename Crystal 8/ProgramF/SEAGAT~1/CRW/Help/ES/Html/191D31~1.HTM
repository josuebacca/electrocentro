Fórmula 8

Funciones y operadores usados

If-Then-Else, Menor que (<), Negativo (-), Agregar (+), Round, Average([matriz]), Paréntesis ().

Propósito de la fórmula

Para controlar mejor sus compras e inventario, el dueño de un tienda desea que el informe de inventario incluya una columna "Cantidad a pedir". Quiere que la columna le indique qué cantidad necesita pedir para mantener disponible la cantidad promedio vendida mensualmente durante los últimos tres meses.

  • Si hay existencias disponibles, va a pedir la cantidad promedio vendida menos las existencias disponibles.
  • Si no hay existencias disponibles pero no hay pedidos pendientes, va a pedir la cantidad promedio vendida.
  • Si no hay existencias disponibles y sí hay pedidos pendientes, va a pedir lo suficiente para liquidar los pedidos pendientes más la cantidad promedio vendida.

Fórmula

If {archivo.DISPONIBLE} < 0 Then
((- ({archivo.DISPONIBLE})) + Round
((Average([{archivo.MES1}{archivo.MES2},
{archivo.MES3}]))))
Else
((Average([{archivo.MES1}{archivo.MES2},
{archivo.MES3}])) - {archivo.DISPONIBLE}))

Explicación

  • La fórmula usa el operador If-Then-Else para configurar un conjunto de cálculos si hay pedidos pendientes ({archivo.DISPONIBLE}< 0), y un conjunto diferente de cálculos si no hay pedidos pendientes.
  • La parte If de la fórmula usa el operador menor que (<) para ver si la cantidad disponible es menor que cero. Esto indica que, no sólo se vendieron todas las existencias sino que hay pedidos que no se han entregado (una situación de pedido pendiente).
  • Si hay un pedido pendiente, la parte Then de la fórmula calcula el monto a pedir como ventas promedio mensuales durante un período especificado de tres meses más el monto de pedidos pendientes (expresados como número positivo).
  • Usa el operador de negativo (-) para convertir el valor negativo de {archivo.DISPONIBLE}en un valor positivo.
  • Usa la función Average([matriz]) para promediar las ventas de producto durante {archivo.MES1}, {archivo.MES2}y {archivo.MES3} (los tres meses usados para determinar un patrón típico de ventas para el producto).
  • Usa la función Round para redondear las ventas promedio del producto hasta el número entero más cercano.
  • Usa el operador de adición (+) para sumar la cantidad disponible (ahora expresada como positivo) a la cifra redondeada de ventas promedio mensuales.
  • El resultado es el monto que se va a pedir.
  • Si no hay pedidos pendientes, la parte Else de la fórmula calcula el monto a pedir como ventas promedio mensuales durante el período especificado de tres meses menos la cantidad ya disponible.
  • Usa la función Average para promediar las ventas de producto durante {archivo.MES1}, {archivo.MES2}y {archivo.MES3} (los tres meses que permiten determinar un patrón típico de ventas para el producto).
  • Usa la función Round para redondear las ventas promedio del producto hasta el número entero más cercano.
  • Usa el operador de sustracción (-) para restar la cantidad disponible de la cifra redondeada de ventas mensuales promedio.
  • El resultado es el monto que se va a pedir.
