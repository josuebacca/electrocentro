Array (x,...)

Sintaxis básica

Array y MakeArray (x, ..) son funciones equivalentes. Sin embargo, Array no puede usarse en sintaxis Crystal ya que es una palabra clave para declarar variables de matriz en esa sintaxis.

Argumentos

Los argumentos dados son elementos de matriz con los cuales inicializar la matriz. El número de elementos de matriz puede ser cualquier número entre 1 y 1000. Todos los argumentos deben ser del mismo tipo. A diferencia de Visual Basic, no puede tener 0 número de elementos.

Devuelve

Matriz de valores.

Acción

La función Array crea una matriz, la inicializa con los elementos de matriz dados y la devuelve.

Uso típico

Para crear e inicializar variables de matriz en sintaxis básica.

En su lugar, en la sintaxis Crystal puede usar la notación con corchetes para las matrices.

Ejemplo

El siguiente ejemplo es aplicable a la sintaxis básica:

Lo siguiente crea una matriz de días de la semana, empezando con el lunes.

Dim semana, primerDía, últimoDía
semana = Array"Lunes", "Martes", "Miércoles", "Jueves", "Viernes", _
             "Sábado", "Domingo")
Rem primerDía contiene "Lunes"
primerDía = semana(1)
Rem últimoDía contiene "Domingo"
últimoDía = semana(7)
formula = primerDía

Devuelve "Lunes".

Comentario(s)

Esta función está diseñada para trabajar exactamente como la función de Visual Basic del mismo nombre.
