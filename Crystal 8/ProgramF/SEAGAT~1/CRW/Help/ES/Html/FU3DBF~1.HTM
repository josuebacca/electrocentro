MakeArray (x, ..)

Sintaxis básica y Crystal.

Array (x,...) y MakeArray son funciones equivalentes. Sin embargo, Array no puede usarse en sintaxis Crystal ya que es una palabra clave para declarar variables de matriz en esa sintaxis.

Argumentos

Los argumentos dados son elementos de matriz con los cuales inicializar la matriz. El número de elementos de matriz puede ser cualquier número entre 1 y 1000. Todos los argumentos deben ser del mismo tipo.

Devuelve

Matriz de valores.

Acción

La función MakeArray crea una matriz, la inicializa con los elementos de matriz dados y la devuelve.

Uso típico

Para crear e inicializar variables de matriz , especialmente en sintaxis básica.

En su lugar, en la sintaxis Crystal puede usar la notación con corchetes para las matrices.

Ejemplo

El siguiente ejemplo es aplicable a la sintaxis Crystal:

Local NumberVar Array x := MakeArray (1, 1000, 10, 100, 10);

Crea e inicializa una matriz x con cinco elementos Número: 1, 1000, 10, 100, 10.
