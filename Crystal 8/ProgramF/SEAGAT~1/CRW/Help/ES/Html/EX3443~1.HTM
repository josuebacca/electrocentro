Ficha Carta modelo

Use la ficha Carta modelo para agregar y eliminar campos y establecer las propiedades de formato de un informe de carta modelo.

Sección

Esta lista muestra todas las secciones de informe actualmente disponibles que puede usar en un informe de carta modelo.

Campos disponibles

Esta lista muestra una lista de todos los campos de base de datos disponibles que puede usar en un informe de carta modelo.

Examinar datos

Haga clic en este botón para examinar los datos del campo seleccionado en la lista Campos de base de datos.

Agregar a texto

Haga clic en este botón para agregar el campo seleccionado de la lista Campos de base de datos a la lista Campos de carta modelo.

Cuadro de edición Carta modelo

Esta lista muestra todos los campos actualmente en uso en una carta modelo.

Importar texto

Haga clic en este botón si desea importar texto ya preparado para usarlo en su informe de carta modelo. Cuando haga clic en este botón, aparece un cuadro de diálogo Abrir donde puede elegir el archivo de texto deseado.

Dar formato a texto

Haga clic en este botón para establecer las propiedades de formato de un informe de carta modelo. Cuando hace clic en este botón, aparece la ficha Redondeado (Editor de Formato) donde puede establecer opciones de párrafo, fuente y tabulación.

Eliminar

Haga clic en este botón para quitar el campo seleccionado de la lista Campos de carta modelo.
