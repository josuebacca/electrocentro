Funciones

Las funciones son procedimientos integrados o subrutinas utilizadas para evaluar, hacer cálculos o transformar datos.

Cuando usted especifica una función, Crystal Reports elabora un conjunto de operaciones integradas dentro de la función sin que usted tenga que especificar cada operación por separado. De esta forma, una función es una clase de proceso estructurado que hace la creación de informes más fácil y rápido para usted.

Se proporcionan ejemplos simples y directos para cada función. Además, para muchas funciones se proporcionan referencias para fórmulas específicas en Crystal Reports. Estas fórmulas muestran ejemplos más avanzados para resolver problemas cotidianos en la generación de informes.
