Trabajar con objetos de texto

Usted utilizará muchas de las capacidades de los objetos de texto cuando crea cartas modelo. Una breve descripción de los objetos de texto le facilitará crear cartas modelo en la siguiente sección.

  • Un objeto de texto puede contener tanto texto como campos. Usted utilizará ambos en este ejemplo.
  • Un objeto de texto puede ser ajustado; usted ajustará el tamaño del objeto de texto para que se imprima como una carta.

Los objetos de texto operan de dos formas:

  • el modo Mover/Ajustar, y
  • el modo Editar.
