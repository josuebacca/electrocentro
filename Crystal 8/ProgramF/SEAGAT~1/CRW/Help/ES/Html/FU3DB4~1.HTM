DefaultAttribute

Sintaxis básica y Crystal.

Devuelve

Puede devolver cualquier tipo de campo

Acción

El valor que devuelve DefaultAttribute depende del valor seleccionado para el atributo que está siendo formateado en la interfaz de valor de atributo.

Ejemplo(s)

DefaultAttribute puede usarse en conjunto con CurrentFieldValue para formatear condicionalmente campos sobre la base de su valor y condición preestablecida.

Los siguientes ejemplos son aplicables para la sintaxis Crystal:

if CurrentFieldValue < 0 then DefaultAttribute else crBlack

En sintaxis básica, la fórmula será:

If CurrentFieldValue < 0 Then
    formula = DefaultAttribute
Else
    formula = crBlack
End If

Da formato al color de fuente del objeto a imprimirse en el color preestablecido cuando el valor del campo es menor que cero , e imprime en negro cuando el valor del campo es mayor que cero.

Comentario(s)

DefaultAttribute sólo está disponible cuando se introducen fórmulas de formato.
