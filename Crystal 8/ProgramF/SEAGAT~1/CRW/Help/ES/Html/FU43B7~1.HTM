Time

Sintaxis Crystal.

Las funciones CTime y TimeValue son equivalentes a Time. Sin embargo, Time sólo puede usarse en sintaxis Crystal, ya que en sintaxis básica es un nombre tipo.

Sobrecargas
  • Time (número)
  • Time (cadena)
  • Time (fechaHora)
  • Time (HH, MM, SS)
Argumentos

Time (número)

  • Devuelve un valor Hora dado un número en unidades de 24 horas.
  • número también puede ser fraccional o negativo

Time (cadena)

Devuelve un valor Hora que representa la hora, dada un expresión de Cadena que especifique una hora entre 0:00:00 (12:00:00 A.M.) y 23:59:59 (11:59:59 P.M.), inclusive.

Time (fechaHora)

fechaHora es el valor Fecha/hora.

Time (hora, min, seg)

  • Hora es un número entero que representa la hora del día.
  • Min es un número entero que representa un minuto.
  • Seg es un número entero que representa un segundo.

Devuelve

TimeValue

Acción

Crea un valor Hora a partir de la información proporcionada.

Uso(s) típico(s)

Use esta función siempre que necesite trabajar con un valor Hora que se obtenga del valor Fecha/hora o de elementos individuales de hora (es decir, hora, minuto, segundo).

Ejemplo(s)

Los siguientes ejemplos son aplicables para la sintaxis Crystal:

Time(.2) y Time (1.2)

Ambas devuelven 4:48:00 AM

Time(-.2)

Devuelve 7:12:00 PM

Time("3:05pm")

Devuelve 3:05:00 PM

Time(DateTime(1996, 1, 1, 12, 10, 10)

Devuelve 12:10:10.

Time(10, 20, 22)

Devuelve 10:20:22.
