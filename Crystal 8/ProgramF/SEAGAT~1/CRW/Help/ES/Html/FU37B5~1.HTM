Hour (x)

Sintaxis básica y Crystal.

Argumentos

x es un valor de hora o valor de fecha/hora.

Devuelve

Un número entero

Acción

La función Hour extrae la porción de un valor de hora y lo devuelve como un número entero.

Uso(s) típico(s)

Utilice esta función siempre que necesite desplegar sólo la porción de hora de un valor de tiempo o para realizar cálculos basados en horas.

Ejemplo(s)

Los siguientes ejemplos son aplicables a las sintaxis básica y Crystal:

Hour(CTime(10, 20, 22))

Devuelve 10.

Hour(CTime(10, 00, 30)) - Hour(CTime(6, 00, 00))

Devuelve 4.

Comentario(s)

Esta función está diseñada para trabajar exactamente como la función de Visual Basic del mismo nombre.
