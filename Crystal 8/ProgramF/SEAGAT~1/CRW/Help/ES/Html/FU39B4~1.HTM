FREarningsPerCommonShare (beneficioNeto, dividendoPreferencial, númeroDeAccionesComunes)

Sintaxis básica y Crystal.

Formato

FREarningsPerCommonShare (beneficioNeto, dividendoPreferencial, númeroDeAccionesComunes)

Argumentos
  • beneficioNeto es un número o fórmula. beneficioNeto es la cantidad de ganancias que una compañía recibe durante el año por la venta de sus productos o servicios, menos todos los gastos. Es la figura final en el estado de pérdidas y ganancias de una compañía. Algunos analistas podrían usar las ganancias netas antes de los elementos extraordinarios y/o impuestos de ingresos para calcular esta relación.
  • dividendoPreferencial es un número o fórmula. dividendoPreferencial es el total de dividendos pagados a los accionistas preferidos durante el período contable bajo estudio. Esta información puede encontrarse en el estado de pérdidas y ganancias y estado de cambio en la posición financiera de una compañía. También se puede derivar en el balance general de una compañía.
  • númeroDeAccionesComunes es un número o fórmula. Por lo general, númeroDeAccionesComunes es el número de acciones comunes emitidas y pendientes. Esta información se puede encontrar en el balance general de una compañía.
Acción

FREarningsPerCommonShare devuelve la relación entre las ganancias netas distribuibles a los accionistas comunes (beneficioNeto-dividendoPreferencial) y númeroDeAccionesComunes (por ejemplo, EarningPerCommonShare = (beneficioNeto - dividendoPreferencial) / númeroDeAccionesComunes).

Uso(s) típico(s)

Esta relación se usa para calcular la cantidad de ganancias a las cuales tiene derecho una acción común. Esta relación se usa lado a lado con dividendo por acción común para determinar la relación de dividendos pagados. Algunos analistas pueden querer realizar un análisis de ganancia por acción completamente diluida. En este caso, tendrán que tener en cuenta además, todas las acciones comunes que podrían ser convertidas de las acciones preferidas convertibles.

Ejemplo(s)

Los siguientes ejemplos son aplicables a las sintaxis básica y Crystal:

FREarningsPerCommonShare(550000, 50000, 100000)

Devuelve 5.

FREarningsPerCommonShare(450000, 10000, 2000000)

Devuelve 0.22.

Nota:    

  • Ninguno de los argumentos de entrada pueden ser negativos; debe revisar si hay valores negativos antes de pasar los argumentos de entrada.
  • beneficioNeto no puede ser más pequeño que dividendoPreferencial, de otra manera usted recibirá un mensaje de error.
Comentario

FREarningsPerCommonShare es una de las muchas herramientas para análisis financiero que se usan para interpretar la posición financiera de una compañía. Como en el caso de todas las relaciones, se debe usar con precaución. Se puede usar sólo como una guía y no como prueba firme para formar una opinión sobre el estado financiero de una compañía.

Ninguna de las relaciones se puede usar exclusivamente para hacer análisis financieros y no hay reglas fijas sobre los resultados de las relaciones. Aparte de las diferencias entre los tipos de industrias, el resultado varía entre las compañías diferentes dentro de la misma industria y dentro de períodos de cuenta diferentes. Los resultados sólo se deben usar para hacer comparaciones relativas y análisis de tendencias, y no se deben tratar como algo absoluto.
