Fórmula 4

Funciones y operadores usados

Fórmulas anidadas, If-Then-Else, Sustraer (-), Multiplicar (*), Mayor que (>), Mayor o igual que (>=), Sum, Paréntesis ()

Propósito de la fórmula

Una tienda de computadoras vende hardware, software y libros. Le paga a sus representantes de ventas un 7% de todas las ventas de hardware (mensualmente) superiores a $5.000, un 10% de todas las ventas de software (mensualmente) superiores a $10.000, y un 5% de todas las ventas de libros (mensualmente) superiores a $1.000. El gerente de ventas quiere calcular la comisión de cada categoría para cada representante, totalizar la comisión debida a cada representante e indicar aquellos representantes que están facultados para recibir más de $5.000 en comisión total durante el mes. Lo calcularemos usando fórmulas anidadas, es decir, usando una fórmula como uno de los elementos de otra fórmula.

Fórmula(s)

@HARDWARE 

If (({archivo.VENTAS DE HARDWARE}-5000)> 0) Then 

    .07 * ({archivo.VENTAS DE HARDWARE}-5000) 

Else 

    0

@SOFTWARE 

If (({archivo.VENTAS DE SOFTWARE}-10000)> 0) Then 

    .10 * ({archivo.VENTAS DE SOFTWARE}-10000) 

Else 

    0

@LIBROS

If (({archivo.VENTAS DE LIBROS}-1000)> 0) Then 

    .05 * ({archivo.VENTAS DE LIBROS}-1000) 

Else 

    0

@COMISTOTAL

Sum([{@HARDWARE}, {@SOFTWARE}, {@LIBROS}])

@INDICADOR 

If {@COMISTOTAL} >= 5000 Then 

    " **** " 

Else 

    ""

Explicación

  • Las tres fórmulas (@HARDWARE, @SOFTWARE y @LIBROS) funcionan de la misma forma.
  • Usan el operador If-Then-Else para probar una condición, realizar una acción si la condición es verdadera; realizar otra si la condición es falsa.
  • Toman ventas totales de la categoría, usan el operador de sustracción (-) para restar el monto de ventas sobre el cual no se va a pagar comisión y prueban para ver si el monto restante es un número positivo (>0).
  • Si es así (>0), calculan las comisiones usando el operador de multiplicación (*) para multiplicar el porcentaje apropiado de comisión (expresado como decimal: .07, .10, .05) por el monto sujeto a comisión ({archivo.VENTAS DE HARDWARE}-5000}, etc.).
  • Si el monto restante es un número negativo (<0), no se calcula ninguna comisión y la fórmula imprime un monto cero (0.00).
  • @COMISTOTAL usa la función Sum para totalizar las comisiones adeudadas. En vez de incluir los cálculos para cada fórmula (doble tiempo y esfuerzo) sustituye el nombre de la fórmula en lugar de los cálculos. Cuando el programa encuentra el nombre de la fórmula sabe usar los cálculos subyacentes de la(s) fórmula(s) a las que se hace referencia.
  • @INDICADOR usa el operador If-Then-Else para evaluar los valores calculados por @COMISTOTAL y marcar como (****) a aquellos valores de $5.000 o más (>= 5000). @INDICADOR anida con éxito dos niveles de otras fórmulas: anida @COMISTOTAL, que es una fórmula que a su vez anida otras tres fórmulas: @HARDWARE, @SOFTWARE y @LIBROS. Cuando el programa ve @COMISTOTAL, ejecuta todos los cálculos subyacentes a los que hace referencia esa fórmula y las fórmulas que se usan en @COMISTOTAL.
  • Los paréntesis () se usan en toda la fórmula para controlar el orden de cálculo.
