Fórmula 3

Funciones y operadores usados

Concatenar (+), ToText, ToNumber, Multiplicar (*), Calcular rango (to), Indexar [], Paréntesis ()

Propósito de la fórmula

Un fabricante de velas, como parte de su contrato de préstamo con su banco, tiene que presentar un análisis mensual de inventario. El análisis debe incluir el costo de la tela en inventario de materia prima y el costo de la tela por número de elemento para cada uno de los elementos en el inventario de productos terminados.

La compañía usa una tela para todas las velas que fabrica y usa los caracteres 5º y 6º del número de elemento de cada producto para representar la cantidad de metros de tela (redondeado al metro más cercano) necesaria para fabricar dicho elemento.

En la carta modelo que el gerente le envía a su banco cada mes quiere que el computador inserte automáticamente la cantidad disponible, el número de elemento y el valor monetario de la tela para cada número de elemento.

Fórmula

ToText({archivo.CANTIDAD}) + " unidades, Elemento " + 
{archivo.ELEMENTO} +
", $ " + ToText({archivo.CANTIDAD} * 
ToNumber({archivo.ELEMENTO}[5
to 6]) * {archivo.COSTO DE LA TELA})

Resultado

Con un costo de tela de 14,88 dólares por metro, la fórmula entrega el siguiente resultado:

Explicación

  • Esta ecuación usa ToText para convertir el valor de {archivo.CANTIDAD} (un campo numérico) en texto, de modo que se pueda usar como parte de una cadena de texto concatenada.
  • Luego usa el operador de concatenación (+) para unir la versión en texto {archivo.CANTIDAD}con la cadena " unidades, Elemento ".
  • Vuelve a usar el operador (+) para unir la cadena resultante con el número de elemento, almacenado como el valor del objeto de texto {archivo.ELEMENTO}.
  • Usa el operador (+) una última vez para unir la cadena de texto concatenada con el costo de tela calculado (expresado como texto).
  • Para calcular el costo de tela:
      • La fórmula usa el operador de indexación [] para extraer los elementos 5º y 6º (metros de tela usados) de {archivo.ELEMENTO} (un objeto de texto).
      • El operador de cálculo de rangos (to) se usa para establecer el rango 5 a 6.
      • La función ToNumber convierte esos elementos en un número, de modo que se pueda usar en un cálculo numérico.
      • Usa el operador de multiplicación (*) para multiplicar ese número por el valor de {archivo.COSTO DE LA TELA} por la cantidad de unidades en inventario ({archivo.CANTIDAD}) para llegar al precio total.
      • Finalmente, usa la función ToText para convertir el precio total (un número) a texto que se pueda usar en la cadena concatenada.
