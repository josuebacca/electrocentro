ExchGetSite (dirección)

Sintaxis básica y Crystal.

Argumento(s)

dirección es la dirección del emisor/receptor (tipo de datos de cadena).

Devuelve

Una cadena

Acción

La función comenzará por determinar si la dirección está en formato X500 o X400. Una vez que esto ha sido solucionado, entonces la función determinará el Nombre del sitio.

  • Si el campo está en formato X500, la función extraerá la última instancia del código "/OU=" (no sensible a mayúsculas/minúsculas).
  • Si el campo está en formato X400, extraerá la instancia del código "O=" (no sensible a mayúsculas/minúsculas).

Nota:    Los campos deben seguir los estándares de tipo de dirección para las funciones para operarlos.

Ejemplo(s)

Los siguientes ejemplos son aplicables para la sintaxis Crystal:

Campo = /o=Seagate Software/ou=CRYSTALNT/
/cn=Configuration/cn=Servers/cn=ESPRESSO/cn=Fredt
ExchGetSite({Campo})

Devuelve "CRYSTALNT".

Campo = c=US; a= ;p=Microsoft; o=apps-wga;
dda:smtp=James@Rich.com.
ExchGetSite({Campo})

Devuelve "apps-wga".
