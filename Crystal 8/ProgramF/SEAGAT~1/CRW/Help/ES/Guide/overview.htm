Seagate Crystal Reports 8 proporciona un gran número de nuevas características y mejoras, incluidas muchas recomendadas por nuestros clientes. En particular, esta sección presenta: 

  • Elaboración de informes de alto rendimiento
  • Mejoras de rendimiento
  • Desarrollo de aplicaciones para Windows y para el web
  • Integración con Microsoft Office
  • Facilidad de uso y mejoras generales

Para averiguar qué características contienen las diferentes ediciones de Seagate Crystal Reports 8, consulte la tabla de características al final de la página.

Elaboración de informes de alto rendimiento 

Web Component Server

El nuevo Web Component Server* (Servidor de componentes para el web) ha sido diseñado para proporcionar una ayuda óptima para el entorno de cable fino del web. 

La arquitectura de niveles múltiples y alto rendimiento proporciona a múltiples usuarios una visualización de informes más rápida, mejor calidad de presentación y un mejor rendimiento general.  El acceso a páginas de informe a la orden le permite descargar sólo las páginas específicas del informe que desea ver, lo que mejora los tiempos de respuesta y reduce el tráfico en el web. Es compatible con muchas infraestructuras existentes del web como son Microsoft, Netscape, Lotus y servidores web CGI.

* Seagate Crystal Reports incluye licencias para 5 usuarios al mismo tiempo.

Visores de informes en el web

Visualice informes con calidad de presentación instantáneamente, en tiempo real, usando visores de informes para Microsoft y Netscape.
  • El nuevo visor de informes DHTML proporciona informes con calidad de presentación a través de exploradores del web compatibles sin necesidad de descargar ningún software al equipo cliente.    
  • El nuevo complemento de Netscape permite usar el Visor de informes para ActiveX en un explorador Netscape.  
  • El Visor de informes mejorado para Java se ha actualizado para usar el entorno de ejecución Java 2.0 de Java y ahora proporciona una mejor compatibilidad para impresoras.
  • El Visor de informes para ActiveX ahora permite copiar y pegar texto desde el visor a otras aplicaciones. 

Los visores de informes ahora permiten una representación progresiva de secciones de informes para una visualización más rápida en el web, incluso mientras se está procesando el informe. 

Hipervínculos

Use los hipervínculos para establecer vínculos a documentos relacionados, como archivos, direcciones de correo electrónico, sitios web u otros informes de Crystal, desde su informe. Se pueden asociar con la mayoría de los objetos de informe, incluidos campos de fórmula, texto, imágenes y bases de datos. Los hipervínculos se ejecutan simplemente haciendo clic en un objeto vinculado en cualquier visor. Se proporciona una compatibilidad URL total para hipertexto, archivo, correo y nuevos hipervínculos. 

Carpetas web

Si su sistema ejecuta Microsoft Windows 9x, Windows NT con Microsoft Office 2000 o Windows 2000, puede almacenar, actualizar y guardar informes directamente en una carpeta web en un servidor web. Ya no tiene que descargar informes, editarlos localmente y cargarlos al servidor web.

Mejoras de rendimiento

Rendimiento de procesamiento de informes

Seagate Crystal Reports 8 presenta mejoras del rendimiento de informes en todas las áreas de procesamiento. El Motor de informes, más pequeño y con más niveles, incorpora incluso mayor funcionalidad con su tamaño mejorado y mayor capacidad para procesar varias tareas simultáneas. 

Podrá ver considerables mejoras de rendimiento al abrir, actualizar, ordenar, filtrar y dar formato. La nueva compatibilidad para barras de progreso prácticamente elimina el tiempo de espera para visualizar la primera página de informes distribuidos por el web.

Al cargar informes con datos guardados, la visualización de la primera página es muy rápida debido a los cambios de formato RPT que permiten la descompresión de lotes de los datos guardados a la orden, en lugar de todos al mismo tiempo. 

Elaboración de página

La mayoría de los informes que contienen resúmenes, gráficos, mapas y cuadrículas OLAP ahora se elaboran con un paso de todos los registros. También es considerablemente más rápida la apertura de informes con grandes árboles de grupo, totalizador, imágenes o un número total de páginas alto.

Rendimiento SQL

Se ha mejorado la compatibilidad con servidores de bases de datos SQL de forma que se ejecuta una mayor cantidad de operaciones en el servidor, lo cual permite una mayor velocidad de procesamiento y un mejor uso de los recursos de red. Ahora se procesa más SQL en servidores de base de datos compatibles. La selección de elementos distintivos para seleccionar registros únicos devuelve menos datos de la base de datos lo que permite un mejor rendimiento general de procesamiento. La nueva compatibilidad con comunicación asincrónica le permite detener enunciados SQL en el servidor de base de datos.

Almacenamiento de archivos

El formato de archivo RPT se ha mejorado para crear archivos más pequeños con descompresión más rápida y un almacenamiento más eficiente de informes complejos que contengan datos guardados.      

Desarrollo de aplicaciones para Windows y para el web

Report Designer Component (RDC) mejorado

Diseñado para Microsoft Visual Studio y otros entornos de desarrollo basados en COM, el RDC* ofrece el mayor nivel de flexibilidad de elaboración de informes de cualquier componente de desarrollo de Crystal Reports. El RDC, provisto de un gran número de funciones, proporciona a los programadores más control sobre la presentación y formato de los informes. Las nuevas funciones** de creación de informes de ejecución permiten la creación dinámica de informes dentro de su aplicación, ya sea en código o a través de un asistente de creación de informes de ejecución. Para una mayor flexibilidad, el RDC ofrece la capacidad de usar campos no enlazados para crear informes completos sin depender de una conexión con bases de datos existentes.

 

*   En la edición para programadores se incluyen otras herramientas de desarrollo para integrar informes en aplicaciones como la interfaz Automation Server (OLE), Crystal Report Print Engine API, ActiveX (OCX/OLE) Control y Delphi VCL Control. Estas herramientas garantizan la compatibilidad retrógrada a los programadores al hacer la transición a RDC. Sólo están disponibles en tecnología de 32 bits y no incluyen mejoras realizadas al RDC en Seagate Crystal Reports 8. 

 ** Se aplican los derechos de licencia de ejecución.

Servidor ASP mejorado

El servidor ASP ha sido rediseñado para usar el Report Designer Component, lo cual proporciona mayor consistencia para los programadores de VB. Ahora puede aprovechar diversas mejoras de rendimiento como la compatibilidad con barra de progreso para página a la orden. Los programadores en Visual InterDev ahora pueden aprovechar las capacidades de elaboración escalables de alto rendimiento de Seagate Crystal Reports en sus proyectos web.

Controles de integración de informes

Los programadores en Microsoft Visual InterDev 6.0 pueden escribir programas ASP con informes de Crystal integrados usando el mismo modelo de objeto Report Designer Component (RDC) utilizado en los programas Win 32.    

Los programadores en web que usan Microsoft Visual InterDev 6.0 pueden usar los nuevos controles de integración de informes para escribir programas web y generar Seagate Crystal Reports alojados en un servidor de informes web o en un servidor ASP. Los usuarios finales de estos programas pueden entonces visualizar los informes usando un Visor de informes basado en un explorador.

Integración con Microsoft Office

Complementos para Microsoft® Excel / Access

Ofrece una forma sencilla para que los usuarios de Microsoft Office 97 y 2000 puedan crear informes dentro de entornos conocidos como Excel o Access. Se agrega el complemento adecuado al menú "Herramientas" y queda disponible cada vez que se ejecute Access o Excel. Simplemente seleccione los datos para el informe y el Asistente de informes le guiará en el proceso para crear un informe sencillo. Simplemente haga clic en Actualizar para actualizar cualquier cambio. El informe resultante se puede guardar o abrir en Seagate Analysis o Seagate Crystal Reports para visualizarlo y editarlo.

Exportación

Se ha mejorado la exportación de informes a Word, Excel y RTF para lograr la integración más directa posible de informes existentes a documentos de Office.

Compatibilidad con OLE DB

Seagate Crystal Reports 8 es compatible con OLE DB para una fácil conexión con datos almacenados en SQL Server y otras fuentes de datos que permiten este estándar.

Mejoras en la interfaz de usuario

La interfaz de usuario de componentes como las barras de herramientas y el nuevo Explorador de datos ofrecen un aspecto y un comportamiento familiares de Microsoft Office para reducir la curva de aprendizaje para quienes lo usan por primera vez.

Facilidad de uso y mejoras generales

Report Designer

La interfaz Report Designer ofrece un control mucho mejor de la colocación de objetos de informe con la capacidad de mover, alinear, ajustar el tamaño, copiar y pegar múltiples objetos y nuevas opciones de alineación y regla. Esto es especialmente útil para diseñar informes que se imprimen en formularios impresos.

Lenguaje mejorado de fórmula

Se ha mejorado el Lenguaje de Fórmula con una nueva sintaxis denominada Sintaxis Basic, que es un lenguaje parecido a Microsoft Visual Basic®. Puede crear fórmulas usando la sintaxis Crystal o Basic. Casi cualquier fórmula escrita con una sintaxis se puede crear con la otra, y los informes pueden contener ambos lenguajes de fórmula. Seagate Crystal Reports 8 incluye más de 60 nuevas funciones, operadores y estructuras de control, incluida la compatibilidad para enunciados de mayúscula/minúscula, repetición de ciclo y matrices dinámicas.

Explorador de datos

La conexión con los datos es más rápida y fácil con una interfaz gráfica similar al Explorador para seleccionar fuentes de datos. Los usuarios pueden elegir entre los controladores disponibles, ordenados por tipo de conexión, elegir conexiones que hayan usado (Historial) o guardado (Favoritos) previamente o crear conexiones nuevas mediante ODBC u OLE DB.

Exportación de archivos DHTML y PDF

Se ha mejorado la exportación de archivos para proporcionar una mejor preservación de formato.  La nueva compatibilidad con formatos de archivos Adobe AcrobatÒ (PDF) y DHTML (HTML 4.0) permite una elaboración más precisa de informes y visualización, almacenamiento o distribución fuera de línea.

Importación de texto preformateado

Existe la opción de especificar objetos de texto o campos de informe como texto RTF o HTML preformateado. Esto permite a Seagate Crystal Reports 8 interpretar cualquier información incrustada de formato y mostrar el texto en forma apropiada dentro de las capacidades de formato de Seagate Crystal Reports. El texto formateado se puede pegar en un informe o se puede leer en un campo de base de datos.

Elaboración de informes OLAP / tabla cruzada

Se ha mejorado el asistente de Informes OLAP / tabla cruzada para ofrecer 18 estilos predefinidos de cuadrícula y un mayor control de la presentación de cuadrícula, formatos, asignación y gráficos, incluida la capacidad de dar formato condicionalmente a filas o columnas individuales. La filtración, el agrupamiento y el ordenamiento de dimensiones OLAP en el Asistente es más fácil y más flexible ahora.

Agrupamiento y resumen avanzados

Ahora puede usar un agrupamiento jerárquico para organizar datos en un informe para mostrar relaciones jerárquicas en los datos. Para describir mejor un grupo, use un nombre de campo de base de datos o fórmula de Nombre de grupo para especificar el texto que aparece en el Encabezado de grupo. Elija si desea insertar un resumen o un subtotal en cada grupo o especificar el campo que se va a resumir o subtotalizar. Puede especificar esta opción sin seleccionar un campo antes. Vea el porcentaje de participación de un grupo en otro más grande. Por ejemplo, use el campo Resumen porcentual para mostrar el porcentaje de ventas en cada ciudad de un país basado en las ventas totales del país.

Formato de texto

La función de formato de texto se ha ampliado para admitir tamaños fraccionarios de fuente, tamaños condicionales de fuente, espaciado de carácter y línea, sangrados y rotación. Se han mejorado los bordes para admitir bordes redondeados , bordes circulares y ovalados. También puede elegir entre más formatos de fecha, hora y fecha-hora.

Gráficos

Se ha mejorado el Asistente de gráficos para ofrecer un mayor control sobre el formato de gráficos y leyendas, gráficos de totales acumulados y una mejor compatibilidad con tipos especiales de gráficos como los gráficos de burbuja. Las fichas Analizador, Vista previa y Profundizar admiten zoom. La información sobre herramientas muestra más información detallada para datos de gráficos.

Asignación geográfica

En la ficha Analizador, una nueva lupa opcional de mapa hace más fácil la navegación de mapas a un alto nivel de zoom y proporciona una forma rápida de navegar dentro de un mapa. Las inconsistencias entre los nombres de lugares en los mapas y los nombres de bases de datos se pueden resolver ahora en una forma similar a la asignación de campos. La información sobre herramientas ahora muestra información más detallada sobre los valores para regiones de los mapas. También puede crear mapas basados en grupos geográficos únicos.

Compatibilidad con formatos de imágenes

Seagate Crystal Reports ahora admite más formatos de imágenes en los informes, como imágenes BMP, JPEG, PICT, PNG, TIFF y Targa.

Guardar informes

Ahora se admite el autoguardado y los informes se pueden guardar con una "imagen" de vista previa de la primera página.

Controlador original de Lotus Notes

El nuevo controlador original elimina la necesidad de configurar el controlador SQL de Lotus Notes en cada escritorio.

Formato de archivo de la versión 7 

Los archivos Seagate Crystal Reports se pueden guardar o exportar en formato de archivo de la versión 7 para facilitar la distribución de informes a usuarios de la versión 7. Las características de Seagate Crystal Reports 8 no se admiten en los archivos de la versión 7.
