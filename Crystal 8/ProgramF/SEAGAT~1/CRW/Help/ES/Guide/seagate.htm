Seagate Analysis

Seagate Analysis es una herramienta de escritorio que le ayuda a comprender información importante de forma rápida y sencilla. Seagate Analysis ofrece consulta ad hoc, análisis multidimensional de datos (OLAP) y elaboración básica de informes. Los informes creados en Seagate Analysis se pueden abrir y editar en Seagate Crystal Reports. Visite el sitio web de Seagate Software para obtener más información.

Seagate Info

Seagate Info vincula Seagate Crystal Reports con un sistema probado escalable a nivel de la compañía para acceder, analizar y compartir información en toda la organización. Seagate Info ofrece tecnologías de última generación para consulta ad hoc, diseño de informes, análisis multidimensional de datos (OLAP) y elaboración de informes de empresa sobre una arquitectura multinivel que ha demostrado ser escalable en forma eficiente, fiable y segura para decenas de miles de usuarios. Visite el sitio web de Seagate Software para obtener más información.
